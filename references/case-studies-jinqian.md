金錢卦（六爻）案例集
本檔示範如何用金錢卦+納甲做完整斷卦。

案例一：求財近期得否（持世化生）
背景
- 問：本月能否求得財？
- 日：甲子，月：寅
- 擲卦結果（自下而上）：背背字、字背字、背背背、字字字、背字背、字背背
  → 8 7 9 6 8 8 → 陰 陽 陽 陰 陰 陰
- 本卦：地風升 (000110)
- 動爻：第3爻（老陽 → 變陰）、第4爻（老陰 → 變陽）
- 變卦：雷水解 (001010)

裝卦
- 升卦 = 震宮四世卦，世第4、應第1，宮五行木
- 六獸（甲日起青龍）：青朱勾螣白玄

|位|六獸|六親|干支|五行|動變|備註|
|---|---|---|---|---|---|---|
|6|玄武|官鬼|癸酉|金|—|—|
|5|白虎|父母|癸亥|水|—|旬空|
|4|螣蛇|妻財|癸丑|土|→午火(子孫)|世，動，日合|
|3|勾陳|官鬼|辛酉|金|→午火(子孫)|—|
|2|朱雀|父母|辛亥|水|—|旬空|
|1|青龍|妻財|辛丑|土|—|應，日合|

用神判定
- 求財 → 用神妻財
- 卦中妻財兩現：第1爻辛丑、第4爻癸丑（持世且動）
- 取動爻為主 → 第4爻癸丑（世位）

旺衰
- 妻財丑土：寅月（木旺克土）→ 月休
- 日支甲子，子丑相合 → 日合救
- 化午火（子孫）回頭生 → 大救

整體
- 妻財持世，雖月衰但日合且化回頭生 → 財可得
- 第3爻官鬼動 → 表「為事費心」或「同事競爭」，但化子孫即被解 → 無大礙
- 第5爻父母旬空 → 文書類事務暫無動靜，與求財無直接關聯

應期
- 用神丑：沖實之日「未」、合之日「子」（今日即合）
- 動爻：今日或巳午之日（子孫旺日）

結論
今日或未來 3-5 日內有財訊，金額不大但能到手。建議：把握機會，勿錯過。

【策略建議】
本卦：升
類型：吸引子（升卦）
策略：留
【下一步】維持現狀，不宜改變。目前位置有利，變動反而損失。

案例二：問丈夫工作能否升遷（用神不上卦取伏神）
背景
- 妻問丈夫近期升職能否成功
- 日：丙午，月：午
- 擲卦：六次皆得少陽 → 六爻全靜（罕見）
- 本卦：乾為天 (111111)

裝卦
- 乾為天 = 乾宮本宮卦，世第6、應第3，宮五行金
- 六獸（丙日起朱雀）：朱勾螣白玄青

|位|六獸|六親|干支|五行|備註|
|---|---|---|---|---|---|
|6|青龍|父母|壬戌|土|世|
|5|玄武|兄弟|壬申|金|—|
|4|白虎|官鬼|壬午|火|月日同建|
|3|螣蛇|父母|甲辰|土|應|
|2|勾陳|妻財|甲寅|木|旬空|
|1|朱雀|子孫|甲子|水|日破|

用神判定
- 妻問夫工作 → 用神官鬼
- 第4爻壬午火（官鬼）→ 用神上卦

旺衰
- 用神官鬼壬午火：午月當令旺 → 月建
- 日丙午：又是日建
- 月日雙旺 → 用神極旺
- 然而：靜爻不動，需看與世爻關係

世應關係
- 世第6爻壬戌（父母）→ 火生土，官鬼生世 → 對問卦者有利
- 但乾為六沖卦（純陽，上下相沖）→ 事易反覆

整體
- 用神官鬼月日同旺，事必成
- 然乾卦為六沖純陽 → 升遷過程多波折，可能有變數
- 卦無動爻 → 靜中待動，需外力推動

應期
- 用神火，旺則值令之日／月即應
- 午月本月，午日（每旬甲午、丙午、戊午、庚午、壬午皆是）即應
- 結論：升遷即將下達，但伴隨人事震盪（六沖卦特性）

【策略建議】
本卦：乾
類型：排斥子（乾卦）
策略：走（積極改變）
【下一步】積極改變，離開當前狀態。此位置不利久留，宜主動求變。
變卦路徑：乾→履（變3爻得50%）→ 對丈夫而言，第3爻動才轉好

案例三：問遺失物（用神伏神判斷）
背景
- 問：昨日遺失的錢包能否找回？
- 日：辛卯，月：卯
- 擲卦：本卦 天澤履 (111011)
- 動爻：第2爻動（九二老陽 → 變陰）
- 變卦：天雷无妄

裝卦
- 履卦 = 艮宮五世卦，世第5、應第2，宮五行土
- 六獸（辛日起白虎）：白玄青朱勾螣

|位|六獸|六親|干支|五行|備註|
|---|---|---|---|---|---|
|6|螣蛇|兄弟|壬戌|土|—|
|5|勾陳|子孫|壬申|金|世|
|4|朱雀|父母|壬午|火|月卯生火？卯木生午火 → 旺|
|3|青龍|兄弟|丁丑|土|—|
|2|玄武|官鬼|丁卯|木|應，日月同建|
|1|白虎|父母|丁巳|火|—|

用神判定
- 問遺失錢包（妻財）→ 卦中無妻財！
- 取本宮首卦艮為山的妻財伏神
- 艮卦六爻：丙辰土、丙午火、丙申金、丙戌土、丙子水、丙寅木
- 艮宮土，水為妻財 → 第5爻丙子水
- 飛神：第5爻壬申金（子孫）

判斷
- 飛神壬申金 → 金生水 → 飛生伏（小吉，但飛神壓制力強）
- 伏神丙子水：辛卯日，子卯刑 → 受傷
- 月卯木：水生木 → 伏神洩氣
- 玄武未臨用神（用神不上卦故無從臨）→ 但應爻丁卯木（官鬼）臨玄武
- 玄武官鬼旺於應位 → 物被人取走（盜賊象）

整體
- 伏神不旺、飛神壓住 → 難現
- 應爻官鬼旺、玄武 → 偷盜可能性大
- 結論：錢包難以找回，可能被取走

應期
- 伏神出現：需沖飛神之日（寅日沖申）或飛神空之日
- 但伏神衰弱 → 即使出現也難得實物

建議
若報案，可在寅日（沖飛之日）詢問警方進展，或可獲線索。但卦象不利，宜做好損失準備。

【策略建議】
本卦：履
類型：福地（履卦）
策略：守
【下一步】穩守不動，靜觀其變。位置尚可，不主動出擊，等待時機。

案例四：問疾病（子孫動制官鬼）
背景
- 問：母親感冒能否快好？
- 日：戊申，月：申
- 擲卦：水雷屯 (010001)
- 動爻：第2爻動（庚寅 → 庚卯，木進神）

裝卦
- 屯卦 = 坎宮二世卦，世第2、應第5，宮五行水
- 六獸（戊日起勾陳）：勾螣白玄青朱

|位|六獸|六親|干支|五行|備註|
|---|---|---|---|---|---|
|6|朱雀|兄弟|戊子|水|—|
|5|青龍|官鬼|戊戌|土|應|
|4|玄武|父母|戊申|金|月日同建|
|3|白虎|官鬼|庚辰|土|—|
|2|螣蛇|子孫|庚寅|木|世，動→卯木（進神）|
|1|勾陳|兄弟|庚子|水|—|

用神判定
- 問母病 → 用神父母
- 第4爻戊申金（父母）→ 用神上卦
- 月申、日申 → 父母月日兩建，極旺

判斷
- 父母旺 → 母親體質強
- 子孫持世動化進神（寅→卯）→ 醫藥力量增強，能克官鬼
- 卦中官鬼兩現：第3爻庚辰、第5爻戊戌（皆土）
- 子孫木動可克土（官鬼）→ 病可制
- 但官鬼戊戌得月生（土被金洩，反之，土被金生？金洩土）→ 官鬼略衰

旺衰
- 父母戊申金：月日同建 → 大旺
- 官鬼戊戌土：月申金洩土 → 休
- 用神（病人）旺、官鬼（病）衰、子孫（藥）動進 → 三利

整體
- 病能愈
- 應期：卯日（子孫進神到位日），或子日（沖戌、官鬼破日）

結論
母親身體底子強，加上藥物見效，數日內可愈。建議：按時服藥、注意保暖。

【策略建議】
本卦：屯
類型：一般
策略：觀
【下一步】觀察局勢，再做決定。情況中等，需要更多信息才能判斷。
變卦路徑：屯→比（變1爻得67%）

斷卦驗證清單（六爻版）
裝卦準確
- 上下卦正確？
- 各爻納甲（干支）查表正確？
- 八宮歸屬正確？
- 世應位置正確？
- 六親（依宮五行判定）正確？
- 六獸（依日干）正確？
- 旬空（依日干支）正確？
用神判定
- 問事類型對應正確？
- 用神是否上卦？
- 不上卦時是否取伏神？
- 用神兩現時取主用神原則？
旺衰判斷
- 月令旺衰判斷正確？
- 日辰生克判斷正確？
- 動爻作用判斷正確？
- 變爻回頭生克判斷？
吉凶綜判
- 用神 + 原神 + 忌神 + 仇神綜合？
- 反吟、伏吟、進神、退神有無？
- 三合、六合、六沖局有無？
- 化空、化破、化墓、化絕？
應期合理
- 用神靜 → 值或沖實之日？
- 用神動 → 值日或變爻之日？
- 用神空 → 出旬之日？
策略建議
- 不只說吉凶，也提供下一步行動？
- 結合梅花易數策略表交叉驗證？
- 倫理用語（傾向、可能、僅供參考）？
