現代參考書目（Modern References）

本檔註冊三本對六爻納甲／易經本體有重要橋樑作用的現代著作。
與 classics/（公有領域清代原典全文鏡像）不同，這些是仍有版權的現代教材，
因此本檔只記錄：書目資訊、價值定位、與本知識庫既有檔案的對應關係，
並摘錄其概念框架（不抄錄整段文字）。

==============================================================
為什麼需要現代橋樑
==============================================================

直接啃古籍（增刪卜易、卜筮正宗）的主要障礙：

1. **語境失配**：古人問「科舉、田宅、僕從」，但用戶問的是「簽證、合約、加密貨幣、程式碼憑證」。
   六親（父母／兄弟／子孫／妻財／官鬼）的本質沒變，但需要重新對應到現代生活場景。
2. **排盤密度**：納甲、世應、地支、五行、長生十二宮等多層對應，純文字描述難消化。
   圖解、樹狀圖、對照表能大幅降低初學門檻。
3. **義理失重**：只看干支五行的「參數運算」會把易經當成命理機械，
   但本卦與變卦的卦辭、爻辭依然提供決策的哲理框架。

下面三本書分別解決這三個問題。

==============================================================
一、《周易預測學》系列（現代六爻專書）
==============================================================

代表作者：邵偉華、王慶文、李洪成、廖墨香（不同流派）

**價值定位：六親的現代化映射**

把古代問事類型對應到現代生活：

古代用神對象|對應六親|現代映射範例
科舉、文書、官印|父母爻|簽證、合約、學位、執照、護照、憑證、API key、SSL 證書
田宅、車輛、房契|父母爻|不動產、車輛、租約、SaaS 訂閱
妻、妾、奴婢|妻財爻|配偶、伴侶、員工、外包、現金流、營收
夫、丈夫、官員|官鬼爻|配偶（女占）、上司、政府機關、訴訟、疾病、競爭對手攻擊
子嗣、僕從、藥|子孫爻|子女、晚輩、員工（部分）、寵物、解決方案、醫療、防火牆、解毒劑
兄、弟、同窗|兄弟爻|手足、同事、競爭者、合夥人、劫財事件

**對應本知識庫**：
- liuyao-yongshen.md 六親總原則 → 加入現代映射的延伸案例時參考此處
- case-studies-jinqian.md → 補充現代問事案例時，依此映射建構

**使用情境**：
用戶問「我的工作簽證會不會過？」→ 父母爻為用神（文書類），不是官鬼。
用戶問「這個 SaaS 訂閱要不要續？」→ 妻財（出財）+ 父母（契約）雙看。
用戶問「我寫的程式碼憑證會不會被吊銷？」→ 父母爻為用神，看是否被官鬼動克。

**注意**：流派不同，邵偉華偏重「梅花六爻合流」、王慶文偏重「火珠林正宗」、
李洪成偏重「實戰應期」、廖墨香偏重「商業預測」。引用時需註明流派。

==============================================================
二、《圖解易經》系列（視覺化排盤教材）
==============================================================

代表出版品：台灣知音出版社、大陸華齡出版社、紫微堂等「圖解XX」系列

**價值定位：納甲排盤規則的視覺化**

納甲體系初期需要記憶大量對應：
- 八純卦地支裝法（乾納甲壬子寅辰午申…）
- 八宮 64 卦的演化順序（一變、二變、…、遊魂、歸魂）
- 世應位置（八純世六、一變初、…、遊魂四、歸魂三）
- 六親判定（依本宮五行屬性決定誰是父誰是子）
- 長生十二宮（每個天干在十二地支的長生、沐浴、冠帶…墓、絕）
- 三合會局、六合、六沖、三刑、六害

**對應本知識庫**：
- najia-tables.md ← 是本工具最接近「圖解書」精神的檔案，但仍純文字
- najia-guide.md ← 操作流程描述
- scripts/najia.py ← 程式化裝卦，等同於「動態的圖解」

**使用情境**：
- 初學者裝卦時對著樹狀圖／表格學會更快，建議搭配此類圖書視覺練習。
- 本工具的 najia.py 可作為自動化的「圖解替代品」：輸入六次擲幣結果，直接輸出完整排盤。
- 對於 LLM 解卦無直接需求（LLM 不需視覺輔助），但用戶若要學習，這是最好的入門。

==============================================================
三、傅佩榮《易經白話解》／《解讀易經》
==============================================================

作者：傅佩榮（台大哲學系教授）
代表著作：《傅佩榮解讀易經》、《易經入門》、《易經占卜大全》

**價值定位：卦辭爻辭的中性哲理解讀**

特色：
1. **不怪力亂神**：純從義理（哲學、處世智慧）解卦，避免迷信化敘事。
2. **體例嚴謹**：每卦給出卦辭白話、彖傳白話、象傳白話、六爻爻辭白話 + 哲理引申。
3. **儒家本位**：以《易傳》（孔子十翼）的義理為主軸，與「象數派」（納甲、京房）形成互補。

**對應本知識庫**：
- yaoci.md（384 條爻辭全文）← 古文原文已在此
- zhouyi-zhuan.md（彖傳象傳）← 古文原文已在此
- 64gua.md（六十四卦簡介）← 卦象、結構說明

**使用情境**：
- 算出本卦＋變卦後，除了納甲層次的「五行干支力學分析」，
  還可結合傅佩榮的卦辭哲理，提供「決策層面的哲理建議」。
- 例如算出「澤地萃」（聚集卦），納甲分析告訴用戶財運走向，
  傅佩榮的解讀則提醒「萃者聚也，王假有廟」——強調聚眾需有共同信念，
  讓預測從「會不會發生」升級到「該如何面對」。
- 對應本工具 SKILL.md 強調的「策略建議」核心理念高度契合。

==============================================================
LLM 解卦時的引用原則
==============================================================

1. **古籍優先**：力學依據、用神原則、特殊情況辨析 → 引 classics/ 對應章節（已語譯）。
2. **現代橋樑補位**：當用戶問題涉及現代場景（簽證、加密貨幣、程式碼、SaaS、AI 模型等）
   而古籍無直接對應 → 依本檔「六親現代映射表」延伸，並在回答中註明「依現代六爻派延伸」。
3. **義理升華**：解卦末段提供「策略建議」時 → 可引《易傳》原文（已在 zhouyi-zhuan.md）
   並參考傅佩榮類型的義理解讀風格（中性、不怪力亂神、聚焦處世智慧）。
4. **不抄襲**：本檔僅記錄「概念框架」與「對應關係」，引用具體用語應改寫為自己的論述。
   不引用任何完整段落，避免版權問題。

==============================================================
取得方式（提示用戶）
==============================================================

- 《周易預測學》：邵偉華著，敦煌文藝出版社／團結出版社等版本流通，台灣可購繁體版。
- 《圖解易經》：知音、華齡、紫微堂等多家出版社均有圖解系列，建議先翻書選自己喜歡的呈現風格。
- 傅佩榮系列：天下文化、立緒文化出版繁體版，簡體版有中信出版社。
