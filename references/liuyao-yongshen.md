六爻用神對照與十八類分占
本檔對應 18-divinations.md（梅花易數版），但以「六爻納甲」體系重新組織。
原典詳論：增刪卜易卷三、卷四（130 章中約 100 章為十八類分占）、卜筮正宗卷二「十八論」。
路由表見 classics/classics-index.md。

用神判定總原則
六爻體系下，問事即取相應的「六親」為用神：
六親|代表事物
父母|長輩、文書、房屋、車輛、學業、契約、信件、文件
兄弟|自身、兄弟姐妹、同事、朋友、夥伴、競爭者
子孫|子女、晚輩、下屬、寵物、醫藥、僧道、解神（剋官鬼）
妻財|妻、財、買賣、貨物
官鬼|夫、官、考、病、訟、鬼神、災禍
判斷邏輯：
1. 看用神是否上卦（即卦中是否有該六親）
2. 若不上卦，從本宮首卦取伏神
3. 看用神旺衰（月令、日辰、動爻扶抑）
4. 看用神動靜（動則有變、靜則待時）
5. 看用神是否空、破、墓、絕
6. 看用神所臨六獸（青龍喜、白虎凶等）
7. 看世應與用神關係

十八類分占（六爻版）

一、婚姻占
男占：
- 用神：妻財（妻子）
- 兼看：世爻（自己）、應爻（對方）
- 忌：兄弟動克妻財
- 喜：子孫動生財（兒子緣份）
- 凶兆：妻財化空、化破、化絕 → 婚不成
女占：
- 用神：官鬼（丈夫）
- 兼看：世爻、應爻
- 忌：子孫動克官鬼
- 喜：父母動生官鬼？要看具體。一般父母動傷子孫間接護官
- 凶兆：官鬼化空 → 對方無心、無誠意
合婚信號：
- 世應相生（雙方有緣）
- 世應相合（情投意合）
- 三合局含世應（緣分深）
- 用神與世爻相合（你愛對方）

二、求財占
- 用神：妻財
- 原神：子孫（生財）
- 忌神：兄弟（劫財）
- 仇神：父母（克子孫、洩官鬼）
判斷：
- 妻財旺、持世或應動生世 → 大吉
- 兄弟動克妻財 → 破財
- 子孫動生妻財 → 財源不斷（化財）
- 妻財空、破 → 求而不得
- 妻財化兄弟 → 破耗
- 官鬼動洩妻財（傳統說財生官、官洩財）→ 因公支出
求財場合：
- 求正財：看妻財是否清純不雜
- 求偏財（投機、博弈）：看妻財是否動，且子孫扶
- 求合作財：看應爻是否生世
- 求工資：看父母（憑文書工作領薪）

三、疾病占
- 用神：官鬼（疾病本身）
- 病人本人：世爻
- 醫生：子孫
- 藥物：子孫
判斷規則：
- 官鬼旺動克世 → 病重
- 子孫動克官鬼 → 病輕、易愈
- 官鬼空、破 → 病退
- 官鬼入墓 → 病拖延、纏綿
- 世爻空 → 危險
- 世化空、化破 → 危
應期：
- 官鬼之沖日 → 病轉
- 子孫旺日 → 病愈
- 官鬼出墓之日 → 病再發
慎言：占病涉及生死，斷語必含「請就醫」「卦象僅供參考」。

四、官運占（事業、考試、升遷）
- 用神：官鬼（官位）
- 原神：妻財（生官）
- 忌神：子孫（傷官）
- 兼看：父母（文書、考試）
科舉考試特別：
- 用神：父母（科甲、文書）
- 喜官鬼旺生父母
- 子孫動傷官鬼 → 考試不利
升遷判斷：
- 官鬼旺持世或應動來生 → 升遷有望
- 妻財旺動生官 → 因財得官（或行賄之嫌）
- 父母旺 → 文書到位（任命書）

五、訴訟占
- 用神：官鬼（對方、官府）
- 兼看：世（自己）、應（對方）、父母（狀紙、判決書）
- 子孫：解神（化解、和解、撤訴）
判斷：
- 子孫動克官鬼 → 案可解、易勝
- 官鬼旺克世 → 官司不利
- 父母動 → 文書動，有判決
- 官鬼動傷世 → 受刑罰
- 世旺、應衰 → 我勝
- 應旺、世衰 → 對方勝

六、出行占
- 用神：自身用世爻；交通工具用父母
- 子孫：解神（化解險阻）
- 官鬼：阻礙、險阻、盜賊
- 兄弟：劫財，旅途破費
判斷：
- 世動 → 必行
- 世空 → 不去或臨時取消
- 官鬼動克世 → 路上有險
- 父母動 → 文書、交通工具相關（航班、車票）
- 子孫旺 → 旅途平安

七、行人占（問遠人何時歸）
- 用神：行人所對應的六親
  - 父母：問長輩
  - 兄弟：問同輩、朋友
  - 子孫：問晚輩、子女
  - 妻財：問妻、情人
  - 官鬼：問丈夫
判斷：
- 用神動來生合世 → 即將歸來
- 用神靜 → 暫不歸
- 用神空、破 → 未動身或迷失
- 用神入墓 → 在外有阻、生病
應期：用神出墓、值日、沖墓之日

八、失物占
- 用神：依物品性質
  - 妻財：錢、物品
  - 父母：文書、證件、車輛
  - 子孫：小物、藥物、寵物
- 找回判斷：
  - 用神旺、不空 → 可找回
  - 用神入墓 → 物在隱蔽處（墓庫方向）
  - 用神化兄弟 → 被人拿走
  - 玄武臨用神 → 被偷
  - 用神空、破 → 難找
方位：用神所在卦位定方位（後天八卦方位）

九、求事占（廣義事務）
- 用神：所求之事的對應六親
- 應爻：辦事人或對方
- 世爻：自己
判斷：
- 世應相生、相合 → 事成
- 世應相沖、相克 → 阻礙
- 應動克世 → 對方為難我
- 子孫動 → 喜事、解決方案

十、家宅占
- 用神：父母（房屋）
- 兼看：六親在卦中之安否
判斷：
- 父母旺穩、無沖無破 → 家宅平安
- 父母動 → 有遷動、修繕之兆
- 官鬼旺於宅卦 → 有怪異或疾病
- 玄武臨 → 防盜
- 白虎臨 → 有血光、傷災

十一、天時占
- 看坎水卦氣（雨）
- 看離火卦氣（晴）
- 看震卦（雷）、巽卦（風）
- 子孫旺發 → 天氣轉好
- 父母（雨）旺 → 雨多
- 妻財（晴）旺 → 晴

十二、農事占
- 用神：妻財（收成）
- 子孫：生財原神
- 兄弟動 → 損失、害蟲
- 父母動 → 雨水、文書（補助、文件）

十三、買賣占
- 用神：妻財（利潤）
- 應爻：對方（買家或賣家）
- 兄弟動 → 競爭、價格被壓
- 子孫旺 → 商品熱銷

十四、借貸占
- 借入：用神看自己（世），對方（應）
- 借出：用神看妻財（能否收回）
- 兄弟旺動 → 借出不還
- 妻財旺、不空 → 可成

十五、考試占
- 用神：父母（學業、文書）
- 兼看：官鬼（功名）
- 子孫動克父母 → 考試不利、心散
- 父母旺、官鬼相生 → 高中

十六、求職占
- 用神：官鬼（職位）
- 應爻：公司、面試官
- 父母：履歷、推薦信
- 妻財：薪水
判斷：
- 官鬼旺生世 → 工作來
- 應動克世 → 對方刁難
- 子孫動克官鬼 → 自己不想去、放棄
- 父母旺 → 文書到位

十七、合夥占
- 用神：妻財（生意）+ 應爻（夥伴）
- 兄弟旺動 → 拆夥、利益衝突
- 世應相合 → 合作順利
- 應動生世 → 對方有誠意

十八、孕產占
- 男方占妻孕：用神妻財（妻），子孫（孩子）
- 女方占自孕：用神子孫
- 子孫旺 → 孩子健康
- 子孫空、破 → 胎不穩，宜醫
- 男女判斷：子孫陽爻為男、陰爻為女（傳統說法，僅供參考）

用神特殊情況處理

(一) 用神出現兩個（或多個）
原則：
1. 若有動爻 → 取動爻為主
2. 若無動爻 → 取持世者
3. 若世應皆有 → 取應爻（外因）為動因，世爻為結果
4. 同類六親在不同位 → 看哪個與日月關係密切

(二) 用神不上卦（取伏神）
詳見 najia-guide.md 第六節。

(三) 用神兩現一伏
- 卦中已有用神，但本宮首卦對應位有更強的同六親
- 此情況罕見，以卦中為主

(四) 用神化用神（同六親）
- 動爻變化後仍是同類六親
- 化進神 = 加強
- 化退神 = 減弱
- 同支不變 = 伏吟

(五) 用神入墓
- 用神地支正好在動爻或日辰之墓庫
- 墓 = 隱藏、停滯、被收藏
- 應期：沖墓之日

格局判斷速記
旺相用神 + 日月生 + 動而化進 = 大吉
休囚用神 + 旬空 + 化退化破 = 大凶
用神持世 + 旺 + 動 = 自主有力
用神受克 + 化絕 + 入墓 = 事敗
卦逢六合 + 用神安穩 = 和諧成事
卦逢六沖 + 動爻多 = 動盪不安
反吟伏吟卦 = 反覆、停滯
游魂歸魂卦 = 心散、收尾、回歸
重要提醒
- 六爻判斷需綜合多項因素，不可單看一爻
- 不同流派對細節有分歧（如六獸起例、應期取捨）
- 本工具採主流《增刪卜易》《卜筮正宗》系統
- 結果僅供參考，重大決策請結合實際情況
