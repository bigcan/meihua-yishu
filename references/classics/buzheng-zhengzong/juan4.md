第九問：	用神不現，看伏神在何爻之下，得出不得出何以論之？

答曰：	伏神得出者有四：

蓋日月生者，日月持之者一也．

飛神生伏動爻生者二也．

日月動爻衝克飛神三也．

飛神空破休囚墓絕於日者四也．

伏神不得出者，亦有四：

休囚無氣者，日月克者一也．

飛神旺相日月生助，飛神克害伏神者二也．

伏神墓絕於日月及飛爻者三也．

伏神伏囚兼旬空月破者四也．

此四者乃無用之伏神，雖有如無，終不得出，凡用神旺相如遇旬空，出空之日則出矣．

休囚者待時而進二也．

動爻變爻有一而逄空破衝合者待期填補合衝而進三也．

退神之法亦有三：

旺相者或有日月動爻生扶，占近事暫時而不退者一也．

休囚者實時而退二也．

動爻變爻有一而逄空破衝合者，待期填補合衝而退三也．

### 第九問：用神不現之伏神斷法

**問**：用神不現於卦中，看伏神在何爻之下，能否得出？如何論之？
**答**：**伏神得出有四**：
1. **日月生伏 / 日月持伏**——伏神得日月之氣
2. **飛神生伏 / 動爻生伏**——飛神或他爻生扶
3. **日月或動爻衝克飛神**——飛神被破，伏神得透
4. **飛神空破休囚墓絕於日**——飛神自身無力，無法壓伏

**伏神不得出亦有四**：
1. **伏神休囚無氣 / 日月克伏**——伏神自身衰弱
2. **飛神旺相日月生助、又克害伏神**——飛神強壓
3. **伏神墓絕於日月飛爻**——伏神入死地
4. **伏神休囚兼旬空月破**——衰極又雙破

此四者乃無用之伏神，雖有如無；用神旺相遇旬空者，待出空之日即出。

### 第十問：進退神之斷

**問**：進退神為動爻變出之神，吉凶禍福有喜忌之分，如何論之？
**答**：**吉神宜於化進、忌神宜於化退**。進退之法各有三：

**化進三法**：
1. **旺相者乘勢而進**
2. **休囚者待時而進**
3. **逢空破衝合者**待期填補合衝而進

1. **旺相者** + 日月動爻生扶占近事 → 暫時不退
2. **休囚者**即時而退
3. **逢空破衝合者**待期填補合衝而退

### 第十六問：盡靜盡發之斷

**問**：盡靜、盡發如何斷？
**答**：六爻安靜無日衝動爻 = **盡靜**；六爻俱動 = **盡發**。盡靜如**春花含蕊**——人未見其妙，一沾雨露油然漸放；盡發如**百卉齊放**——人多見其艷，一遇狂風翻然而損。**故靜者恆美、動者常咎**。

### 第十八問：誠卜不驗之故

**問**：卜者誠心、斷者精明，亦有不驗者何也？
**答**：**病在卜者不在斷者**——卜者意雖誠，但**密事難以語人**、或**問此而意別有在**——所以致不驗。

**代表案 1**：占伯父何日回 = 火山旅 → 艮為山（離宮，戊申日，酉月，空亡寅卯，缺官、父）

| 九三 | 白虎 |   |   | 申金財 |       | 亥水官伏 |
| 六二 | 螣蛇 |   |   | 午火兄 |       |       |
| 初九 | 勾陳 | 世 |   | 辰土孫 |       | 卯木父伏 |

**斷曰譯**：此卦若問伯父平安否，卯木父伏不現、又被日月動爻克衝 → 必不安。但今問「**回來否**」非問平安——只可斷用神**伏藏受克不來**（未必死）。後果不來、在外平安——**設問口徑不同則斷法不同**。

**代表案 2**：占流年 = 艮為山（離宮錯標，實為艮宮，癸亥日，未月，空亡子丑）
- 卜者口稱「占流年」，實為往軍前求名而問

- **若以流年斷之則謬矣**——卜者之意不誠則斷者亦無從應

**補充案要旨**（餘 4 案）：
- 占家宅：應飛入宅臨父 → 必有外姓長者同居（彼曰：從無）；改問才知實為功名——「**家宅以鬼為禍害、功名以鬼為官星**」，兩者反斷；後業師中式而占者頭場貼出
- 占現任吉凶：子孫持世 → 占缺不得、占現任休官；經詢實占升遷 → 此缺不得，在任甚安
- 因母病占流年：表問流年實占母病；卦中財爻持世化午火生合本財運大發 → 應「**令堂甲辰日危**」（世爻辰土出旬之日）——果驗
- 占功名（但實為父叫子占）：父動克子 → 此子未戌日而亡（**真實占主非席間應對者**之深義）

**通則**：王洪緒此問為全書收尾——**最高層次的卦理不在於斷者技巧，而在於「設問即得卦」之機關**；卜者意若藏匿，斷者再精也應其表象而失其實。故占卜的本質是**問者意誠 + 斷者明理**之雙重結合。
