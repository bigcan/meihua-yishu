占卦法：

占卦時。首先將三个銅錢平入于手心﹐兩手合扣﹐問事之人要集中意念﹐腦子只準專想所要預測之事﹐反覆搖動手中銅錢﹐然後將銅錢擲于盤中﹐看銅錢的背和字的情況。

一个背﹐兩个字﹐稱作「單」。畫作「⚊」為少陽。

兩个背﹐一个字﹐稱作「拆」﹐畫作「⚋」為少陰。

三个背﹐沒有字﹐稱作「重」﹐畫作「○」為老陽﹐是變爻。

三个字﹐沒有背﹐稱作「交」﹐畫作「ㄨ」為老陰﹐是變爻。

共搖六次﹐第一次為初爻﹐畫在卦的最下面﹐依次上升﹐第六次為第六爻﹐畫在卦的最上𨘢。如遇有ㄨ、○﹐再畫出變卦來。

現搖一卦為例：

第一次搖得一背﹐畫作⚊；二次得兩背﹐畫作⚋；三次得三背﹐畫作「○」；四次得三字﹐畫作「ㄨ」；五次得三背﹐畫作「○」；六次得兩背﹐畫作⚋；畫法如下。

初學點卦時﹐見到一點﹐二點容易理解﹐見到○、ㄨ時﹐可能一時不明白符號的意義﹐須要細心、慢想。現以上卦為例﹐把前卦第三爻「○」改為「⚊」﹐成為離中虛卦﹐為內卦。把第四爻「ㄨ」改作「⚋」﹐第五爻「○」又改作「⚊」﹐成為坎中滿卦﹐為外卦。因為離卦為火﹐坎卦為水﹐所以此卦成為水火旣濟。因為點卦時﹐由下至上﹐排卦名時﹐是由上往下排﹐此卦上卦為坎﹐為水﹐下卦為離﹐為火﹐所以此卦卦名為水火旣濟卦。

友問：何以謂之子孫持世？

答：子孫與世字同在一爻者卽是子孫持世。倘得官鬼與世字同在一爻者卽是官鬼持世﹐其餘兄弟、妻財持世皆由此說。

或問：何謂六沖卦？答：六沖卦者﹐乾為天、坎為水、艮為山、震為雷、巽為風、離為火、坤為地、兌為澤﹐此八宮第一卦皆是六沖卦﹐再者天雷無妄、雷天大壯也是六沖卦一共十卦﹐其餘不是。

八卦各自對應一個家庭成員，這套對應源自《易經‧說卦傳》，是六爻體系的人倫底層：

| 卦 | 符號 | 家庭角色 |
|---|---|---|
| 乾 | ☰ | 父（一家之主） |
| 坤 | ☷ | 母 |
| 震 | ☳ | 長男（家中長子） |
| 巽 | ☴ | 長女 |
| 坎 | ☵ | 中男（次子） |
| 離 | ☲ | 中女（次女） |
| 艮 | ☶ | 少男（么子） |
| 兌 | ☱ | 少女（么女） |

**判讀原則**：陽卦（乾、震、坎、艮）為男性方位，陰卦（坤、巽、離、兌）為女性方位。「長／中／少」依爻位次序——震、巽以初爻為主，坎、離以中爻為主，艮、兌以上爻為主。
**實用價值**：占家宅、人事、走失、人物推測時，可由所得內外卦反推「家中哪一位」涉事。

卦象圖第二（六爻記號與占卦法）
==============================================================

一、八卦記憶口訣（取於宋人朱熹《周易本義》卦歌）

| 卦 | 符號 | 口訣 | 構成（從初爻到三爻） |
|---|---|---|---|
| 乾 | ☰ | 乾三連 | 單‧單‧單 |
| 坤 | ☷ | 坤六斷 | 拆‧拆‧拆 |
| 震 | ☳ | 震仰盂 | 單‧拆‧拆 |
| 艮 | ☶ | 艮覆碗 | 拆‧拆‧單 |
| 離 | ☲ | 離中虛 | 單‧拆‧單 |
| 坎 | ☵ | 坎中滿 | 拆‧單‧拆 |
| 兌 | ☱ | 兌上缺 | 單‧單‧拆 |
| 巽 | ☴ | 巽下斷 | 拆‧單‧單 |

「單」為一橫（陽爻 ⚊），「拆」為兩短（陰爻 ⚋）。

二、銅錢起卦法（三錢卜法）

擲三枚銅錢，依「字」（陰）與「背」（陽）的比例判讀：

| 銅錢組合 | 名稱 | 記號 | 性質 |
|---|---|---|---|
| 1 背 2 字 | 單 | ⚊ | 少陽（靜爻，不變） |
| 2 背 1 字 | 拆 | ⚋ | 少陰（靜爻，不變） |
| 3 背 0 字 | 重 | ○ | 老陽（動爻，變陰） |
| 0 背 3 字 | 交 | ㄨ | 老陰（動爻，變陽） |

共擲六次，第一次為初爻（最下），依序往上，第六次為上爻。遇到 ○ 或 ㄨ 時，要再畫「變卦」。

三、起卦範例

書中示範一卦：依次擲得「單、拆、重、交、重、拆」，得到：

```
   六爻 ⚋
   五爻 ○   ← 動，變 ⚋
   四爻 ㄨ   ← 動，變 ⚊
   三爻 ○   ← 動，變 ⚋
   二爻 ⚋
   初爻 ⚊
```

外卦（4-5-6 爻）為「坎中滿」（☵，坎為水）；內卦（1-2-3 爻）為「離中虛」（☲，離為火）。
**卦名**：以「上卦在前、下卦在後」報出 → 水火既濟。
**變卦**：把動爻的 ○ 變 ⚋、ㄨ 變 ⚊ → 重新讀卦名。本例變卦為「澤山咸」（外坎變兌、內離變艮）—— 但原書未明說，請以本工具實算為準。

八宮章第三（六十四卦的歸宮）
==============================================================

六十四卦不是平鋪的，而是分屬「八宮」，每宮 8 卦，由「八純卦」（乾坤震巽坎離艮兌）演變而來。**這是六爻體系最重要的結構**——它決定每個卦的「五行屬性」、用神不上卦時的「飛伏取法」、以及世應的位置。

一、八宮卦序

每宮的 8 卦演化規則：八純卦 → 一爻變（初爻）→ 二爻變 → 三爻變 → 四爻變 → 五爻變 → 四爻變回（遊魂卦）→ 內三爻全變（歸魂卦）。

| 宮 | 五行 | 八純卦 | 後 7 卦（依次：1爻變、2爻變、3爻變、4爻變、5爻變、遊魂、歸魂）|
|---|---|---|---|
| 乾 | 金 | 乾為天 | 天風姤、天山遁、天地否、風地觀、山地剝、火地晉、火天大有 |
| 坎 | 水 | 坎為水 | 水澤節、水雷屯、水火既濟、澤火革、雷火豐、地火明夷、地水師 |
| 艮 | 土 | 艮為山 | 山火賁、山天大畜、山澤損、火澤睽、天澤履、風澤中孚、風山漸 |
| 震 | 木 | 震為雷 | 雷地豫、雷水解、雷風恆、地風升、水風井、澤風大過、澤雷隨 |
| 巽 | 木 | 巽為風 | 風天小畜、風火家人、風雷益、天雷無妄、火雷噬嗑、山雷頤、山風蠱 |
| 離 | 火 | 離為火 | 火山旅、火風鼎、火水未濟、山水蒙、風水渙、天水訟、天火同人 |
| 坤 | 土 | 坤為地 | 地雷復、地澤臨、地天泰、雷天大壯、澤天夬、水天需、水地比 |
| 兌 | 金 | 兌為澤 | 澤水困、澤地萃、澤山咸、水山蹇、地山謙、雷山小過、雷澤歸妹 |

**判讀關鍵**：每個卦的「五行屬性」 = 它所屬宮的五行。例如「水火既濟」屬坎宮 → 五行屬水（不是看外卦「坎水」決定，而是看歸宮）。這直接影響「六親」如何裝（見第五章）。

二、八純卦的特殊地位

「八純卦」（乾、坎、艮、震、巽、離、坤、兌為本宮首卦）的特殊用法：
- 它們是各宮的「祖卦」，五行屬性最純
- 當其他卦的「用神不上卦」時，要回到本宮首卦借爻為「伏神」（v3.2-d 後將詳論）
- 對應本工具：[najia-guide.md](../../najia-guide.md) 第六節（飛伏神）

三、各宮的世應與納甲

八宮全圖是六爻最重要的查表——每卦的「世爻位置」「應爻位置」「六爻納甲（地支）」「六親（父母／官鬼／兄弟／妻財／子孫）」都已固定。本書 L142-502 已逐卦列出全圖。

世爻位置可由「卦在宮中的順位」推算：
| 順位 | 世爻位置 | 卦名（以乾宮為例）|
|---|---|---|
| 1（本宮首卦）| 第六爻 | 乾為天 |
| 2（1爻變）| 初爻 | 天風姤 |
| 3（2爻變）| 第二爻 | 天山遁 |
| 4（3爻變）| 第三爻 | 天地否 |
| 5（4爻變）| 第四爻 | 風地觀 |
| 6（5爻變）| 第五爻 | 山地剝 |
| 7（遊魂卦）| 第四爻 | 火地晉 |
| 8（歸魂卦）| 第三爻 | 火天大有 |

**應爻 = 世爻 + 3 爻**（隔兩爻），如世在第六則應在第三，世在初則應在四。

「納甲」是把天干地支「裝入」六爻的方法，源自漢代京房易學。每個卦的六爻都對應特定的地支（天干在簡化版本中通常省略，本工具以地支為主）。

六親歌章第五（六親的歸宮裝法）
==============================================================

「六親」（父母／官鬼／兄弟／妻財／子孫）的裝法不是看單爻的五行，而是**以該卦所屬宮的五行為「我」，再用五行生克判定**。

一、五大歸宮裝法口訣

| 歸宮（五行）| 兄弟（同我）| 父母（生我）| 妻財（我克）| 官鬼（克我）| 子孫（我生）|
|---|---|---|---|---|---|
| 乾兌宮（金）| 金 | 土 | 木 | 火 | 水 |
| 坎宮（水）| 水 | 金 | 火 | 土 | 木 |
| 坤艮宮（土）| 土 | 火 | 水 | 木 | 金 |
| 離宮（火）| 火 | 木 | 金 | 水 | 土 |
| 震巽宮（木）| 木 | 水 | 土 | 金 | 火 |

**核心邏輯（五行生克對應人倫）：**
- 生我者為「父母」（如金被土生 → 土為父母）
- 我生者為「子孫」
- 克我者為「官鬼」（克 = 管束、約束）
- 我克者為「妻財」（克 = 占有、驅使）
- 比和者為「兄弟」（同類）

二、實用範例：乾為天的六親

乾為天屬乾宮 → 五行屬金 → 「我」是金。
六爻地支（從初爻到上爻）：子水、寅木、辰土、午火、申金、戌土。

| 爻位 | 地支 | 五行 | 與「金」之關係 | 六親 |
|---|---|---|---|---|
| 上爻 | 戌土 | 土 | 生金 | 父母 |
| 五爻 | 申金 | 金 | 比和 | 兄弟 |
| 四爻 | 午火 | 火 | 克金 | 官鬼 |
| 三爻 | 辰土 | 土 | 生金 | 父母 |
| 二爻 | 寅木 | 木 | 金克木 | 妻財 |
| 初爻 | 子水 | 水 | 金生水 | 子孫 |

故乾為天：父母（戌、辰）、兄弟（申）、官鬼（午）、妻財（寅）、子孫（子）。

三、變卦的六親裝法（重要陷阱）

**「變卦的我，依本卦之我」**——這是後人按語的關鍵補正。
意思：變卦的六親不是依「變卦所屬宮的五行」來裝，而是繼續沿用本卦的「我」（本卦所屬宮的五行）。
例：澤天夬變為天風姤，雖然天風姤本屬乾宮，但因為夬卦屬坤宮（土），所以變卦的丑土、戌土仍按坤宮（土）算「比和」→ 標為「兄弟」，不是按乾宮（金）標為「父母」。

這對「化爻六親」的判斷至關重要（見動變章第七）。

世應章第六（自己與對方的位置）
==============================================================

「世爻」代表占卜者自己（或所占之事的主體），「應爻」代表對方（或外部環境）。每卦的世應位置固定，由「卦在宮中的順位」決定。

一、八純卦及其後 7 卦的世應位置（以乾宮為例）

| 卦 | 順位 | 世爻位置 | 應爻位置 |
|---|---|---|---|
| 乾為天 | 1（本宮首卦）| 六爻 | 三爻 |
| 天風姤 | 2（1爻變）| 初爻 | 四爻 |
| 天山遁 | 3（2爻變）| 二爻 | 五爻 |
| 天地否 | 4（3爻變）| 三爻 | 六爻 |
| 風地觀 | 5（4爻變）| 四爻 | 初爻 |
| 山地剝 | 6（5爻變）| 五爻 | 二爻 |
| 火地晉 | 7（遊魂卦）| 四爻 | 初爻 |
| 火天大有 | 8（歸魂卦）| 三爻 | 六爻 |

**核心規則：應爻 = 世爻 + 3 爻（隔兩爻）**。即世在初則應在四，世在二則應在五，世在三則應在六，世在四則應在初，世在五則應在二，世在六則應在三。

二、其他七宮的世應，照此規則推算

世爻位置只看「卦的順位」，與宮無關。
例：水火既濟（坎宮第三卦，2爻變）→ 世在第二爻；風澤中孚（艮宮第六卦，5爻變）→ 世在第五爻。

三、世應的占斷意義

- **自占**（占自己、占自身吉凶）：以世爻為用神
- **占他人**（占對方、外部、合作對象）：以應爻為用神
- **占婚姻、合夥、訴訟對手**：兼看世應（世為我方、應為他方）
- **世應相生**：雙方有緣／和諧
- **世應相沖**：雙方對立／不合
- **世應相合**：情投意合（亦防黏滯）

==============================================================
動變章第七（動爻與變爻的基本規則）
==============================================================

六爻有「動」與「靜」之分。靜爻不變化，動爻必變——這是六爻納甲體系與梅花易數的最大差異之一（梅花以「上下卦動」為主，六爻以「個別爻動」為主）。

一、動變的兩個方向

| 動爻記號 | 性質 | 變後 |
|---|---|---|
| ○（重，老陽）| 陽動 | 變陰（⚋） |
| ㄨ（交，老陰）| 陰動 | 變陽（⚊） |

二、單爻動的範例

- 乾卦初爻動：⚊⚊⚊ → ⚊⚊⚋ = 巽卦
- 乾卦初、上爻動：⚊⚊⚊ → ⚋⚊⚋ = 坎卦
- 坤卦中爻動：⚋⚋⚋ → ⚋⚊⚋ = 坎卦
- 坤卦三爻全動：⚋⚋⚋ → ⚊⚊⚊ = 乾卦

**判讀流程**：先把動爻變化，重新讀八卦記號 → 得變卦的內外卦 → 得變卦卦名。

三、完整裝卦範例：水天需 變 天水訟

```
   妻財子水 ㄨ   ← 動    兄弟戌土 ⚊
   兄弟戌土 ⚊            子孫申金 ⚊
   子孫申金 ㄨ 世 ← 動   父母午火 ⚊
   兄弟辰土 ○   ← 動    父母午火 ⚋
   官鬼寅木 ⚊            兄弟辰土 ⚊
   妻財子水 ○ 應 ← 動    官鬼寅木 ⚋
```

四、變爻六親的標註規則（重要）

**「變出之爻安六親者，仍照正卦而推」**——這是本章最重要的一句。
意思：變爻的六親標註，不是依變卦所屬宮重算，而是繼續沿用本卦的「我」。
為什麼？因為占卜的「主體」是本卦（你問的當下時點），變卦只是「事態發展」，所以六親身份要鎖定在本卦的視角。

實例驗證（水天需屬坤宮土，「我」為土）：
- 子水（變後仍是水）→ 對土而言是「我克之水」→ 妻財 ✓
- 戌土（變後）→ 對土而言是「比和」→ 兄弟 ✓
- 申金（變後）→ 對土而言是「我生之金」→ 子孫 ✓
- 午火（變後）→ 對土而言是「生我之火」→ 父母 ✓
- 辰土（變後）→ 比和 → 兄弟 ✓
- 寅木（變後）→ 對土而言是「克我之木」→ 官鬼 ✓

五、靜爻與動爻的占斷意義

| 狀態 | 性質 | 占斷暗示 |
|---|---|---|
| 用神靜旺 | 待時 | 事在進行中，待應期 |
| 用神靜衰 | 蟄伏 | 事勢未明 |
| 用神動旺 | 進取 | 事勢主動發展 |
| 用神動衰 | 妄動 | 動而不應 |
| 動而化進神 | 強化 | 越動越強 |
| 動而化退神 | 衰退 | 動意已弱 |
| 動而化回頭克／生 | 自食苦果／得救 | 動爻本身被變爻反作用 |
| 動而化空、化破 | 延後／落空 | 動不為空，待沖空 |

用神章第八（核心：五類用神對應）
==============================================================

本章建立六爻納甲體系最重要的對應規則：問什麼事 → 取哪一爻為「用神」。
野鶴的歸納非常實用：先看「我關心的對象與我的關係」，再對應到「父母／官鬼／兄弟／妻財／子孫」五類六親之一。物類無窮但可變通，原則是抓住「該六親在卦中所代表的核心意義」。

一、父母爻為用神（庇護我身的一切）

人：祖父母、伯、叔、姑、姨、父母，凡是輩分在我之上，或與我父母同輩的親屬、師長、妻方父母、乳母養父；以及「三父八母」（古制中的各種父母輩稱呼，包括嫡母、繼母、養母等）；僕人占主人時也以父母爻為用神。
物：天地、城池、牆垣、住宅、屋宇、舟車、衣服、雨具、綢緞、布匹、章奏、文書、書館（學校）的文契。
通則：一切「庇護我身」的人事物，都歸父母爻。物類繁多，可依此原則靈活推類。

二、官鬼爻為用神（拘束我身的一切）

人事：功名、官府、雷霆鬼神（自然界威猛之力）、妻占夫。
凶類：亂臣、賊盜、邪祟（原文作「邪崇」，當為「邪祟」之誤）。
通則：一切「拘束我身、令我受制」的人事物，都歸官鬼爻。物類也很多，原則同上。

三、兄弟爻為用神（同類競爭者）

人：自己的兄弟、姐妹、同族中的兄弟、姑姨、妹夫，以及結拜兄弟。
角色定位：兄弟是「同類之人」——得志時會欺凌我，見財時會奪取，因此：
- 占財物時，兄弟爻是「劫神」（劫財）
- 占謀事時，兄弟爻是「阻神」（阻礙）
- 占妻妾婢僕時，兄弟爻是「刑傷克害之神」（克妻財）
特例：占姊夫、妹夫的事，野鶴主張改用「世爻」為用神，他自己屢試屢驗。

四、妻財爻為用神（我所驅使、所擁有的）

人：妻、妾、婢、僕、役——凡是我所驅使的人。
物：貨財、珠寶、倉庫，以及一切日常使用的財物。
原文「以要財爻為用神」中的「要」字當為「妻」之誤。

五、子孫爻為用神（晚輩、福德、解憂）

人：子、女、女婿、侄、甥、門生，凡屬我子孫輩的人。
特殊類別：忠臣、良將、醫藥、僧、道、兵卒（這些都是「為民解難」的角色，故歸子孫）。
動物：六畜、禽獸。
核心性質：子孫為「福德之神」——
- 是「制鬼之神」（剋官鬼，故為避災解禍之鑰）
- 是「解憂之神」（憂愁因子孫動而解）
- 是「剝官之神」（剋掉官星）
吉凶傾向：諸事見子孫為喜，唯獨占功名者忌之（因子孫剋官鬼，求名反而落空）。

用神元神忌神仇神章第九（四神關係與生克鏈）
==============================================================

本章建立六爻體系的「四神」框架：用神是主角，元神是助力（生用神），忌神是敵人（克用神），仇神是敵人的助手（生忌神並克元神）。野鶴透過一個具體醫案說明：用神兩現以上時要擇旺者用，且還要看其他爻如何透過「合住」「沖去」改變局勢。

一、四神定義

- 用神：即前章所定的「占某事取某六親」之爻。
- 元神：生用神之爻（如用神為土，則生土之火爻即元神）。
- 忌神：克用神之爻（如用神為土，則克土之木爻即忌神）。
- 仇神：克元神而生忌神之爻（如元神為火、忌神為木，則克火生木之水爻即仇神）。

四者形成一條五行生克鏈：仇神 → 忌神 → 用神 ← 元神 ← 仇神之被克者。
判讀重點是「這條鏈的力量分佈」：用神身邊是助力多還是阻力多。

二、實例詳解：辰月戊申日，占父近病

卦象：乾為天 變 風天小畜（六沖卦）

```
   父母戌土⚊ 世   妻財卯木⚊
   兄弟申金⚊       官鬼巳火⚊
   官鬼午火 ○      父母未土⚋     ← 午火動，化出未土
   父母辰土⚊ 應   父母辰土⚊
   妻財寅木⚊       妻財寅木⚊
   子孫子水⚊       子孫子水⚊
```

求問者一上來說：「近病逢沖即愈，這卦六爻相沖，但父親病得很重，到底何時會好？」

野鶴的分析步驟（六爻斷卦的標準流程）：

**步驟 1：定用神 ——「用神兩現以上時擇旺者」**
卦中有三重父母爻（戌土、未土、辰土），不能全用，要選最旺的一個。
辰月戊申日當令——辰土父母「臨月建」（即「值月」），最旺。
∴ 取辰土父母為用神。

**步驟 2：找忌神 ——「目下病重」的原因**
寅木克辰土，寅木是忌神。
寅木雖然原本安靜，但戊申日「申沖寅」——靜爻被日辰沖即為「暗動」。
∴ 寅木暗動克辰土，這是當下父親病重的直接原因。

**步驟 3：審視潛在救應 —— 午火動是否能生用神？**
求問者反問：「卦中午火發動，寅木雖然暗動，午火可以洩寅生土啊，怎會克得了辰土？」
野鶴指出陷阱：午火動，但動而「化出未土」（即動爻自己變出未土），午與未本是六合（地支六合中的「午未合」）。
午火被未土合住 → 動而被羈絆 → 不再生辰土。
∴ 辰土失去元神之助，孤受寅木之克 → 病重。

**步驟 4：推應期 —— 什麼時候會好？**
要解這個僵局，必須打破「午未合」。
丑日「丑沖未」——沖去未土後，午火脫絆而恢復「動」的生力 → 辰土得生而病退。
∴ 應丑日起牀。後果驗。

**此例的教學重點：**
1. 用神「兩現」要擇旺，不能機械地都看。
2. 「動爻被合住」是動而不應的特殊情況——這在現代工具中對應到 najia-guide.md 第十節「合處逢沖、沖中逢合」的判斷。
3. 應期不是看用神本身，而是看「打破局勢」的關鍵點（此例是沖去合神，讓元神復活）。
4. 反駁「近病逢沖即愈」的機械化口訣——六沖卦也要看具體用神受不受克。

三、原劍按（後人補注）

後人補注解釋「仇神」之名的由來：
- 生我者為元神，克我者為忌神。
- 仇神之所以叫「仇」，是因為它對「我所克」的東西反過來成為仇敵。
- 以木為例：木生於水（水為元神），克於金（金為忌神），而「克水生金」之物為土（仇神）。本來木是克土的，但因為土反過來生金（忌神）並克水（元神），所以五行循環中土反成木之大敵。

這段補注解釋了「仇神」這個名稱的邏輯。實際應用中只需記：克元神 + 生忌神 = 仇神。

元神忌神衰旺章第十（四神的有力／無力總辨）
==============================================================

本章是第九章「四神定義」的力學版：用神身邊的元神、忌神，不能光看「在不在卦中」，要看「有沒有力」。野鶴給出 5+6+5+7 共 23 條判定，再用一個自占病的醫案作反證——即使「忌神與元神同動」（七條無力忌神之第七，照理是化凶為吉），若用神本身月破日克、根本無根，元神再有力也救不了。
這是六爻體系最重要的「力學鏈條」：用神有根 → 元神有力 → 才能真正生扶。三者缺一不可。

一、元神能生用神者有五（有力之元神）

1. **元神旺相，或臨日月，或日月動爻生扶**
   元神得月令／日辰扶持，或被卦中其他動爻生（接力相生），力量充足，能生用神。

2. **元神動，化回頭生，或化進神**
   元神是動爻，且變爻反過來生它（回頭生），或變爻是進神（如寅變卯、申變酉之類）—— 元神不但動，還越動越強。

3. **元神長生、帝旺於日辰**
   日辰是元神的長生或帝旺位（十二長生法），如金的元神為土，土長生在申、帝旺在午——遇申日或午日，土元神得地利。

4. **元神與忌神同動（貪生忘克）**
   忌神動本來要克用神，但若元神同時動，忌神「貪戀」生元神（同類相親，生比克近），忘了克用神 → 反而成為用神的助力。這就是「貪生忘克」的經典格局。

5. **元神旺動，臨空或化空**
   元神旺而動，雖然臨旬空、或變爻是旬空 —— 看似落空，但「動不為空」（野鶴反覆強調的鐵則）：動爻只是延遲生效，待沖空（衝空爻使其發力）或實空（出旬之日）即有用。

二、野鶴破「動而臨空為無用」之謬

當時流行的說法：「動爻臨空、化空為無用。」
野鶴駁斥：這是錯的。
正解：動爻不算空，只是「應期延後」——待沖空之日（如旬空爻為寅，遇申日沖寅，激發其用）或實空之日（寅木出旬，自然生效），便顯吉用。
以上五條為「有力之元神」，諸占皆吉。

三、實例一：酉月辛亥日占求財，得 兌為澤 變 雷水解

卦象（從上爻到初爻）：

```
   父母未土⚋ 世   父母戌土⚋
   兄弟酉金 ○      兄弟申金⚋     ← 動
   子孫亥水⚊       官鬼午火⚊
   父母丑土⚋ 應   官鬼午火⚋
   妻財卯木⚊       父母辰土⚊
   官鬼巳火 ○      妻財寅木⚋     ← 動，化出寅木妻財
```

【背景】辛亥日為甲辰旬，旬空為寅、卯。
求財取妻財為用神 → 妻財為卯木（在第二爻，靜），且「月破又旬空」：酉月沖卯為月破，卯又值旬空。

求問者反駁野鶴的判斷：「寅日得財」？怎麼可能：
- 用神卯木月破又旬空，又被五爻動的酉金兄弟所克
- 初爻巳火官鬼動雖然生世（火生未土），但亥日沖巳（沖散）
- 巳火動還化出寅木——寅木又是旬空
這還能算吉嗎？

野鶴的回答：
- **「神兆機于動」**：神靈的徵兆顯露在動爻上 —— 我從來不接受「動而被沖叫散」這套話。動爻只會延後應期，不會作廢。
- 巳火動「化空」（變爻寅木旬空）→ 目下不顯，要等寅木出空。
- 應期分析：甲寅日（出旬）→ 寅木「實空」生效 → 寅木生巳火、巳火生未土世爻 → 透過「財→官→世」的接力生鏈，得財。
- 應驗：寅日早晨見木，當日得財。

**此例的教學重點：**
1. 妻財自身月破旬空，看似絕望，但卦中存在「變出之妻財」（寅木由巳火化出）——這是「飛伏／變出」邏輯。
2. 卯木靜爻被沖被克時，可改循「動爻財官生世」的接力路徑。
3. 應期落在「沖空 / 出空」的關鍵日 —— 寅日讓寅木出空、生官、官生世，三步銜接完成。
4. 此案應對的是元神五條件之第五：「旺動臨空化空」—— 巳火動而所化之爻（寅）為空，待出空便有用。

四、元神雖現又不能生用神者有六（無力之元神）

1. **元神休囚，不動，或動而休囚又被傷克**
   元神本身衰弱，又無動力或反被克 → 無力。

2. **元神休囚，又逢旬空、月破**
   元神已弱，再加旬空或月破，雙重打擊。

3. **元神休囚，動而化退神**
   元神衰，動爻所化為退神（如卯變寅、酉變申）—— 越動越弱。

4. **元神衰而又絕**
   元神衰，又值「絕」於日辰（十二長生之絕位）。

5. **元神入三墓**
   元神被「墓」收住（日墓、動墓、化墓三種）—— 動彈不得。

6. **元神休囚，動而化絕、化克、化破、化散**
   元神弱，動而所化之爻為絕／克／破／散 —— 自取滅亡。

以上六條：元神雖然「現」（在卦中有位置），但完全無力生用神 —— 「雖有如無」。

五、忌神動而克害用神者有五（有力之忌神）

1. **忌神旺相，或日月動爻生扶，或臨日月**
   忌神得月令／日辰／動爻三方加持。

2. **忌神動，化回頭生，或化進神**
   （原文「動化回頭生扶退神」應為 OCR 訛誤，對照元神同條應為「化進神」。正確意義：忌神動而被回頭生、或變為進神，越動越強。）

3. **忌神旺動，臨空或化空**
   忌神旺而動，雖臨空或化空 —— 動不為空，待沖空實空之日發力為害。

4. **忌神長生、帝旺於日辰**
   日辰是忌神的長生／帝旺位。

5. **忌神與仇神同動**
   仇神生忌神，雙重攻擊用神。

以上五條：忌神「如斧戟」（如刀斧戟矛之利器），諸占大凶。

六、忌神雖動，不能克用神者有七（無力之忌神）

1. **忌神休囚不動，或動而休囚又被日月動爻克**
2. **忌神靜，臨旬空、月破**
3. **忌神入三墓**
4. **忌神衰，動而化退神**
5. **忌神衰而又絕**
6. **忌神動，化絕、化破、化克、化散**
7. **忌神與元神同動（貪生忘克）**
   忌神反過來生元神，元神再生用神 —— 接續相生，化凶為吉。

以上七條：忌神「無力」，諸占化凶為吉。

七、野鶴的總結警告（重中之重）

> 「以上論元神、忌神之有力、無力者，亦要用神有气，倘若用神無根，謂之元神有力亦難生，忌神無力何足喜。」

翻譯：以上 5+6+5+7 共 23 條，都是在「用神本身有根」的前提下才有意義。如果用神自己月破日克、根本無根，那麼：
- 元神再有力，也生不起這個無根的用神
- 忌神再無力，也沒什麼好高興的（反正用神自己已經垮了）

**這一句是整章的鎖鑰**：力學鏈條是「用神有根 → 元神有力 → 才有效」，不能跳過第一步直接看第二步。

八、實例二（反證）：巳月乙未日自占病，得 澤風大過 變 火風鼎

卦象（從上爻到初爻）：

```
   妻財未土 ㄨ      子孫巳火⚊       ← 動，化出巳火
   官鬼酉金 ○      妻財未土⚋       ← 動
   父母亥水⚊ 世   官鬼酉金⚊
   官鬼酉金⚊       官鬼酉金⚊
   父母亥水⚊       父母亥水⚊
   妻財丑土⚋ 應   妻財丑土⚋
```

【背景】自占病 → 用神為世爻亥水。

**表面分析（看似化凶為吉）：**
- 忌神：未土（克亥水），動爻
- 元神：酉金（生亥水），也是動爻
- 未土動 → 反過來生酉金（土生金，貪生忘克）→ 酉金 → 亥水
- 接續相生，完美對應「忌神與元神同動」（忌神無力第七條）→ 應化凶為吉

**但野鶴指出致命破綻：**
- 亥水世爻「月沖」（巳月沖亥）→ 月破
- 「日克」（乙未日，未克亥）→ 雙重打擊
- 用神本身月破日克 = 「無根」
- 雖有元神酉金生扶 → 「生之不起，如樹無根，寒谷不回春」

**結果：** 卒於癸卯日。
**應期分析：** 卯日 沖去 元神酉金（卯酉相沖）→ 失去最後的生扶 → 應卯日。

**此例呼應第七節結語：**
- 滿足「忌神與元神同動」（化凶為吉）的條件，但因為用神無根，整個救應鏈條失效。
- 應期不是看用神自身（亥水），而是看「沖去元神」的關鍵日（卯沖酉）。
- 對照第九章醫案：第九章是「沖去合神而活元神」（救應成功），本案是「沖去元神而失救應」（救應失敗）—— 兩案對稱，是六爻應期推算的兩個方向。

**為什麼這個案例放在第十章而不是第十一、十二章？**
因為它證明「力學鏈條的順序」：用神根 > 元神力 > 忌神勢。
這是全章的壓軸 —— 不是給你 23 條口訣就完事，而是強調 23 條前面那個「用神有根」的前提。

五行相生章第十一（生扶的四種來源）
==============================================================

第十章建立了「用神有力／無力」的力學鏈條後，第十一章開始拆解「力」的來源 —— 哪些力可以稱為「生」？野鶴給出四種，並用一個救人於死刑的醫案說明「動而化回頭生」可以瞬間扭轉局勢。

一、五行相生口訣

金生水、水生木、木生火、火生土、土生金 —— 循環不息。

二、用神、元神宜「逢生」的四種來源

凡用神、元神宜於逢生，生的來源有四：

1. **月建生**：所占月份的地支生用神／元神（如寅卯月生巳火）
2. **日建生**：所占日辰的地支生用神／元神（如卯日生午火）
3. **動爻生**：卦中其他動爻生用神／元神
4. **動化回頭生**：用神／元神本身是動爻，變爻反過來生它

這四種「生」中，第 4 種「動化回頭生」最為決定性 —— 因為它是用神／元神自己變出來的力量，等於「自食其果」的反向版（自食其福）。

三、實例：卯月己卯日，弟占兄已得重罪，叩問能救否，得 復卦 變 震卦

卦象（從上爻到初爻）：

```
   子孫酉金 ⚋        兄弟戌土 ⚋
   妻財亥水 ⚋        子孫申金 ⚋
   兄弟丑土 ㄨ 應 ← 動  父母午火 ⚊
   兄弟辰土 ⚋        兄弟辰土 ⚋
   官鬼寅木 ⚋        官鬼寅木 ⚋
   妻財子水 ⚊ 世    妻財子水 ⚊
```

野鶴的分析步驟：

**步驟 1：定用神** —— 占兄事，以兄弟爻為用神。卦中兄弟兩現（丑土、辰土），擇動者用 → 取丑土兄爻為用神。

**步驟 2：審用神受力情況**
- 卯月 → 月建卯木克丑土
- 己卯日 → 日建卯木克丑土
- 月日皆克 → 重罪難脫，似乎必死。

**步驟 3：找救應 ——「動化回頭生」**
丑土動，化出午火。午火為父母（離宮對應），火生土 → **回頭生丑土**。
父母動爻來生兄爻 → 解卦關鍵：化爻反過來救回頭土。

**步驟 4：斷與應期**
野鶴斷：「速速行之。父母化兄爻相生為吉。」
應驗：後蒙恩免死。

**此例的教學重點：**
1. 月、日同克時，看似必凶，但「化回頭生」的爆發力可以扭轉乾坤。
2. 化爻的「父母」對應到實世界：往往是「年高者的恩免」「文書批示的救應」「上級的庇護」—— 本案是「援例恩留免死」（後人按語暗示）。
3. 應期推算未明說，但「父母動」通常應在「父母當令之日」或「化父母出旬之日」。

五行相克章第十二（克害的四種來源與聖人作易的提醒）
==============================================================

第十一章談「生」，第十二章談「克」—— 兩章對稱。野鶴用兩個同日的訴訟案，一案至重刑、一案克處逢生免死，演示「克」與「救」的辨識。

一、五行相克口訣

金克木、木克土、土克水、水克火、火克金。

二、忌神、仇神宜「逢克」的四種來源

凡忌神、仇神宜於逢克，克的來源有四：

1. **月克**：月建克用神／元神（對忌仇而言，月建克忌神則為救應）
2. **日克**：日建克用神／元神
3. **動爻克**：卦中其他動爻來克
4. **化回頭克**：本爻動，變爻反過來克它

**判讀規則**：
- 用神、元神若**逢一克且他處無生扶** → 凶兆
- 占吉事則「樂極生悲」，占凶事則「急急回避」

三、實例一：卯月戊寅日，占父官事已擬重罪，得 澤地萃 變 天火同人

卦象：

```
   父母未土 ㄨ      父母戌土 ⚊
   兄弟酉金 ⚊ 應   兄弟申金 ⚊
   子孫亥水 ⚊      官鬼午火 ⚊
   妻財卯木 ㄨ      子孫亥水 ⚊
   官鬼巳火 ⚋ 世   父母丑土 ⚋
   父母未土 ㄨ      妻財卯木 ⚊
```

**分析：**
- 占父 → 用神為父母爻（卦中外卦六爻未土、初爻未土兩現）
- 外卦未土父母：卯月（月建）克之 = **月克**
- 內卦有亥、卯、未三爻 → 「亥卯未三合木局」→ 木局克土 = **三合動爻克**
- 月克 + 日刑 + 三合克 → 全無助力
- **結果**：父果至重刑。

**為何如此凶？** 用神「全無生扶」——這完全對應第十章的「忌神有力第一條：月日生扶」+ 用神反過來自身無根。沒有元神救應的克勢，必凶。

四、實例二：同日妹占兄官事（同案），得 天地否 變 天水訟

```
   父母戌土 ⚊ 應   父母戌土 ⚊
   兄弟申金 ⚊       兄弟申金 ⚊
   官鬼午火 ⚊       官鬼午火 ⚊
   妻財卯木 ⚋ 世   官鬼午火 ⚋
   官鬼巳火 ㄨ      父母辰土 ⚊
   父母未土 ⚋       妻財寅木 ⚋
```

**分析：**
- 占兄 → 申金兄爻為用神
- 巳火官鬼動 → 巳刑克申金（巳申相刑，火克金）= 內動爻克
- 看似重罪定矣，但⋯
- **幸喜辰日沖動戌土**（暗動）→ 戌土父母被沖而暗動 → 土生金（生申）
- **克處逢生** —— 申金被刑克，但戌土暗動來生救之
- **後因父年八十，援例恩留免死**

**對比兩案：** 同日兩占同一案，前者父母用神被月、日、三合三方圍克，無救；後者兄爻用神有「日辰沖戌而暗動相生」這一線生機 → 結局完全不同。
這是「克處逢生」的經典範例 —— 也是第十三章的引子。

五、聖人作易的訓誡（本章末段）

> 「聖人作易，令人趨吉避凶。未卜而不知者，是大數也；卜之而神不告，亦大數也；既已卜之，神已先告之，明知故犯者，亦不可盡委之于數也、命也。」

**翻譯：** 聖人創《易》是為了讓人趨吉避凶。沒占卜而不知吉凶，那是天數使然；占了卜但神靈不肯告知，也是天數。但是 —— 已經占了卜、神靈已經明明白白告訴你了，還明知故犯，這就不能全推給「天命」「氣數」了，是自找的。

**此段的價值：** 是對占卜倫理的重要表態 —— 卜易不是宿命論，而是「告之以吉凶後仍有選擇」。對應本工具 [SKILL.md](../../../SKILL.md) 的倫理基調：占卜給的是判斷依據，不是免責的命運辯詞。

克處逢生章第十三（生克對抗的力量比較學）
==============================================================

本章是第十一、十二章的綜合 —— 當用神同時受到「克」和「生」時，誰勝？野鶴給出簡明判準，並用一個癌症臨危案說明：即使月克、日不生，只要動爻接力相生，仍可絕處逢生。

一、克處逢生的定義

「受此處之克，得他處之生」即為克處逢生。
類似的還有「絕處逢生」（用神絕於日，但動爻來生）—— 兩者邏輯相同。

二、占斷判準

| 對象 | 宜 | 忌 |
|---|---|---|
| 用神、元神 | 克少生多（為吉）| 克多生少 |
| 忌神 | 克多生少（為吉）| 克少生多 |

**核心邏輯**：忌神是敵人，敵人多被克才好。所以忌神「宜克不宜生」 —— 你不希望忌神被生扶。

三、實例：辰月丙申日，占弟癌癥業已臨危，得 既濟 變 革卦

卦象：

```
   兄弟子水 ⚋ 應   官鬼未土 ⚋
   官鬼戌土 ⚊       父母酉金 ⚊
   父母申金 ㄨ      兄弟亥水 ⚊      ← 動
   兄弟亥水 ⚊ 世   兄弟亥水 ⚊
   官鬼丑土 ⚋       官鬼丑土 ⚋
   子孫卯木 ⚊       子孫卯木 ⚊
```

**分析：**
- 占弟 → 用神為兄弟爻（卦中亥水兄、子水兄兩現），擇旺者 → 取**世爻亥水**為用神（自占／代占之主體爻）。
- **克**：辰月（月建辰土）克亥水 = 月克 → 凶
- **生**：丙申日 → 申金生亥水 = 日生 → 救
- **動爻接力生**：四爻申金父動 → 申金生亥水 = 動爻生
- 月克 ↔ 日生 + 動爻生 = **克一生二**（克少生多）→ 吉

**斷與應期：**
- 野鶴斷：「臨危有救」
- 應驗：當日酉時遇名醫救活（酉時 = 申日酉時，三合金局成形之時）；至己亥日（亥日 = 用神本氣值日）全生。

**此例的力量學習：**
1. 月建克的份量大，但「日 + 動爻」雙重生扶可以抵銷一個月克。
2. 應期不是看克的消解，而是看「生扶力到位」的時點（酉時、亥日）。
3. 「臨危有救」≠ 「立即無事」—— 酉時遇醫只是轉危為安，到亥日才全生。
4. 對應第十章「忌神無力第一條：休囚不動或動而被克」—— 本案的克勢（辰月）雖在，但卦中也無強動的忌神接力，給了生扶喘息的空間。

四、後人按語

> 「本質上就是對比沖克與生合的力量的彊弱。」

這句話點出全章的數學本質：**六爻斷卦是力量的代數和**，而不是「逢克即凶」的二元判斷。

==============================================================
動靜生克章第十四（動爻 vs 靜爻的力量規則）
==============================================================

第十一至十三章談「生與克」，第十四章新增一個維度：**動 vs 靜**。同樣是「克」，動爻克 vs 靜爻克的效力完全不同。這是六爻體系最容易誤判的環節之一。

一、核心規則

> 「六爻安靜旺相之爻，可以生得休囚之爻，亦可以克得休囚之爻 —— 蓋旺相者有力之人也。」

**翻譯：** 卦中若全為靜爻，那麼**旺相的靜爻**可以生扶或克制衰弱的靜爻（旺者有力，可主動發揮作用）。

**反推：** 衰弱的靜爻不能克制旺相的靜爻 —— 力不足以及人。

二、實例一（靜爻情境）：春天（寅卯月）占得坤卦

```
   子孫酉金 ⚋ 世
   妻財亥水 ⚋
   兄弟丑土 ⚋
   官鬼卯木 ⚋ 應
   父母巳火 ⚋
   兄弟未土 ⚋
```

**各爻力量分析（春天當令為木）：**
- 卯木官鬼：**當令旺相**
- 巳火父母：木生火 → 相
- 酉金子孫、亥水妻財：水金休囚
- 丑未兄弟（土）：木克土 → 休囚無氣

**分占判斷：**
- 占父母：卯木旺生巳火 → 父爻旺；父爻旺反過來可克子孫 → 子孫衰
- 占子孫：受父克，**子孫衰矣**
- 占兄弟：丑未二土被當令之卯木所克 → **休囚無氣**

**規則應用：** 全靜卦中，旺爻自然發揮作用，不需「動」這個觸發。

三、實例二（動爻翻盤）：春天占得 兌卦 變 歸妹卦

```
   父母未土 ⚋ 世   父母戌土 ⚋
   兄弟酉金 ○       兄弟申金 ⚋      ← 動
   子孫亥水 ⚊       官鬼午火 ⚊
   父母丑土 ⚋ 應   父母丑土 ⚋
   妻財卯木 ⚊       妻財卯木 ⚊
   官鬼巳火 ⚊       官鬼巳火 ⚊
```

**力量分析：**
- 春天 → 卯木旺、土休囚
- **酉金兄弟動**：金本是休囚（春木旺，金被克），但「動」這個動作賦予它「主動克」的能力
- 卯木妻財：當令旺相

**規則突破：** 一般情況「衰金不能克旺木」，但**動爻打破靜態的力量計算** —— 酉金雖休囚，動則能克卯木。
所以野鶴說：「酉金發動雖則休囚，動而能克旺相之卯木。」

**連鎖反應：** 卯木既被酉金所傷，原本能克丑未二土的力量也削弱了 → 「亦難克土」。
這是一個動爻引發的「制衡鏈反轉」：衰金動 → 制旺木 → 救衰土。

四、總結：動與靜的力學差異

| 狀態 | 對他爻的影響 |
|---|---|
| 旺相靜爻 | 可生可克（自然發揮作用）|
| 休囚靜爻 | 無力主動發揮（被動受作用）|
| 旺相動爻 | 全力生克 |
| **休囚動爻** | **動而能克／生**（雖力弱，但動作本身具觸發力）|

**這條規則是「無力動爻」斷卦的關鍵** —— 不能因為某爻休囚就斷定它無作用，要看它是否動。

==============================================================
動變生克沖合章第十五（變爻的特殊性與日月的超然地位）
==============================================================

第十四章談動靜，第十五章拆解「動爻與變爻」的特殊關係 —— 變爻只能影響它對應的本位動爻，不能跟其他爻互動。這是六爻體系最重要的「邊界規則」，誤用會導致完全錯誤的斷卦。

一、變爻的兩條鐵則

1. **變爻能生克沖合本位的動爻**（變爻 ↔ 動爻）
2. **變爻不能生克他爻**（不影響其他靜爻、不影響其他動爻）

反之亦然：
3. **他爻也不能生克變爻**（其他爻動而不動，都碰不到變爻）

**為什麼？** 因為變爻是「本位動爻的內部變化」—— 它是動爻的「未來態」，還沒有真正進入卦的主場。

二、實例：子月卯日，占得 坤卦 變 火地晉

```
   子孫酉金 ㄨ 世   父母巳火 ⚊      ← 動，化巳火
   妻財亥水 ⚋       兄弟未土 ⚋
   兄弟丑土 ㄨ      子孫酉金 ⚊      ← 動，化酉金
   官鬼卯木 ⚋ 應   官鬼卯木 ⚋
   父母巳火 ⚋       父母巳火 ⚋
   兄弟未土 ⚋       兄弟未土 ⚋
```

**動爻 ↔ 變爻關係（合法的影響）：**
- 世爻酉金動 → 變出巳火：火克金 = **變爻回頭克本位動爻**（酉金被巳火回頭克）
- 四爻丑土動 → 變出酉金：土生金 = 變爻回頭生本位動爻

**不合法的影響（鐵則禁止）：**
- 四爻丑土動 雖然動，但**不能生變出的巳火**（變爻是世爻的，不歸丑土管）
- 變出的酉金（四爻變爻）**不能生克他爻**（不能去生本位的世爻或亥水）

**例外：** 唯有「日辰、月建」能跨越這條鐵則 —— 它們能生克動爻、靜爻、變爻、飛爻、伏爻一切爻。
- 子月（月建子水）克世爻變出的巳火（水克火）→ 月建可影響變爻 ✓
- 卯日（日建卯木）沖變出的酉金（卯酉相沖）→ 日建可沖變爻 ✓

三、黃金策的名言：「爻傷日月，徒受其名」

> 「日月如天，能生克動爻、靜爻、變爻、飛爻、伏爻；而諸爻不能傷日月。黃金策曰：『爻傷日月，徒受其名。』」

**翻譯：** 日辰、月建就像天 —— 能影響卦中所有的爻；反之，卦中的爻不能影響日月。
即使卦中有一爻克日月、或合住日月，那只是「名義上的關係」（徒受其名），實際上日月不受任何影響，繼續履行它們的判官職能。

**這句話的價值：** 確立了六爻斷卦的「權力等級」：

```
日辰、月建（神靈級，不受爻影響）
   ↓
卦中諸爻（動爻、靜爻、變爻、飛爻、伏爻）
   ↓
彼此按鐵則互動（變爻 ↔ 本位動爻、靜爻間接受旺相影響等）
```

四、本章的判讀流程（總結用）

斷一卦時，應依以下順序考察「力量」：

1. **看日月對用神的影響**（最高優先級：生／克／沖／合／日月臨／長生帝旺）
2. **看用神是否動**（動有變、靜待時）
3. **看變爻對本位動爻的回頭生克**（用神為動爻時必查）
4. **看卦中其他動爻對用神的影響**（透過生克的接力鏈）
5. **看靜爻間的旺衰互動**（旺相靜爻可生克休囚靜爻）

**絕對禁止：** 用「他爻」去生克「變爻」、或用「變爻」去生克「他爻」—— 這是六爻體系的非法操作。

四時旺相章第又十五（一年十二月的旺衰循環）
==============================================================

第十至十五章建立了「力學代數和」的計算規則，但所有判斷都依賴一個前提：**爻在什麼月份是旺的？什麼月份是衰的？** 第又十五章就是這張「時間衰旺表」—— 一年十二月的五行旺衰循環。

一、五行旺衰的基本邏輯

每月有一個「月建」（當月的地支），月建所屬五行為「旺」，月建所生之五行為「相」，月建所克／所被克之五行為「休囚」。

公式：**旺 → 生 → 相**；克／被克 → 休囚。

|---|---|---|---|---|---|
| 正月 | 寅 | 木（寅）| 卯 | 火 | 金、水、土 |
| 二月 | 卯 | 木（卯）| 寅 | 火 | 金、水、土 |
| 三月 | 辰 | 土（辰）| 丑、未 | 金（土生金）| 木、火、水（金有餘氣）|
| 四月 | 巳 | 火（巳）| 午 | 土 | 金、水、木 |
| 五月 | 午 | 火（午）| 巳 | 土 | 金、水、木 |
| 六月 | 未 | 土（未）| 辰、戌 | 金（土生金）| 木、水（火有餘氣）|

三、「土月」的特殊性（辰、戌、丑、未四個土月）

土月（三、六、九、十二月）的判讀有兩個特點：
1. **同行土爻**：除了本月建之土最旺外，其他三個土（辰戌丑未）也算「次旺」（同屬土）
2. **餘氣**：上個月當令的五行雖然衰退，但仍有「餘氣」—— 如三月辰土當令時，正二月的木仍有餘氣；六月未土當令時，四五月的火仍有餘氣

**為什麼有餘氣？** 因為土月處於季節交替（春→夏、夏→秋、秋→冬、冬→春），舊勢未完全退去。

==============================================================
月將章第十六（月建的最高權威與月破、月合的判定）
==============================================================

本章建立六爻體系最核心的權威——**月建**（又稱月將、月令）。它「掌一月之權，司三旬之令」（管一個月、統三旬即三十日的權柄），是斷卦時的最高判官。野鶴用六個實例詳論月建的各種作用方式，並反駁兩個傳統錯說。

一、月建的六大職能

> 「月將乃當權之帥，萬卜以之為綱領。」

月建可對卦中所有爻施加六種作用：

1. **扶弱**：合、比、拱、扶衰弱之爻 → 衰而亦旺
2. **抑強**：沖、克、刑、破強旺之爻 → 旺而亦衰
3. **制動變**：可制服變爻克動爻的局面、或動爻克靜爻的局面
4. **生伏神**：用神伏藏被飛神壓住時，月建沖克飛神 → 救出伏神為用
5. **代用神**：卦中若無用神，直接以月建為用神，不必尋伏神
6. **強化動爻**：月建入卦動而作元神 → 福更大；動而作忌神 → 禍更深

二、月合 vs 月破

| 名稱 | 定義 | 性質 |
|---|---|---|
| 月合 | 月建與某爻相合（如寅月合亥爻） | **有用之爻**（被月建拱衛） |
| 月破 | 月建與某爻相沖（如寅月沖申爻） | **無用之爻**（被月建破除） |

**核心對照：**
- 月建一旦入卦（月建之支恰為卦中某爻）→ 「愈見剛強」
- 月建不入卦 → 它依然在外圍施加作用，只是「緩之」（影響稍慢）

三、野鶴反駁兩個傳統錯說

**錯說一：「逢空不空」**
傳統說：「爻值月建旺相者，逢空不空、逢傷無害。」
野鶴駁：**錯！在旬內者畢竟為空**，要等出旬之日才真正不空。

實例：寅月庚戌日占求財，得火天大有：
- 用神寅木妻財（在二爻），克世為吉 → 此財必得
- 但「目下尚空」（庚戌日在甲辰旬，旬空為寅卯）
- **要到甲寅日出空才能得到**
- 應驗：果於甲寅日得財

**結論：** 用神逢空不可立刻判為無用，但也不能說「不空」—— 正確的說法是「出空之日才有用」。

四、「逢傷亦傷卻有待時之用」

爻值月建本是旺相，但若被他爻克 → 「逢傷」。
判讀：占病者目下不愈、占事者目下不成 —— **但「沖去傷爻之日」就解除了**，病必愈、事必成。

實例：酉月丙寅日占謁貴，得山風蠱變山水蒙：
- 世臨酉金官（世動），月建酉金 → 旺
- 但化出午火 → **回頭克世**（火克金，傷）
- 解法：等「子日沖去午火」（子午相沖）→ 失去回頭克的力量
- 應驗：果於丙子日得拜謁

**這條規則的核心：傷爻可以被「沖去」，回頭克可被「沖去回頭爻」化解** —— 應期就是這個沖的日子。

五、月建作元神生用神的雙重利好

實例：寅月丙申日占官之升遷，得艮卦變山雷頤：
- 世臨寅木官星，**月建寅木臨世** → 大旺
- 申日沖寅 → 看似月破
- **但有救：申子辰合成水局**（卦中四爻申、二爻子、上爻辰）→ 水局生官（水生木）
- 結果：寅木官得水生而堅固
- 應驗：三月高遷（辰土出空之月，三合落實）、升任雲南（世與官皆在六爻 = 西南/南方對應雲南）

**這是「月建被沖而有救」的典型** —— 三合局能保護被日辰沖的月建用神。

六、月建作忌神，化退神延緩凶禍

實例：午月丁未日占弟被訟，得困卦變雷風恆：
- 用神酉金兄（在五爻）
- 午月克酉金（火克金）→ 月克
- 未日生酉金 → 日生
- 月克日生 → 看似「可以相敵」
- **但壞在四爻午火動！** —— 午火為月建之同類，動入卦中作忌神 → 「月建作忌神，為禍不淺」
- 大凶之象

**應期推算：** 酉金兄爻「動化退神」（變出申金，酉退為申）→ 化退神延緩凶禍
- 當年（辰年）→ 太歲辰與酉相合 → 暫時無妨
- **退至申年**（酉退到申）→ 無路可退 → 大凶
- 應驗：本年下獄，至申年重刑

**此案的雙層教學：**
1. 「月建作忌神入卦」比一般忌神更狠
2. 化退神可以暫時拖延，但不能根本解決 —— 等退到變爻的時候，就應期已到

七、「天象凶者出月遭殃」

> 「天象吉者從斯而泰，天象凶者出月遭殃。」

**翻譯：** 月建若給用神帶來好局面，從本月開始就順遂；若帶來壞局面，**在本月之內還可勉強撐住，出了這個月才真正遭殃**。

這是月建對「應期」的影響：吉的應期從當月起算，凶的應期常常落在出月之後。

八、用神衰遇時則發，旺極遇沖則敗

> 「物窮則變，器滿則傾。」

兩個對稱的時間規律：

1. **物窮則變**：用神衰時，等到「當令之時」就發力。例：用神臨火、冬天占→火不旺，等到春末夏初則旺。
2. **器滿則傾**：用神極旺時，等到「沖克之時」就敗壞。例：正月寅木月建大旺，等到秋季（七、八月）遇金沖克 → 不無破敗。

實例：寅月丁酉日占開鋪面，得艮卦變明夷：
- 用神世爻寅木官 → **月建寅木大旺**，目下熱鬧
- 但日辰酉金克寅 + 世動化卯木（化回頭克非也，這裡是化進神... 實際上卯為寅的進神，但配合日辰酉金，仍是被沖被克）
- **生少克多 + 六沖卦** → 凶
- 應驗：六月疾至（木墓於未，未為六月）、八月夥計盜盡（酉金當令沖克世）、財被盜盡（四爻子財落空亡）

**這是「當時旺相無傷，過時有害」的典型** —— 月建保你一時，保不了一世。

九、月生日克 vs 月克日生

> 「月克日生遇幫而愈旺，月生日克逢他削而亦衰。」

**判讀公式：**
- 月克日生 → 用神還有救（日辰勝月建？不，是兩者勢均力敵，但日辰是「目下」，月建是「整月」）
- **得幫扶 → 愈旺**；遇削克 → 終究衰
- 月生日克 → 同樣勢均力敵，但**遇進一步削克 → 衰**

實例：午月戊辰日占妹臨產，得火地晉：
- 用神世爻酉金兄
- 午月克酉金（月克）
- 辰日生酉金（日生）
- 月克日生 → 相敵，許之無礙
- 應期：**「卯日卯時必生」**（卯與辰合 = 沖開合住 = 沖開的時點）
- 應驗：果於次日卯時生，母子平安

**黃金策的名言：** 「若逢合住，必待沖開。」—— 用神被合住時，要等「沖去合神」之日才能發力。

==============================================================
日辰章第十七（日辰的同權地位與「神兆機于動」的核心信念）
==============================================================

月建是「整月之帥」，日辰則是「當下之主」。第十七章與第十六章對稱：兩者「同功」（同樣有最高權威），但職能略有不同。野鶴在本章中表態了一個非常重要的信念：「**神兆機于動，動則必驗**」—— 反對傳統的「沖散」說。

一、日辰的特殊地位

> 「日辰為六爻之主宰，司四時之旺相。」

**月建 vs 日辰的對比：**

| 維度 | 月建 | 日辰 |
|---|---|---|
| 影響範圍 | 整月（三旬，30 日）| 當下（這一天）|
| 季節性 | 春令生、秋令殺，依季節變化 | **四時俱旺**，不受季節影響 |
| 沖爻效果 | 旺相之爻沖即為月破 | **旺相靜爻沖為暗動**（愈得其力）；衰弱靜爻沖為日破 |
| 對伏神 | 沖克飛神而生扶伏神 | 同 |
| 與卦的關係 | 不入卦也有用、入卦更剛強 | 同 |

二、日辰的核心職能

1. **沖空則實**（用神逢旬空，日辰沖之則發力）
2. **沖合則開**（用神被合住，日辰沖合住之神則開）
3. **沖旺靜為暗動**（靜爻遇旺日沖之，化暗動發力）
4. **沖衰靜為日破**（衰爻遇日沖則破，愈加無用）
5. **生扶衰爻** → 如時雨潤苗
6. **克害旺爻** → 如秋霜殺草

三、野鶴反駁「沖散」說

> 「他書有云：爻逢月建日沖而不散」 ⋯⋯「最重要者『散』，獨予屢試不驗。」

**傳統說：** 動爻被日辰沖叫「散」—— 動而不應，吉凶皆作廢。
**野鶴駁：** 我四十年實占從沒驗證過這個。「**神兆機于動，動則必驗**」—— 神靈把徵兆顯露在動爻上，動爻必然應驗。

**唯一可能「散」的例外：** 休囚無氣的動爻被沖時，**百中僅有一二例「真散」**。
其他情況：旺相動爻被沖，反而「愈強」。

**這個信念貫穿整本書。** 之前第十章求財案、本章後文都是這個信念的體現：動爻臨空也好、被沖也好、化空也好 —— 都不算作廢，只是延後應期。

四、「逢月破而不破」「遇沖克而無傷」

當爻臨日辰時，類似於臨月建 —— **似與月建同權，中天日麗，旺相之機**。

但有兩個前提：
1. 生多克少 → 錦上添花
2. 生少克多 → 寡不敵眾

**「破亦為破，傷亦為傷」的例外：**
酉月卯日占卦，爻臨卯木（月破但臨日辰）—— 表面是「逢破不破」。
**但若卦中動出申酉之金、或卯爻動化申酉** → 寡不敵眾 → 終究破矣、傷矣。

**這條規則的精髓：** 日辰的保護不是無限的，遇到夠多的克勢仍會被打破。

五、實例一：申月戊午日占病，得 天山遁 變 天風姤

```
   父母戌土 ⚊       父母戌土 ⚊
   兄弟申金 ⚊ 應   兄弟申金 ⚊
   官鬼午火 ⚊       官鬼午火 ⚊
   兄弟申金 ⚊       兄弟酉金 ⚊
   官鬼午火 ㄨ 世   子孫亥水 ⚊      ← 動
   父母辰土 ⚋       父母丑土 ⚋
```

**分析：**
- 自占病 → 用神世爻午火
- 戊午日 → **世爻午火臨日辰**，本主旺相
- 但⋯⋯申月（月建申金）生亥水（變爻）→ 亥水反過來克世午火 = **回頭克**
- 月建透過變爻克世，**日辰的保護不夠**
- 應驗：卒於亥月（亥月當令，回頭克力量到頂峰）

**此例對應本章第四節：** 「逢沖克而無傷」是有條件的。當月建透過變爻反克時，日辰也救不了。

六、實例二：巳月亥日占僕何日回，得 夬卦 變 履卦

```
   兄弟未土 ㄨ      兄弟戌土 ⚊      ← 動
   子孫酉金 ⚊ 世   子孫申金 ⚊
   妻財亥水 ⚊       父母午火 ⚊
   兄弟辰土 ○      兄弟丑土 ⚋      ← 動
   官鬼寅木 ⚊ 應   官鬼卯木 ⚊
   妻財子水 ⚊       父母巳火 ⚊
```

**分析：**
- 用神：亥水妻財（在四爻，靜）—— 占僕從財看驅使之人
- **亥水月破**（巳月沖亥）
- 但**亥水臨日建**（亥日）→ 看似「逢破不破」
- 但卦中**四重土動**（未、辰、戌、丑）—— 兩動兩變土合計四 → 「雙拳不敵四手」
- 寡不敵眾 → 月破終究是月破，亥水終遭克害

**結果：** 不獨難望歸期，更要預防不測。應驗：午月卯日得信，僕中途已遭害。

**這是「破亦為破，傷亦為傷」的典型例證** —— 即便日辰扶持，敵勢過多時保護失效。

七、本章總注

> 「旺之者，爻之帝旺於日月也；比之者，爻與日月同也；扶之、拱之者，爻與日、月同類也；墓之、絕之者，爻墓絕於日月也。」

**翻譯：** 解釋了四個常用術語：

| 術語 | 定義 | 例 |
|---|---|---|
| 旺 | 爻在日月處於十二長生的帝旺位 | 火爻在午日／午月 |
| 比 | 爻與日月地支完全相同 | 寅爻在寅日／寅月 |
| 扶 / 拱 | 爻與日月同屬一行（不完全相同）| 寅木爻在卯日／卯月 |
| 墓 / 絕 | 爻在日月處於十二長生的墓位／絕位 | 火爻在戌日（火墓）／亥日（火絕）|

這四個術語是後續章節（特別是長生章、墓絕章）的基礎。

八、李我平的補注

> 「易以爻臨日辰莫能散之、莫能空之，謂逢空而不空也。然動逢日沖謂散，既作日辰又有日辰來沖乎？旬空者乃旬內之所無也，既是旬空其又有臨日辰之事？故知讀書之不足如此。」

**翻譯：** 李我平指出兩個邏輯謬誤：
1. 既然爻已經「臨日辰」（爻 = 日辰），怎麼還會有「日辰來沖它」的事？日辰不會自己沖自己。
2. 既然爻「旬空」（在本旬不存在），又怎麼可能「臨日辰」？這兩者邏輯互斥。

**結論：** 「故知讀書之不足如此」—— 諷刺其他書讀者沒搞清基本邏輯就亂用術語。

==============================================================
六神章第十八（六獸的象意與「附合之神」的定位）
==============================================================

第十七章談「日辰」這項斷卦最高權柄，第十八章轉向一個爭議性很大的話題：**六神（六獸）究竟有沒有用？** 傳統諸書幾乎一面倒地以青龍為吉、白虎為凶，野鶴在這章直接反駁這個成見，並用兩個反例（青龍臨子孫卻夭折、白虎螣蛇齊現卻病癒）證明：**六神不能獨立決定生死，只是「附合之神」—— 它附在五行生克的本質判斷上，渲染色彩，不能逆轉本質。**

一、六神起例（按日干起六神）

每日的六神排列依日干而定，由初爻到上爻配六神：

| 日干 | 初爻 | 二爻 | 三爻 | 四爻 | 五爻 | 上爻 |
|---|---|---|---|---|---|---|
| 甲乙日 | 青龍 | 朱雀 | 勾陳 | 螣蛇 | 白虎 | 玄武 |
| 丙丁日 | 朱雀 | 勾陳 | 螣蛇 | 白虎 | 玄武 | 青龍 |
| 戊日 | 勾陳 | 螣蛇 | 白虎 | 玄武 | 青龍 | 朱雀 |
| 己日 | 螣蛇 | 白虎 | 玄武 | 青龍 | 朱雀 | 勾陳 |
| 庚辛日 | 白虎 | 玄武 | 青龍 | 朱雀 | 勾陳 | 螣蛇 |
| 壬癸日 | 玄武 | 青龍 | 朱雀 | 勾陳 | 螣蛇 | 白虎 |

**注意**：原文行序為「上爻→初爻」，本表按由下而上（初爻→上爻）整理，更便於閱卦時對照。

二、野鶴反駁傳統六神斷生死的成見

野鶴列舉古書四種說法：
- 《天元賦》：「身旺龍持多吉處。」
- 《碎金賦》：「龍動家有喜，虎動家有喪。」
- 《卜筮元龜》：「螣蛇白虎憂尊長。」
- 《卜筮大全》：「畏口禦刀之白虎，喜赴水之青龍。」
- 《閱奧章》占疾病：「螣蛇主死，白虎主喪。」

野鶴的批評：

> 「此皆不以五行，竟以六神而斷生死。」

——這些書都拋開了五行生克的本質，僅憑六神顏色就論吉凶，是本末倒置。

野鶴唯獨贊同《千金賦》的說法：

> 「虎興而遇吉神不害其吉，龍動而逢凶曜難掩其凶。」

—— **六神不能凌駕於五行生克之上**。

三、六神的真實作用：附合之神

> 「然則六神而不驗耶乎？乃附合之神也。」

**附合之神** 意即：六神附著於五行判斷之上，渲染色彩、提供細節，但不主導吉凶本質。

- 卦本吉者，**逢青龍而更吉**（喜上加喜）
- 卦本凶者，**逢白虎而更凶**（凶上加凶）
- 玄武主盜賊、朱雀主口舌 —— 在用神已決定吉凶的前提下，六神可指明事件的「性質類別」
- 占家宅、墳塋這類事，六神「不可少也」（用於辨象、辨方位）

**先定五行吉凶，再用六神描色**。

四、反例一：青龍臨子孫卻夭折

戊子日占生產，得 山地剝 變 風地觀：

```
   朱雀  妻財寅木 ⚊         妻財卯木 ⚊
   青龍  子孫子水 ㄨ 世 ← 動  官鬼巳火 ⚊
   玄武  父母戌土 ⚋         父母未土 ⚋
   白虎  妻財卯木 ⚋         妻財卯木 ⚋
   勾陳  官鬼巳火 ⚋ 應      官鬼巳火 ⚋
   螣蛇  父母未土 ⚋         父母未土 ⚋
```

- 占生產以子孫爻為用神 → 五爻子水子孫持世
- 子水動而化出巳火 —— **化絕**（子水絕於巳）+ **化鬼**
- 戊子日子水值日本應旺，但動而化絕變鬼 → 本日落草而亡

**野鶴反詰**：「卻是青龍臨子孫，亦可謂之喜耶？」 —— 青龍救不了化絕變鬼的子孫。

五、反例二：白虎螣蛇齊現卻病癒

申月甲辰日占兄病，得 屯卦 變 震卦：

```
   玄武  兄弟子水 ⚋         官鬼戌土 ⚋
   白虎  官鬼戌土 ○ 應 ← 動  父母申金 ⚋
   勾陳  父母申金 ㄨ ← 動    妻財午火 ⚊
   螣蛇  官鬼辰土 ⚋         官鬼辰土 ⚋
   朱雀  子孫寅木 ⚋ 世      子孫寅木 ⚋
   青龍  兄弟子水 ⚊         兄弟子水 ⚊
```

- 取上爻子水兄為用神
- 申月生子水 + 忌神戌土動 → 生元神申金 → 申金動生子水：**忌神元神同動，貪生忘克**
- 戊申日（元神申金值日）沈疴復起

**野鶴反詰**：「其可謂之蛇動死，虎動主喪耶？」 —— 卦中三爻螣蛇、四爻白虎齊動，按古書當主死，但實際病癒。**五行生克鏈遠強於六神渲染。**

六、本章教訓

| 規則 | 內容 |
|---|---|
| 六神不主吉凶 | 吉凶由五行生克決定 |
| 六神是附合之神 | 用以渲染、辨象，不能逆轉判斷 |
| 占病喪 | 不可單憑白虎螣蛇而斷死 |
| 占家宅墳塋 | 六神「方位象意」更為必要 |

==============================================================
六合章第十九（六合的六種形式與「沖變合」的特例）
==============================================================

野鶴明確：合不全然為吉，必須結合用神旺衰判斷。本章還介紹了一個特殊例外：**六沖卦變六合卦**，這時可以「不看用神，竟以吉斷」—— 六爻體系中極少數的「形式凌駕本質」之例。

一、六合的基本對應

> 子與丑合、寅與亥合、卯與戌合、辰與酉合、巳與申合、午與未合。

二、六合的六種形式

| 序號 | 名稱 | 形式 |
|---|---|---|
| 1 | 日月合爻 | 日辰或月建與某爻相合 |
| 2 | 爻與爻合 | 卦內兩爻互相合 |
| 3 | 爻動化合 | 動爻變出之爻與本爻成合 |
| 4 | 卦逢六合 | 本卦自身為六合卦（如否、泰等）|
| 5 | 六沖卦變六合卦 | 八純六沖之卦變為六合卦 |
| 6 | 六合卦變六合卦 | 本卦六合，變卦仍六合 |

三、合的四種狀態與意義

| 狀態 | 條件 | 涵義 |
|---|---|---|
| **合起** | 靜爻逢日月或動爻來合 | 「合而起」—— 雖休囚亦激活 |
| **合絆** | 動爻逢日月或他動爻來合 | 「動逢合而絆住」—— 反不能動 |
| **合好** | 動爻與動爻相合 | 「他來合我，與我和好」|
| **化扶** | 動爻化出之爻回頭相合 | 「得他扶助之意」|

**關鍵**：靜爻合 → 激活（合起）；動爻合 → 絆住（合絆）。**兩個方向相反**，常見的誤判點。

四、合的吉凶總則

> 「凡得諸合，諸占皆以為吉⋯⋯然必用神有氣相宜，用若失陷無益。」

合是吉象，但 **用神必須有氣**。用神失陷時，合只是空架子。

五、實例一（合絆世動）：申月丙子日占出門，得 明夷 變 小過

```
   父母酉金 ⚋          官鬼戌土 ⚋
   兄弟亥水 ⚋          父母申金 ⚋
   官鬼丑土 ㄨ 世 ← 動   妻財午火 ⚊
   兄弟亥水 ⚊          父母申金 ⚊
   官鬼丑土 ⚋          妻財午火 ⚋
   子孫卯木 ○ 應 ← 動   官鬼辰土 ⚋
```

- 世爻丑土動，子日子丑相合 → **合絆，反不能動身**
- 應爻卯木子孫動化辰土為官鬼 → **子孫變鬼**（小女病亡之兆）
- 卯木：申月絕、子日相刑、化鬼 → 三重凶

**斷與應期**：「合住必有事絆，不能動身」；女應死於辰日（卯化辰，辰日合住卯木）。
**結果**：此人未去，女死辰日。**雙重應驗**。

六、實例二（六沖卦變六合 —— 散而復聚）：未月丁巳日占已悔婚還可成否，得 離卦 變 火山旅

> 「諸合皆以用神旺者為吉，獨此六沖卦變六合得不看用神，竟以吉斷。」

**六沖變六合 = 散而復聚、離而必合** —— 可不論用神而直接吉斷。

| 占類 | 六沖變六合斷語 |
|---|---|
| 婚姻 | 「先吳越而後朱陳」（先敵後親）|
| 夫妻 | 先參商後必合好 |
| 功名 | 始艱阻終榮顯 |
| 求謀 | 先難後易 |
| 身命 | 先困後顯 |
| 家宅 | 先衰後盛 |

**應驗**：次年三月仍成婚。

七、六合卦變六合卦的加倍吉象

| 占類 | 斷語 |
|---|---|
| 風水 | 百代顯榮 |
| 宅舍 | 千秋基業 |
| 婚姻 | 白頭相守 |
| 功名 | 仕路亨通 |
| 財帛 | 聚積如山 |

**例外**：訟獄、憂疑、產（合住難下）、用神受克時，六合變六合反凶。

八、占卜倫理示範（風水實例）：卯月甲寅日占祖塋，得 困卦 變 節卦

野鶴的問占程序非常重要：

> 「占祖塋必有其故，自葬後何事不亨？今日之念因何而起？明以告我，方敢決斷。」

—— **野鶴先問占者「為什麼今天問」**，理解占者的疑慮所在，再決斷。

占者答：自葬之後功名屢困、年近五旬尚無子息。
野鶴斷：「龍自右脈來，水亦從左而還源 —— 流水不歸漕。」若使水歸漕：明年起用、申年生子。
占者問：「長久安泰否？」
野鶴答：「**六合萬載安然。**」

九、三合局

四種三合局（凡合化五行）：
- 申子辰 → 水局
- 巳酉丑 → 金局
- 寅午戌 → 火局
- 亥卯未 → 木局

三合的喜忌分占：

| 占類 | 喜合何局 | 忌合何局 |
|---|---|---|
| 功名 | 官局、財局 | 子孫局 |
| 求財 | 財局、子孫局 | 兄弟局 |
| 婚姻 | 財官旺局 | 子孫局（克官）|

> 「凡占求謀喜慶之事宜於成局，永遠堅牢。若占官訟、憂疑而合局者，終身團結，其心難於消釋。」

**喜事宜成局**（堅牢長久），**憂事最忌成局**（糾纏不解）。

**特殊狀況**：
- 三爻有兩動一未動 → 不成局，待後之補湊（虛三待用）
- 一爻明動、一爻暗動 → 也算兩爻動
- 局中一爻空破 → 待填空、補破之日成局
- 局中一爻入墓 → 待沖開之日成局

十、實例三（三合分判）：卯月丁巳日，上下兩村因田水械鬥，得 離卦 變 坤卦

野鶴的分析：
- **內卦為我村，外卦為人村**
- 我村方：亥卯未三合木局（卯月當令）
- 對方村：巳酉丑金局（春木所克，衰金）
- 「衰金不克旺木，不足畏也」
- 加上六沖變六沖 → 「**有人解散必不成訟**」

占者問：「彼此之形必以世應為主，為何不言世應？」

野鶴答：

> 「若無內外合局者，須看世應；今彼此兩村即內外兩卦，人眾同心，彼此合局⋯⋯故舍世應而不用也。」

—— **多人對多人的械鬥事件中，內外卦比世應更能反映「集體立場」**。

==============================================================
六沖章第二十（沖的六種形式與「沖即散」的辨偽）
==============================================================

第十九章談合，第二十章談沖，兩章對稱。傳統占法以「沖即散」為金科玉律，但野鶴的態度更為精細：**沖是散，但「散」要看用神**。用神旺者沖之不破，用神失陷者凶上加凶。本章還細分了「爻沖」的五種變體 —— 月破、暗動、日破、回頭沖、相殺 —— 這是占斷時最容易誤判的環節。

一、六沖的基本對應

> 子午相沖、丑未相沖、寅申相沖、卯酉相沖、辰戌相沖、巳亥相沖。

二、六沖的六種形式

| 序號 | 形式 |
|---|---|
| 1 | 日月沖爻 |
| 2 | 卦逢六沖（本卦為六沖卦）|
| 3 | 六合卦變六沖卦 |
| 4 | 六沖卦變六沖卦 |
| 5 | 動爻變沖（化回頭沖）|
| 6 | 爻與爻沖（卦內兩爻自相沖）|

三、沖的吉凶總則

> 「沖者散也，凡占凶事宜於沖散，占吉事而不宜，必兼用神而言。用神旺雖沖不破，用神失陷凶而又凶。」

**核心邏輯**：
- 占凶事（疾病、官非、糾纏）→ 沖散 = 吉
- 占吉事（婚姻、求財、長久）→ 沖散 = 凶
- **用神旺者沖不破**；**用神失陷者凶上加凶**

四、六合變六沖：有始無終

> 「六合卦變六沖卦者，用神旺者始吉終凶，圖事難成，有始無終也。」

—— 與第十九章「六沖變六合」對稱：六合變六沖是「先合後散」，事情雖暫時成形，但終究會散。

五、爻沖的五種變體（最精細的辨識）

| 變體 | 條件 | 結果 |
|---|---|---|
| **月破** | 爻與月建相沖 | 當月無用（出月再論）|
| **暗動** | 靜爻旺相遇日辰沖 | 不破反動（隱形動）|
| **日破** | 爻休囚而遇日沖 | 真破 |
| **回頭沖** | 動爻化出之爻反沖本爻 | 「如逢仇敵」|
| **相殺** | 兩爻直接相沖 | 自相殺戰 |

**重點**：同樣是「日辰沖」—— **旺則暗動、衰則日破**。結果完全相反。

六、實例一（沖空填實）：巳月戊戌日占財，得 風雷益

- 用神世爻辰土財
- 戊戌日辰土值旬空（旬空為辰巳）
- 戌日沖辰 → **沖空即填實**
- 當日（戊戌日）即得財

七、實例二（化回頭沖反而吉）：午月丙辰日占出外經商，得 恆卦 變 豫卦

- 世爻酉金動，化出卯木 → **卯酉相沖 + 反吟**
- 但**辰日合住酉世**（沖中逢合）+ 戌土應位被辰日沖（暗動生世）
- 化沖被日合救之，暗動生之 → 沖而不克
- 應驗：去而復反者三次，中途發財

—— 化回頭沖一般是凶，但**「沖」不是絕對結局，要看整盤救應**。

八、實例三（六沖卦但用神旺亦不散）：酉月乙未日占子久出不歸，得 坤卦（全靜）

- 用神世爻酉金子孫
- 酉月臨月建 + 未日生 → 用神極旺
- 坤卦為八純六沖卦，但 **「靜爻旺相逢沖 = 暗動，不是破」**
- 斷「此子必歸」；應「子年建功，卯年得意而歸」

—— **六沖卦不等於必散，用神旺則沖不破**。野鶴反駁「逢沖即散」的成見。

九、實例四（六合變六沖：師生關係不久）：巳月甲寅日占嚴師訓子，得 天地否 變 乾卦

- 應爻戌土父為用神（師為彼方）
- 用神有氣 → 「乃飽學之士」
- 但天地否（六合）變乾為天（六沖）→ **不久之兆**
- 子水子孫旬空 + 父動克子 → 兩月後其子病亡，辭師

**對應總則**：

> 「凡得六合變六沖者，諸占先合後離、先親後疏、先濃後淡、始榮終悴⋯⋯惟占官非、盜賊事者宜之。」

十、實例五（六沖變六沖救官非的反例）：申月己卯日，父子七人俱蒙拿問，得 巽卦 變 坤卦

- 世爻卯木化酉金 → 化回頭沖克
- 五爻巳火子孫化亥水 → 化回頭沖克（巳亥沖 + 水克火）
- 巽變坤 + 連動 → **六沖亂室**
- 結果：父子七人俱受刑

野鶴的方法論結語：

> 「占法皆以六沖卦宜於官事，喜其沖散，然必看用之大小，又宜兼用神而言。」

—— **六沖確實「沖散」，但「散」的內容是什麼，要看用神**。本案沖散的不是官非，而是全家性命。

三刑章第二十一（三刑的驗證與野鶴的否定態度）
==============================================================

本章極短，但極具野鶴的個性 —— **他公開承認「三刑」是傳統占法中近乎無效的規則**。古書言三刑必凶，野鶴占了數十年只驗到一個獨例。這是六爻體系中「實證凌駕傳統」的代表章節。

一、三刑的傳統對應

> 「寅刑巳、巳刑申、子刑卯、卯刑午、丑戌相刑、未辰相刑。又云：辰午酉亥謂之自刑。」

| 三刑類別 | 組合 |
|---|---|
| 無恩之刑 | 寅 → 巳 → 申 → 寅（循環）|
| 持勢之刑 | 丑、戌、未（互刑）|
| 無禮之刑 | 子刑卯、卯刑午 |
| 自刑 | 辰、午、酉、亥（各刑自身）|

二、野鶴的實證態度

> 「夫三刑者，予屢試之，或因用神休囚又兼他爻犯之，刑者則見凶；而獨犯三刑得驗者少，占過數十年只驗得一卦。」

**翻譯**：三刑這個說法，多數情況是**用神已經休囚衰敗，再加上他爻犯三刑**，這時三刑才顯凶。**單獨犯三刑而能應驗者極少** —— 數十年只見過一例。

**核心理解**：凶象的本質在於「用神衰敗」，三刑只是壓垮駱駝的最後一根稻草。**單獨三刑不足以斷凶**。

三、唯一獨立應驗的三刑卦：寅月庚申日占子痘症，得 風火家人 變 離卦

```
   兄弟卯木 ⚊          子孫巳火 ⚊
   子孫巳火 ○ 應 ← 動   妻財未土 ⚋
   妻財未土 ㄨ ← 動     官鬼酉金 ⚊
   父母亥水 ⚊          父母亥水 ⚊
   妻財丑土 ⚋ 世       妻財丑土 ⚋
   兄弟卯木 ⚊          兄弟卯木 ⚊
```

**初判（按常理）**：
- 用神巳火子孫，寅月生火 → 旺相
- 「子孫旺相，許之可治」

**應驗反證**：「後死於寅日寅時。」

**野鶴的事後反省**：

> 「後悟月建在寅，日建在申，與巳爻子孫共作三刑，獨此一卦，無他爻之傷也。」

—— 月建寅 + 日建申 + 用神巳火 = **寅巳申三刑** 完整成形！這是 **唯一一個** 沒有其他凶象、純粹靠三刑而應驗的卦例。

**這個獨例的特殊性**：
1. 月、日、用神三爻地支恰好構成完整三刑
2. **三刑由日月直接構成**，這比卦內爻間的三刑威力大得多
3. 用神雖旺，但被「天條級」的日月夾刑

四、野鶴對其他三刑類別的態度

> 「至於子卯、辰戌丑未亦有驗者，皆附和而為凶也。」

**翻譯**：子卯刑、辰戌丑未刑，**有應驗的也都是「附和」其他凶象** —— 必須搭配用神衰敗、忌神有力等本質凶象才會顯凶。

五、本章方法論意義

| 規則 | 內容 |
|---|---|
| 三刑不獨主凶 | 必須用神先衰，他爻再犯三刑 |
| 唯一獨立應驗的形式 | 月、日、用神三爻構成完整三刑 |
| 子卯、辰戌丑未刑 | 「附和而為凶」—— 跟著其他凶象一起應 |
| 自刑（辰午酉亥）| 野鶴未舉例，更弱 |

**呼應**：與第十八章對六神的批評同調 —— **規則要服膺於實證，不可機械套用**。

暗動章第二十二（靜爻逢日沖的隱形動能）
==============================================================

**暗動是六爻體系最隱秘的力量** —— 它不是明動，但卻能夠生克他爻、影響全局，且常常「福來而不知，禍來而不覺」。本章與下章「動散章」並列辨識 —— 兩者都涉及「日辰沖爻」，但結果完全相反。

一、暗動的定義與條件

> 「靜爻旺相，日辰沖之為暗動；靜爻休囚，日辰沖之為破。」

| 靜爻狀態 | 遇日辰沖 | 結果 |
|---|---|---|
| **旺相**（月建臨／生、或日辰生扶）| 沖之 | **暗動**（隱形動，能生克）|
| **休囚**（無根無扶）| 沖之 | **日破**（真破，失效）|

**同一個「沖」動作，結果完全相反** —— 取決於靜爻本身的旺衰。這是六爻最重要的辨識點之一。

二、暗動的吉凶意義

> 「暗動者有喜有忌。」

| 情境 | 性質 |
|---|---|
| 用神休囚，得元神暗動以相生 | **喜**（救應）|
| 忌神明動克用神，得元神暗動生用神 | **喜**（克處逢生的暗動版）|
| 用神休囚無助，遇忌神暗動克害 | **忌**（隱形災禍）|

三、暗動的隱秘性

> 「占以暗動，福來而不知，禍來而不覺。」

當事人事發前不易察覺 —— 因為動爻是隱形的（不像明動有變爻可見），但結果照常應驗。

**糾正一個誤解**：

> 「又曰：吉凶之應於動，有急緩之應，則緩非此論。何當不知不覺，報應亦非緩也。」

**翻譯**：暗動的應期並不慢，**只是隱於不知不覺中**，並不代表事件本身發生得遲。**暗動是「隱形」，不是「慢」**。

四、實例：寅月己未日占女痘，得 坤卦 變 師卦

```
   子孫酉金 ⚋ 世       子孫酉金 ⚋
   妻財亥水 ⚋          妻財亥水 ⚋
   兄弟丑土 ⚋          兄弟丑土 ⚋
   官鬼卯木 ⚋ 應       父母午火 ⚋
   父母巳火 ㄨ ← 動     兄弟辰土 ⚊
   兄弟未土 ⚋          官鬼寅木 ⚋
```

**分析**：
- 用神世爻酉金子孫，寅月木旺金衰（休囚），己未日土生金（日生救起）
- 忌神：二爻巳火父動 → 火克金
- **救應**：未日沖丑（兄爻丑土），丑土暗動 → 土生酉金

**斷與應期**：
- 「花雖密以全生」
- 「今日未申時有救」
- 應驗：申時遇明醫救治

**野鶴的結論性反問**：「**何當緩也？**」 —— 暗動的救應在同一日的時辰內就應驗了。

五、本章與其他章節的辨識對比

|---|---|

| 元神忌神衰旺章第十 | 元神有力的判讀 |

==============================================================
動散章第二十三（動爻能否被沖散的辨偽）
==============================================================

第二十二章談「靜爻旺相逢沖為暗動」，本章回答對應的問題：**動爻能否被沖散？** 野鶴答案再度反駁傳統 —— **旺相動爻沖之不散，有氣動爻沖之不散，唯休囚動爻偶有沖散，仍是千百中之一二**。

一、動爻能被沖散嗎？

> 「占以日辰而沖動爻，謂之沖散；又以爻動沖爻，亦能沖散。」

野鶴的實證結論：

> 「予屢試之，旺相者沖之不散，有氣者沖之不散，休囚者間有沖散，亦千百中之一二也。」

| 動爻狀態 | 沖之結果 |
|---|---|
| 旺相動爻 | 沖之**不散** |
| 有氣動爻 | 沖之**不散** |
| 休囚動爻 | 偶有沖散，「千百中之一二」 |

**結論**：「沖即散」對動爻而言基本不成立。

二、動爻不散的本質原因

> 「神兆機於動，動必有因。雖則今日受制，後逢值日而不散也。」

**翻譯**：動本身代表神的徵兆，必有其因。即使今日因為日辰沖之看似被制，**動爻會在自己的值日（地支當令之日）顯現原本的徵兆**。

**核心邏輯**：靜爻有「散去」可能（無根則破），動爻則「動本身即實」，難以真散。應期推算的關鍵：**動爻被沖時，應在「動爻地支本氣值日」**。

三、實例：丑月丁酉日占父出外一載無音，得 風水渙 變 坎卦

- 用神：上爻卯木父動
- 動而生世 + 化出子水（化回頭生）
- 酉日沖卯，看似「沖散」
- 但按本章規則：**動爻有氣，沖之不散**

**應期**：卯木本氣值月 → 二月（卯月）
**應驗**：「果於二月得意而回。」

野鶴的反問：

> 「此非卯動酉日沖之？何當散也？」

—— 動而不散，依本氣值月應期。

四、李我平的對比按語（清代占學的內部辯論）

> 「李我平曰：黃金策以空亡為重，沖散為輕；易冒最重沖散。細閱占驗多因錯課。」

| 派別 | 看法 |
|---|---|
| 《黃金策》 | 空亡為重，沖散為輕 |
| 《易冒》 | 最重沖散 |
| 李我平 / 野鶴 | **多數誤判都是因為過度看重沖散** |

**對動爻而言，旬空的破壞力 > 沖散的破壞力**。

五、動靜辨識總結表

| 情況 | 結果 |
|---|---|
| 靜爻旺相逢日沖 | **暗動**（第二十二章）|
| 靜爻休囚逢日沖 | **日破**（第二十、二十二章）|
| 旺相動爻逢沖 | **不散**（本章）|
| 有氣動爻逢沖 | **不散**（本章）|
| 休囚動爻逢沖 | **間或沖散**（千百中之一二）|
| 動爻被沖的應期 | **本氣值日／值月** |

**這四章（六合、六沖、暗動、動散）共同構築了六爻「動靜辨識」的完整體系**。

卦變生克墓絕章第二十四（變爻吉凶的五大格局）
==============================================================

第十五章已立下「變爻只生克本位動爻」的鐵則。本章把這條鐵則的應用收斂為五個固定格局：**化生、化克、化墓、化絕、化比和**。野鶴在這裡丟出一個極具殺傷力的反傳統判斷 ——「只要遇到化回頭克者，**不論用神衰旺皆以凶推**」，並用兩個應驗醫案為自己背書。

一、卦變的五大格局

| 格局 | 動爻 → 變爻 | 五行關係 | 吉凶 |
|---|---|---|---|
| **化生**（化回頭生）| 變爻生動爻 | 如巽木 → 坎水 | 吉 |
| **化克**（化回頭克）| 變爻克動爻 | 如震木 → 乾金 | 大凶 |
| **化墓** | 變爻為動爻之墓庫 | 木動化未、金動化丑 | 凶 |
| **化絕** | 變爻為動爻之絕地 | 木動化申、火動化亥 | 大凶 |
| **化比和** | 動爻變爻同五行 | 木化木、火化火 | 中性 |

二、野鶴的反傳統判斷

> 「凡遇卦化克者，不論用神之衰旺皆以凶推。」

**這是非常重的話。** 傳統說法是：用神當令旺相，可以抵抗化回頭克。野鶴反駁 —— 即使月日生扶讓用神「目前」旺相，化回頭克如同「**樹連根被掘出土**」，目前枝葉雖青，過時必朽。

**化回頭克破壞的是卦體的根本**，應期不在當下，而在「過時衰時」—— 用神出了當令月日、沒了月日這層遮罩，根斷的真相就顯現。

三、實例一：卯月辛巳日，代占長輩功名，得 巽卦 變 乾卦

- 巽卦純體變乾卦純體 → **巽木變乾金 → 金回頭克木**
- 木於申絕、於酉胎 → 化絕 + 化克並見
- 卦體已根斷，不必細究用神

**斷與應期**：野鶴直斷「不必問名，壽命不長」。**果於削職後七個月而終** —— 應期落在木氣徹底衰退的「過時」之後（春末占，秋冬亡）。

**教學重點**：
1. 「卦體變」（八純對八純）比「爻變」更具總體性，可直接判讀
2. 代占功名按理難取用神，但卦體已明白給出「絕克並見」
3. 化回頭克的應期常在用神「失令」的下一個季節

四、實例二：午月丙寅日，占主病，得 離卦 變 坎卦

- 離卦變坎卦 → **離火變坎水 → 水回頭克火**
- 世爻巳火為用神，午月當令 + 寅日生 → 目前極旺
- 但化回頭克 —— 「目今夏火雖旺，冬來火衰命絕」

**斷與應期**：「冬令必危」。**果於九月丁亥日卒** —— 亥水當令時節，回頭克的真相浮現。

**應期推算公式**：**化回頭克的應期 = 用神當令過後 + 變爻當令時節**。

五、李我平按語：駁《易冒》

李我平引《易冒·墓絕章》「日月當令非真（用神化絕）」、「半徒往半凶吉」等說法，斥為「種種疑論，皆非經驗」。並補一例反駁《易冒》：寅月癸酉日占長子病，得 **震之兌**（震木變兌金 → 化回頭克），《易冒》誤判作「反復休囚」，沒看出本質是化回頭克。

反伏章第二十五（反吟與伏吟 —— 內外不寧的卦象）
==============================================================

「反伏」是六爻最容易引起誤判的兩個卦象：**反吟**（變爻沖本爻）與**伏吟**（變爻與本爻同支但動靜不同）。傳統把反伏一律視為大凶，野鶴細分 —— 反伏的本質是「反復、不寧」，**真正決定吉凶的還是用神旺衰與是否化回頭沖克**。

一、卦變與爻變的分別

| 類別 | 定義 |
|---|---|
| **卦變反伏** | 內外卦同時動變，本卦與之卦整體相反或相同 |
| **爻變反伏** | 個別爻動變 |
| **外卦反伏** | 只外卦動變、內卦不動 |
| **內卦反伏** | 只內卦動變、外卦不動 |

二、反伏的判斷大綱

> 「內卦反伏，內則不安。外卦反伏，外則不寧。內外反伏者，內外不寧之象也。」

反伏卦的共通象意：**成而敗、敗而成、有而即無、無而即有、得而失、失而得、來而去、去而來、散而聚、聚而散、動而思靜、靜而思動。**

| 占類 | 反伏的應驗 |
|---|---|
| 功名 | 用爻旺者遷而又行；失陷則或升或降 |
| 財物 | 聚散不常 |
| 墳塋宅舍 | 欲遷不遷，或遷之而再遷 |
| 天時 | 晴而即雨，雨而又晴 |
| 婚姻 | 反復難成 |
| 疾病 | 愈而有病 |
| 出行 | 行至中途變反 |
| 形勢 | 內反伏我亂他定；外反伏他亂我定 |

**關鍵判斷**：反伏卦用神旺相不變沖克者，**雖則反復，事之必成**；**最怕用神化回頭沖克**。

三、實例一：卯月壬申日，占隨官府上任，得 比卦 變 井卦（內卦反伏 + 化回頭沖克）

```
   妻財子水 ⚋ 應   妻財子水 ⚋
   兄弟戌土 ⚊      兄弟戌土 ⚊
   子孫申金 ⚋      子孫申金 ⚋
   官鬼卯木 ㄨ 世   子孫酉金 ⚊
   父母巳火 ㄨ      妻財亥水 ⚊
   兄弟未土 ⚋      兄弟丑土 ⚋
```

- 內卦動變 → **內卦反伏**
- 世臨卯木官，卯月月建臨之（看似官旺）
- **但**：申日 → 卯絕於申；世動化酉 → **卯酉相沖 + 金克木**（化回頭沖克）
- 反伏 + 絕日 + 化回頭沖克 → 三重凶象

**斷**：野鶴斷「此行不吉，不去者為上」。
**情節**：辭而不去 → 後又因他故隨去 → 七月城破，與官府同被害。
「不去而又去」恰是內反伏的標準走勢。

四、實例二：卯月己亥日，占升遷，得 臨卦 變 中孚卦（外卦反伏，但用神旺）

- 外卦動變 → **外卦反伏**
- 世臨卯木官，卯月月建臨 + **亥日生**（亥為木長生）
- 世爻不動 → 不化回頭沖克
- 用神極旺，反伏不畏

**斷**：「即升」。
**情節**：本月即由江西升任山東 → 應「卯官值月令」；未及一載，復任江西 → 應「外卦反伏，去之而復反」。

**兩例對比**：

| 項目 | 比之井 | 臨之中孚 |
|---|---|---|
| 反伏位置 | 內卦反伏 | 外卦反伏 |
| 世爻 | 卯木官，月建臨 | 卯木官，月建臨 |
| 日辰 | 申日（絕世）| 亥日（生世）|
| 世爻是否動 | 動，化酉（回頭沖克）| 不動 |
| 結局 | 城破同害 | 升遷又調任 |

—— 反伏不必畏，**怕的是反伏疊加化回頭沖克**。

五、伏吟的處理

伏吟比反吟更靜態：「**內外憂鬱呻吟之象**」。

| 占類 | 伏吟的應驗 |
|---|---|
| 功名 | 久困官途 |
| 求財 | 財源耗散 |
| 墳塋宅舍 | 欲遷不能 |
| 婚姻 | 不樂 |
| 疾病 | 呻吟 |
| 出行 | 不能移動 |
| 行人 | 在外憂鬱 |

**反吟 vs 伏吟的差別**：
- **反吟**：有沖有克，受克則凶
- **伏吟**：用神旺相，**沖開**之年月其志則伸；用神休囚，沖開反而憂鬱不吉

伏吟是「鎖住」的象 —— 旺者鎖住等開鎖，衰者鎖住則苦上加苦。

六、實例三：申月癸巳日，占父外任平安，得 姤卦 變 恆卦（伏吟）

- 戌土父動變戌土、申金動變申金 → **典型伏吟**（外卦同支動變）
- 占父：日建巳火生戌土（父） → 「日建生父，他事無礙」
- 何時歸：「伏吟欲歸而不能，**辰年可歸**」（辰沖戌，沖開伏吟）

**應期推算的伏吟解法**：**沖開本爻或變爻的對沖支之年月**。

七、李我平按語

李我平肯定野鶴的貢獻 —— **不看用神衰旺、竟以反吟為畏途**是錯的。透過比之井（反伏 + 化沖克 → 凶）與臨之中孚（反伏 + 官得地 → 連遷）的對比，「教人之法，何等透澈」。

==============================================================
旬空章第二十六（旬空的真假與「多占法」的破題）
==============================================================

旬空是六爻最被前人神秘化的概念之一。傳統書目把旬空分成十幾種 —— 真空、假空、動空、沖空、填空、墓空、絕空、害空⋯⋯讓占者眼花繚亂。本章野鶴用兩個求財案，丟出他的招牌方法 ——「**多占法**」：占到旬空無法決斷時，**命之再占**，兩卦合決。這是野鶴對占卜方法論最大的貢獻之一。

一、旬空表（六甲旬）

| 旬首 | 旬中十日 | 空亡兩支 |
|---|---|---|
| 甲子旬 | 甲子至癸酉 | 戌、亥 |
| 甲戌旬 | 甲戌至癸未 | 申、酉 |
| 甲申旬 | 甲申至癸巳 | 午、未 |
| 甲午旬 | 甲午至癸卯 | 辰、巳 |
| 甲辰旬 | 甲辰至癸丑 | 寅、卯 |
| 甲寅旬 | 甲寅至癸亥 | 子、丑 |

**旬空原理**：天干十、地支十二，十日一旬，每旬必有兩個地支落空。

二、野鶴的簡化

野鶴一句話**廢掉一半概念**：

> 「旺不為空，動不為空，有日建動爻生扶者不為空，動而化空、伏而旺相皆不為空。月破為空。有卦不動為空，爻反伏而被克為空，真空為空。」

**「真空」的定義** —— 旺月反臨無氣的爻：
- **春土**（春木旺克土）
- **夏金**（夏火旺克金）
- **秋木**（秋金旺克木）
- **三冬逢火**（冬水旺克火）

**判讀準則**：
- 用神**旺相而旬空** → 不是真空，會在「出旬」或「填實」之日應驗
- 用神**休囚又無動爻生扶又旬空** → **真空**，事不諧

三、野鶴的「多占法」

野鶴自承初學時遇旬空無法決斷 —— 想當作「到底全空」吧，卻又應在填實之日；想當作「不空」吧，卻又有真的到底全空的例子。

> 「後得多占之法，凡遇旬空，命之再占，卦吉者許之出旬而不空，卦得凶者許之空矣。」

**多占法的本質**：用第二卦作為「驗證信號」。第一卦旬空真實意義懸而未決，第二卦填補資訊缺口 —— **兩卦合判，覆蓋單卦的盲區**。

四、實例一：辰月乙卯日，占求財，得 家人 變 賁（再占 暌 變 損）

**第一卦：家人 → 賁**
- 用神世爻丑土財，丑爻旬空
- 表面：應爻巳火動 → 火生土（似有救）
- **但**：巳火動而化子水 → **子克巳**（化回頭克），元神被克
- 三月辰月之丑土，野鶴猶豫，**命之再占**

**第二卦：暌 → 損**
- 子水妻財伏於五爻未土之下，**伏而又空、空而被克**

**合決**：

> 「前卦丑則雖空而有氣，後卦子水財空伏於五爻未土之下，伏而又空，空而被克，知其無財而無疑也。」

**應驗**：「後果全無。」

**方法論價值**：第一卦如果單獨判，可以許「出空之日有財」；第二卦的「伏空 + 被克」**完全推翻了單卦判讀**。

五、實例二：子月辛亥日，占遠行求財，得 大畜（純靜）（再占 明夷 變 豐）

**第一卦：大畜（六爻安靜）**
- 世值寅木官鬼 → 子月、亥日皆生世
- 應爻子水妻財 → 世應相生 = 全美之卦
- **但**：寅爻旬空 → 古法「無故自空」應為大凶
- 野鶴猶豫，**命之再占**

**第二卦：明夷 → 豐**
- 世爻丑土動，化午火 → **回頭相生**
- 出月後不為破，開春寅卯月當令

**合決**：「此卦與前卦相同，此行大有財發。」

**應驗**：果於乙卯日起程，寅卯月間到彼地，**滿載而歸**。

**方法論價值**：第一卦六爻安靜 + 世爻旬空，按古法「無故自空」應斷大凶；多占法揭示真相是「旺財生世，腰纏萬貫之徵」。

六、野鶴自結

> 「多占之法，能消許多疑惑，不然只以前卦決斷者，無故自空，如入深淵大凶。實則旺財生世，腰纏萬貫之徵。」

—— **單卦遇旬空時的「凶」判斷有極高的誤判率，多占法是必要的修正機制**。

七、李我平按語

李我平補充三條重要原則：

1. **占近事**：許其出空之日而應之（一旬之內可定時）
2. **占遠之大事**：尚無定期，非旬可以定 —— 大象不吉才可謂「到底空」
3. **大象若吉**：太歲、月建可填之 —— 但「不如多占兩卦，實為妙法」

生旺墓絕章第又二十六（長生十二宮 —— 野鶴簡化版）
==============================================================

長生十二宮（長生、沐浴、冠帶、臨官、帝旺、衰、病、死、墓、絕、胎、養）原是命理體系裡標示五行盛衰循環的十二個階段。野鶴在六爻系統裡**只用四個**：**長生、帝旺、墓、絕** —— 其餘八個「不必用之」。又一個野鶴砍掉冗餘理論的時刻。

一、野鶴的簡化原則

> 「長生、沐浴、冠帶、臨官、帝旺、衰、病、死、墓、絕、胎、養。予得驗者，只驗生旺墓絕，其餘不驗，不必用之。」

| 完整十二宮 | 野鶴用 |
|---|---|
| 長生 | ★ |
| 沐浴 | 不用 |
| 冠帶 | 不用 |
| 臨官 | 不用 |
| 帝旺 | ★ |
| 衰、病、死 | 不用 |
| 墓 | ★ |
| 絕 | ★ |
| 胎、養 | 不用 |

二、四生四旺四墓四絕表（野鶴版）

| 五行 | 長生 | 帝旺 | 墓 | 絕 |
|---|---|---|---|---|
| **金** | 巳 | 酉 | 丑 | 寅 |
| **木** | 亥 | 卯 | 未 | 申 |
| **水** | 申 | 子 | 辰 | 巳 |
| **火** | 寅 | 午 | 戌 | 亥 |
| **土** | 申 | 子 | 辰 | 巳 |

**重要立論**：野鶴與李我平在此章末尾論證 ——**土寄生於申，與水同**（而非命理界常用「火土同源、長生在寅」）。理由是天時驗證 ——「土臨父母」（父母主雨）在申日、子日下雨；「土臨子孫」（子孫主晴）在申日、子日晴 → 申、子確為土的生旺位置。

三、四種觸發方式

「生旺墓絕」可由四個源頭觸發：

| 觸發源 | 範例（用神屬木）|
|---|---|
| **日辰** | 亥日占 → 木長生於亥日 |
| **月建** | 卯月占 → 木旺於卯月 |
| **動爻** | 卦中動出未爻 → 木墓於未 |
| **變爻** | 用神動而變申 → 木化絕 |

四、覺子的補充規則（不能機械套用！）

覺子（野鶴弟子）給出**反向修正規則**：

1. **金長生於巳，但**：金爻若休囚無氣 + 再遇巳午火多 → 「**烈火煎金，論克不論生**」
2. **金墓於丑，但**：若得未土沖動，或卦中土多生金 → 「**論生不論墓**」
3. **土絕於巳，但**：土若旺相或得日月動爻生扶 + 再遇巳 → 「**巳火反能生土，論生不論絕**」
4. **巳長生於寅，但**：日月動變諸爻又逢申金 → **寅巳申三刑** → 「**論刑不論生**」

**核心原則**：長生十二宮只是粗略的力量座標，**最終判斷必須回到「生克沖合的綜合力量代數和」**。

五、實例：午月己卯日，占妻病，得 震卦 變 豐卦

```
   妻財戌土 ㄨ 世   子孫巳火 ⚊
   官鬼申金 ⚋      妻財未土 ⚋
   子孫午火 ⚊      官鬼酉金 ⚊
   妻財辰土 ⚋ 應   妻財辰土 ⚋
   兄弟寅木 ⚋      兄弟寅木 ⚋
   父母子水 ⚊      父母子水 ⚊
```

- 用神：應爻辰土妻財（夫妻關係之他位）
- 「近病逢沖即愈」

**野鶴的三段應期推論**：

| 應期可能 | 機制 |
|---|---|
| 辰日愈 | 辰土逢值（用神當令）|
| 酉日愈 | 辰與酉合 |
| **子日愈**（實際應驗）| **辰土旺於子**（土帝旺於子）|

**情節**：病人「連日昏沈」，未在辰日、酉日見效，**竟於子日起牀**。

**為何應期在子日？**
- 土長生於申、旺於子（野鶴新立）
- 子日為辰土爻的**帝旺日**
- 帝旺日 = 用神力量最強之日 → 病愈

**此例是野鶴「土旺於子」立論的關鍵驗證** —— 病人實在於子日好轉，反推土確旺於子。

六、李我平按語：四大長生 vs 土寄生

| 主張 | 來源 | 評價 |
|---|---|---|
| 「火土長生在寅」 | 傳統命理 | 「未得其據」 |
| 「戌土生於寅、己土生於酉」 | 陰陽分支說 | 「實未得其據」 |
| 「山中有坤土生於申」 | 《易冒》 | 部分可取 |
| **「土長生於申、旺於子」** | **野鶴據天時驗證** | **「萬古之不易也」** |

李我平表態：野鶴是「**專以天時考證者**」—— 用實占應驗反推理論。

**方法論意義**：四大長生（金生巳、木生亥、水生申、火生寅）為「**理之正也**」；土寄生問題是體系的「縫隙」，野鶴以**多占驗證 → 反推理論**的實證主義方式選定「土生申、旺子」—— 這是六爻體系的方法論轉折。

各門類題頭總注章第又二十六（十八類分占的詞彙表 + 四處破執）
==============================================================

從卷二起野鶴將進入「分門別類」的實占階段 —— 占父母、占兄弟、占子孫、占功名、占求財、占婚姻、占胎產、占行人⋯⋯共十八類分占。本章是進入分占之前的「總注」—— 把後文反覆出現的關鍵字眼一次性定義清楚。

但更重要的是：野鶴在本章悄悄丟出**四處對古法的修正**（卦身、三墓、主象、變化），這是「以用神為主、不執形式」的具體展現。

一、用神之「旺」的四種來源

| 來源 | 內容 |
|---|---|
| 1. 臨日月 | 用神本氣即為日辰或月建之地支 |
| 2. 受日月生扶 | 日辰或月建來生用神 |
| 3. 動爻變爻生扶 | 卦中動爻或變爻來生用神 |
| 4. 長生／帝旺 | 用神坐長生或帝旺位 |

**野鶴提醒**：「不可指定旺于四時」—— 不要死守「春木夏火秋金冬水」。**只要符合上述四項之一，皆謂之旺**。

二、化吉與化凶的定義

| 化吉 | 化凶 |
|---|---|
| 化回頭生（變爻生本位）| 化回頭克（變爻克本位）|
| 化長生 | 化絕 |
| 化帝旺 | 化墓 |
| 化扶助 | 化空 |
| 化日月 | 化鬼 |
| —— | 化退神 |

**核心邏輯**：化吉是「自食其福」；化凶是「自食其禍」。

三、歲君與五位

| 名詞 | 定義 |
|---|---|
| 歲君 | 當年太歲 |
| 歲五 | 卦中第五爻為「太歲位」 |
| 五位 | 第五爻為「君位」（《易》九五之尊）|

主要用於占功名、國事、重大公事。

四、野鶴第一處修正：卦身 ≠ 世身（廢之，統一為世爻）

**古法**：身分「卦身」「世身」兩種，推算繁雜。
**野鶴**：「予試之不驗，後遇身者，**即以之為世**，非卦身、世身。」

**讀後文時凡見「身」字，直接讀作「世爻」即可**。

五、野鶴第二處修正：三墓的回歸力學

**古法的「三墓」**：世墓、身墓、命墓（命理化）
**野鶴的「三墓」**：
1. **用爻入日墓**：用神之五行入日辰之墓（如用神為火，戌日為火墓）
2. **用爻入動墓**：卦中有動爻為用神之墓（如午火動，卦中戌土動為其墓）
3. **動而化墓**：用神動，變爻為其墓（如午火動化戌土）

回歸到「用神 + 日月動爻」的力學分析，與卷一整體判讀邏輯一致。

六、野鶴第三處修正（最關鍵）：主象 = 用神，非世爻

**古法**：「以卦中世爻為主象，又名主事爻。」
**野鶴**：「**主象即定事吉凶之爻**」—— 占父母則父母爻為主象、占財則妻財爻為主象⋯⋯**主象＝用神**，依占事類別決定，不是固定的世爻。

**這是六爻體系從「世爻中心論」轉向「用神中心論」的轉捩點**。世爻只在「自占、世為求測者本身」時才同時是主象。

七、野鶴第四處修正：變與化同義

**古法**：
- 變：老陰 → 少陽（陰退而陽進）
- 化：老陽 → 少陰（陽進而陰退）
- 古人據此認為「變」「化」吉凶不同

**野鶴**：「予試之不驗。化進神、化生旺，雖化亦為吉。變鬼、變克，雖變亦為凶。」

**結論**：「各章所言變即是化，化即是變。」—— 後文「變」「化」二字完全同義。

八、為何這四處修正如此重要？

| 形式化古法 | 野鶴的方法論 |
|---|---|
| 卦身世身分別推算 | 統一為世爻 |
| 世墓身墓命墓 | 用爻入日／動／化墓 |
| 世爻為主象 | 用神為主象 |
| 變化方向定吉凶 | 看實質生克 |

**正面建設**：以用神為核心，以日月動爻為力學。
**反面破執**：所有形式化古法皆不可執。

讀者進入卷二、卷三、卷四的十八分占之前，必須先把這把「破執之刀」磨利 —— 否則十八分占中任何一個古法形式都會把讀者帶偏。

各門類應期總注章第又二十六（應期推算的十八條規則總綱）
==============================================================

前章是「詞彙表」，本章是**應期推算的完整工具箱**。六爻斷卦最終要回答的問題是「何時發生」 —— 不會應期，斷對了卦也只能說「將來吉／凶」，無法給出具體時點。野鶴把應期所有判讀規則一次性彙整，是後文十八分占應期推算的總目錄。

應期的本質是「**用神在何時被觸發 / 解開 / 補足 / 衝破**」—— 卦象只是潛在的力學圖，應期是這力學在時間軸上落地的瞬間。

二、四組對稱規則的判讀邏輯

**第一組：靜 vs 動（規則 1、2）**
- 靜爻沉睡 → 等「敲鐘」：值（自身當令）或沖（被驚醒）
- 動爻已醒 → 等「停下來」：合（被合住固定）或值（本氣兌現）

**第二組：太旺 vs 太衰（規則 3、4）**
- 太旺 → 等「墓沖」：墓是收斂、沖是潰決，旺到極處需要洩
- 太衰 → 等「生旺」：被生扶起來、被當令補足

**第三組：墓 vs 合（規則 5、6）**
- 入墓 → 沖開墓庫之日
- 六合 → 沖開合住之日
- **共通邏輯**：被困住的狀態，等「解封」之日

**第四組：月破 vs 旬空（規則 7、8）**
- 月破 → 喜填合或填實
- 旬空 → 喜填實或沖空
- **共通邏輯**：虛位的狀態，等「實補」之日

三、進退神應期的雙重可能（規則 11、12）

| 類型 | 應期 A | 應期 B | 為何雙重 |
|---|---|---|---|
| 化進神（申→酉）| 申月日 | 巳月日（合住申）或酉月日（變爻）| 「進」的瞬間 vs 進完之後 |
| 化退神（酉→申）| 申月日（退到的位置）| 寅日月（沖申，阻止退完）| 「退到」的瞬間 vs 阻止退的時點 |

**判讀提示**：
- 進神：力量充足時應 A（早成），不足時應 B（遲成、需借力）
- 退神：吉事應 B（被沖留住）、凶事應 A（退到位完成凶事）

四、克生鏈條應期（規則 9、10）—— 兩個極端

**規則 9：大象吉 + 局部受克 → 等「克神被克」之日**
- 卦不會立即兌現吉，需先「除掉克神」這個障礙
- 應期：克神被沖克之日（如克神為寅，遇申日沖寅）

**規則 10：大象凶 + 局部受克 → 等「克者得生」之日**
- 凶不會立即兌現，需要克者「得勢」才會發作
- 應期：克者當令日，或克者得援之日

**關鍵語**：「元神來助來扶，貴看用神衰旺；忌神來沖克，須觀元氣興衰。」

五、遠近分判（規則 15）

| 占事性質 | 應期刻度 | 例證 |
|---|---|---|
| 遠事 | 年、月 | 占十年內官職、占胎產、占大病 |
| 近事 | 日、時 | 占今日交易、占明日出行 |

應期刻度不是死的，要結合卦中力量總和、用神所處狀態綜合判斷。

六、再占一法（規則 16）—— 占卜倫理

> 「倘遇卦之不明，再占是一法。卦有恍惚者，再占一卦，不可妄斷。」

—— **野鶴對「占卜倫理」的重要表態**：寧可承認卦象模糊、需要再占，也不要為了顯示自己「能斷」而妄下判決。對應 [SKILL.md](../../../SKILL.md) 倫理基調。

呼應本卷旬空章第二十六的「多占法」 —— 兩者是同一個方法論的兩個應用面。

七、世空元動 vs 世衰元靜（規則 17、18）

**規則 17：世空元動 → 待元神逢值**
- 世爻空亡，但元神動 → 應期：元神本氣值日

**規則 18：世衰元靜 → 必然是元氣逢沖**
- 世爻衰，元神靜 → 應期：元神逢日沖之日（暗動，啟動後援）

**對稱性**：規則 17 是「動者待值」，規則 18 是「靜者待沖」—— 與規則 1、2 完全對稱，只是把對象從用神換成了元神。

八、應期推算的綜合心法

1. **辨狀態**：用神是動 / 靜 / 旺 / 衰 / 墓 / 合 / 破 / 空 / 進 / 退 中的哪一種？
2. **配觸發**：對應狀態的觸發時點是值 / 沖 / 合 / 墓 / 生 / 旺 中的哪一個？
3. **校刻度**：用年 / 月 / 日 / 時 哪一級時間單位來應？

**勿謂爻之不驗** —— 應期不準，多半不是卦不靈，而是讀者沒分清遠近、沒辨對狀態。

==============================================================
歸魂遊魂章第又二十六（八宮特殊卦的破執與用神中心論的點題收束）
==============================================================

卷一收尾的最後一章。歸魂、遊魂是八宮第七、第八卦的專稱 —— 古人據此衍生出一整套「形式化斷法」：見遊魂卦則斷遠行、見歸魂卦則斷不出疆⋯⋯野鶴在卷一最末以一章篇幅，把這套規則**剝皮見骨**地展示一遍，然後兩次斬釘截鐵地說：「**須以用神為主**」。

這不是對歸魂遊魂的全盤否定，而是對「捨用神、執形式」的根本警示 —— 也是整個卷一方法論的點題收束。

一、歸魂、遊魂的定義（八宮位置）

每一宮共八卦，按本宮卦 → 一變至五變 → 遊魂 → 歸魂的順序排列：

| 宮位 | 第七卦（遊魂）| 第八卦（歸魂）|
|---|---|---|
| 乾宮 | 火地晉 | 火天大有 |
| 坤宮 | 水天需 | 水地比 |
| 震宮 | 澤風大過 | 澤雷隨 |
| 巽宮 | 山雷頤 | 山風蠱 |
| 坎宮 | 地火明夷 | 地水師 |
| 離宮 | 天水訟 | 天火同人 |
| 艮宮 | 風澤中孚 | 風山漸 |
| 兌宮 | 雷山小過 | 雷澤歸妹 |

**位置邏輯**（簡述）：
- 一卦至五變：本宮卦自下而上依次變爻，象徵離家漸遠
- **遊魂（第七卦）**：不繼續向上變，反而回頭把四爻再變一次 —— 象徵「魂遊在外、半路掉頭」
- **歸魂（第八卦）**：在遊魂基礎上，下三爻全變回本宮卦的下三爻 —— 象徵「魂歸本位、回到家門」

二、古法對遊魂卦的形式化判讀

「遊魂行千里」—— 古人對遊魂卦的標準斷法：

| 占類 | 古法斷語 |
|---|---|
| 占身命 | 無安家樂業之所，漂泊不定 |
| 占行人 | 遊遍他鄉，不歸 |
| 占出行 | 行無定止，路途遙遠 |
| 占家宅 | 遷變不常，難以安居 |
| 占墳塋 | 亡靈不能安妥 |
| 占事業 | 心無定向，遷改不常 |

**古法核心隱喻**：遊魂 = 「魂在外」 = 動盪、遠行、不穩、不歸。

三、野鶴對遊魂古法的判決

> 「野鶴曰：**須以用神為主，然後以此參之**。若舍用神執此而斷者，謬也。」

**翻譯**：必須以用神為斷卦的核心，遊魂的「行千里、不安穩」只能作為**參考性質的附加意象**。如果捨棄用神，光憑「這是遊魂卦」就斷遠行不歸，**這是錯的（謬也）**。

**野鶴的兩段論**：
1. 不全盤否定遊魂的意象 —— 「然後以此參之」承認可作參考
2. 絕對禁止形式化套用 —— 「舍用神執此而斷者，謬也」

四、古法對歸魂卦的形式化判讀

「歸魂不出疆」—— 與遊魂相反：

| 占類 | 古法斷語 |
|---|---|
| 占身命 | 安守家業，不出遠門 |
| 占行人 | 將歸或已歸 |
| 占出行 | 行不出疆，難以遠行 |
| 占家宅 | 安穩定居 |
| 占墳塋 | 亡靈安妥 |
| 占事業 | 諸事拘泥，難以開展 |

**古法核心隱喻**：歸魂 = 「魂歸位」 = 收斂、固守、不出、回歸。

五、野鶴對歸魂古法的判決

> 「野鶴曰：**亦須以用神為主**。」

兩字之差（「亦」字）：與前段並列，重申用神中心論。前段已說過「謬也」，這裡只是同樣的原則用於歸魂卦。

六、野鶴為何如此強硬？—— 用神中心論的最終確認

| 形式化古法 | 野鶴的方法論 |
|---|---|
| 卦名定吉凶（遊魂主動、歸魂主靜）| 用神定吉凶（用神旺則成、衰則敗）|
| 卦象配標籤（遊魂 → 遠行）| 卦象作參考（用神為主，卦象為輔）|
| 看到歸魂就斷不行 | 看用神是否有力，再參歸魂意象 |

**舉例延伸**（野鶴未在本章舉例，依其方法論）：

```
情境：占出行得 火地晉（乾宮遊魂卦）
古法：遊魂行千里 → 斷遠行成
野鶴法：先看用神（出行以世爻、或子孫爻為用神）
  - 若世爻臨日月、得元神生扶 → 出行順利
  - 若世爻空破絕墓、忌神來克 → 出行不利
  - 然後再參遊魂意象 → 即使順利，可能路途較遠、滯留時間較長
```

換言之：**遊魂歸魂只能修飾應期或路徑的「形容詞」，不能作為吉凶的「動詞」**。

七、本章在卷一架構中的位置 —— 點題收束

卷一從第一章「八卦」到本章「歸魂遊魂」，野鶴的編排有層次：

1. **第一至第八章**（入門基礎）：建立卦象、八宮、納甲、六親、世應、動變、十二長生的基本語法
2. **第九至第二十五章**（核心理論）：用神、元神、忌神、生克、動靜、進退、空亡、月破、反吟伏吟⋯⋯所有力學分析工具
3. **第又二十六章三節**（總則收束）：詞彙表（題頭總注）、應期工具箱（應期總注）、形式破執（歸魂遊魂）

**為何把「歸魂遊魂」放在卷一的最後一章？**

因為這是卷一所有「破執」言論的**集大成**：野鶴前文已批判過卦身世身、變化區分、世墓身墓⋯⋯本章再把民間最流行、最易執著的「歸魂遊魂吉凶斷法」砍掉。砍完之後，卷一的方法論才真正完整：
- **正面建設**：以用神為核心，以日月動爻為力學
- **反面破執**：所有形式化古法（卦身、變化、世墓、歸魂遊魂）皆不可執

讀者進入卷二的十八分占之前，必須先把這把「破執之刀」磨利。

八、給現代讀者的閱讀提醒

| 讀古占書時常見的誤區 | 野鶴的對應提醒 |
|---|---|
| 看到「遊魂」就想到「漂泊不歸」| 先看用神，遊魂只是附加參考 |
| 看到「歸魂」就想到「安穩不動」| 先看用神，歸魂只是附加參考 |
| 為了套上意象而扭曲用神分析 | 「舍用神執此而斷者，謬也」|

**核心心法**：卦象是地圖，用神是指南針。地圖告訴你地形（遊魂多遷、歸魂多守），但走哪條路、能不能到，要看指南針（用神有力 / 無力）。
