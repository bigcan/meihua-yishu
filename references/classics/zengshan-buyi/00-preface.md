增刪卜易序

野鶴曰：卜易之道﹐乃伏羲、文王、周公、孔子四大聖人之心法也。得其精者可以參天量地；粗知其理亦可趨吉避凶。凡學卜者可以深求﹐亦可淺學。淺學者只要先學裝卦﹐知道動變及卦之六沖﹐卦變六沖看熟用神章中占何人占何事以何爻為用神；再看何為旬空、月破及春夏秋冬四時衰旺、生克沖刑卽知決斷禍福。

假令占功名者得旺官持世﹐或動爻作官星生合世爻﹐求名如拾芥耳。倘遇子孫持世﹐或子孫動于卦中﹐不拘占入場占升遷悉如水中撈月。占求財若得財星持世﹐或日月動爻作子孫生合世爻﹐或官鬼持世﹐財動生之﹐或父母持世﹐財動克之﹐皆許求財之易如摘枝耳。若遇兄弟持世及兄弟爻動于卦中﹐或世臨旬空月破﹐何異緣木求魚。

如占一年月令現任官者﹐宜官星持世﹐財動生之﹐皆許吉慶。若遇官鬼相克﹐日月動爻作子孫沖克世爻﹐或作官鬼沖克世爻﹐或世空世、官破官空﹐或世動化回頭克及子孫持世皆為凶兆。

士民而占流年者﹐最喜財爻及子孫爻持世﹐管許一歲亨通。若遇官鬼持世﹐得日月動爻作財星生合世爻者﹐必見災非。倘世破世空及鬼動克世多見凶災。兄動克世﹐口舌破財。

以上官府士民占流年者﹐合世之月則吉﹐沖世之月則凶。皆不宜世爻變鬼及化回頭之克定見凶危。又不宜財動化父﹐父化財爻﹐鬼化父母必有長上之災。弟兄變鬼﹐鬼變弟兄﹐防手足之厄。財化鬼﹐鬼化財﹐財化兄﹐兄化財﹐主傷克妻妾婢僕。子化鬼﹐鬼化子﹐父化子﹐子化父﹐小口有傷。靑龍天喜持世生世而有喜。虎鬼發動主孝服。騰蛇朱雀臨兄﹐鬼動而克世者﹐須防口舌。元武臨兄﹐鬼動而克世者﹐防賊盜及陰人。

如占避訟防非、仇人為害及行江漂海、深入險地旅店孤眠﹐窮鄕僻壤﹐投寺宿廟﹐或營中貿易﹐錯買盜物﹐或見鄰家火起﹐或聞瘟疫流行﹐防虎狼﹐防盜寇﹐或夜行早起﹐或險偸關﹐或已入是非之場﹐心憂禍患﹐或欲管閒事﹐恐惹災非﹐或人病家以防沾染﹐或誤服毒物恐致傷生﹐或已定重罪而盼赦﹐或已得險病病而防危﹐或問此物此藥可以服否﹐或問歹人烈馬傷害我否﹐凡遇一切防火慮患者﹐但得子孫持世及子孫動于卦中﹐或世動變出子孫﹐或世動化回頭相生﹐或官鬼動以相生﹐卽使身坐虎口﹐管許安如泰山﹐惟忌官鬼持世﹐憂疑不解。鬼克世災禍必侵。世動化鬼及化回頭克者﹐禍已及身﹐避之不及﹐惟世爻空者無憂﹐世爻破者不利。

占病者如自占病﹐若得世爻旺相或日月動爻生合世爻﹐或子孫持世﹐或子孫動于卦中﹐不拘久病近病﹐或求神或服藥﹐立保安康。近病者世值旬空或世動化空﹐或卦逢六沖及卦變六沖﹐不須服藥﹐卽許安痊。久病者官鬼持世﹐遇休囚﹐或遇日月動爻克世﹐或值旬空月破﹐世動化空化破或卦逢六沖﹐卦變六沖﹐或世動化鬼及化回頭克者﹐速宜救治﹐遲者扁鵲難醫。

占父母病以父母爻為用神。若得父爻旺相﹐或日月動爻生父母﹐或父動化旺﹐不拘久病近病﹐求神服藥立見安寧﹐近病者父爻值旬空﹐父動化空﹐或卦逢六沖﹐不藥而痊。久病者父爻值旬空月破﹐父動化空、化破﹐父動化財﹐財化父母﹐卦逢六沖﹐卦變六沖﹐或父爻休囚又出被日月動爻沖克﹐為子者須宜急急求醫﹐親嘗湯﹐勿遠離也。

占兄弟病者﹐若得兄爻旺相﹐或臨日月動爻相生﹐或動化旺化生﹐不拘病之遠近﹐立許全安。近病者﹐兄爻值旬空及動而化空﹐卦逢六沖﹐服藥卽愈。久病者兄爻值旬空月破及動而化空化破﹐卦逢六沖﹐卦變六沖﹐兄動化鬼﹐鬼動化兄﹐或兄爻休囚被日月動爻沖克﹐急急服藥求神﹐遲則難調理。

占子孫病者﹐子孫爻旺相﹐或臨日月﹐或日月動爻生合﹐或子孫爻化回頭生﹐化旺﹐不拘病之新久﹐服藥求神卽愈﹐近病者子孫爻值旬空及動而化空﹐卦逢六沖﹐卦變六沖﹐不藥而愈﹐出痘者不宜六沖﹐久病者子孫逢旬空月破及動而化空化破﹐卦逢六沖﹐卦變六沖﹐子孫動而化鬼﹐鬼化子孫﹐父化子﹐子化父及日月動爻沖克者﹐速宜服藥﹐遲則難于治矣。

占妻妾病者﹐以財爻為用神﹐財爻旺相或臨日月﹐或日月動爻相生﹐或財爻化子孫及化帝旺者﹐不拘久病近病﹐治之卽愈﹐近病者妻財逢旬空及動而化空﹐或爻逢六沖﹐卦變六沖﹐何須服藥﹐卽許災除。久病者財爻逢旬空月破及動而化空化破﹐卦逢六沖﹐卦變六沖﹐或財動化鬼﹐鬼化財爻﹐兄動化財﹐財化兄弟﹐名醫亦難取效。

凡占三黨六親及官長、師生、婢僕諸人之病﹐皆于用神章內以取用神。

占朋友、外人以應爻為用神﹐理之常也﹐往往多有不驗者何也？疑因不甚關切﹐不誠之故耳。

野鶴曰：客有問于予曰：「據爾之言﹐占卜極易事也﹐卽如占功名得旺官持世以成名﹐子孫持世而失望。占疾病近病逢沖逢空﹐不藥而愈；久病逢沖逢空靈丹莫救。如若得此﹐顯然者自是不難知矣。倘占疾病不逢六沖﹐用神不遇旬空﹐旺不旺而衰不衰﹐凶不凶而吉不吉。又如占功名﹐官與子孫皆不持世﹐六爻亂動﹐財父同興﹐何以決之？」予曰：爾若垂廉賣卜﹐每日數卜之占﹐未必盡得顯而易見之卦﹐凶中藏吉﹐吉處藏凶者有之﹐必須奧理深求﹐細心參悟﹐爾欲自知趨避者﹐必然卦不亂占﹐心無雜念﹐每遇一事﹐卽刻卜之﹐神不欺人。如若間有卦之恍惚﹐次早竭誠再卜﹐再遇恍惚﹐還可再卜﹐自然響應﹐只不可心懷兩事而占﹐一念至誠則應﹐若占兩三事者﹐則不靈也。又如占疾病更容易耳﹐一人有病﹐一家俱可代占﹐但有一卦爻逢六沖或卦變六沖﹐或用神值旬空或用神動而化空者卽愈﹐久病逢此者難治。又如防災慮患﹐但得子孫持世﹐便與霹靂同居﹐管許安然無恙﹐有何難耶？客曰：再占有瀆之﹐不敢再三﹐何敢連占幾日？予曰：因此一語﹐誤盡卜卦之人﹐豈不聞三人占聽二人之言﹐一事旣可三處而占﹐何妨再占？然亦有不可再瀆者﹐以此一事一刻而再占也﹐須于次日再卜可也。又有連日亦不可再瀆者﹐如占功名﹐已得子孫持世﹐我心不悅﹐必欲求其官鬼持世而後已﹐此則謂之再三瀆也。然予亦有見其再三瀆者﹐未見神之不應也﹐予因少年復功名占過七次﹐竟有六次而得子孫持世﹐此乃神不厭我多問而屢報也。又有厭予多問者﹐如我問求財卦已明﹐現有財我心知矣﹐我再問之神不告矣﹐而又以我未占之事告我也。如一日占求財﹐旺財持世﹐是我明知辰日得財﹐次日再占一卦﹐果于辰日得財否？卦得申金兄動而不得﹐是何說耶？及到辰日得財﹐至申日因他事而破財。而悟辰日之得財﹐次日而再問之﹐神不告矣﹐報我申日而破財也。故知再三瀆者﹐神亦不見責而又報我未問之事也﹐此事極多。予著此書傳後賢之祕法者﹐無他法也﹐教學者凡遇卦之恍惚﹐心若未明﹐多占無礙﹐倘卦中已明現﹐不可再瀆。至于占病者﹐一人有病﹐一家俱可代占﹐自有顯然之卦﹐再者遇事卽占﹐乘此心而未亂﹐不可多積事情于心﹐事多心亂卽非一念之誠也。教其深學卜者﹐後有分占之法及予所辟諸書之謬﹐宜細味之﹐此皆予四十餘年須臾不離以得之也﹐實先賢之所未傳。須宜通前徹尾﹐細心詳悟﹐自然巧奪天工﹐參天地之化育﹐測鬼神之隱微而不難矣！

旬空計算法

甲子：子-甲=0﹐癸酉﹐酉-癸=0﹐戌亥（11,12）空

甲戌：戌-甲=10﹐癸未﹐未-癸=-2﹐申酉（9,10）空

甲申：申-甲=8﹐癸巳﹐巳-癸=-4﹐午未（7,8）空

甲午：午-甲=6﹐癸卯﹐卯-癸=-6﹐辰巳（5,6）空

甲辰：辰-甲=4﹐癸丑﹐丑-癸=-8﹐寅卯（3,4）空

甲寅：寅-甲=2﹐癸亥﹐亥-癸=2﹐子丑（1,2）空

計算公式是：（地支+11-天干）%12 得到的餘數確定旬空起始位置﹐這樣就不需要判斷是否餘數是否負數

干支配是陽干配陽支﹐陰干配陰支﹐而且是完全匹配﹐干支差若得到奇數﹐則說明提供的干支有誤

三合﹝生旺墓﹞﹐絕

木局：亥卯未﹐申；火局：寅午戌﹐亥；水土局：申子辰﹐巳；金局：巳酉丑﹐寅。

華蓋：三合之三地支見其中之墓位地支﹐如：亥卯未見未。

驛馬：三合之三地支見﹝旺位地支﹢2﹞之地支﹐如：亥卯未馬在巳。

祿神：天干中的陰干陽干分別見地支中的相同五行屬性的陰支陽支﹐如：甲祿寅﹐乙祿卯；辰戌丑未不成祿﹐戊寄祿于丙﹐己寄祿于丁。

天乙：

月德：三合之三地支見其三合成局相同五行之陽干。

天德：三合局中﹐長生位地支見其三合局所屬五行的陰干﹐墓位地支見其三合局所屬五行的陽干﹐旺位地支見其三合局所屬五行的絕位地支。

羊刃：羊刃又稱陽刃。陽干見其相同五行之三合局之旺位地支。戊獲刃于丙。陰干無刃。

國印：祿位+9

將星：三合局之三地支見其旺位地支。

文昌：甲巳乙午文昌位﹐丙戊申宮丁己雞。庚豬辛鼠壬逢虎﹐癸人見卯上云梯。

反吟伏吟

卦有卦之反吟﹐爻有爻之反吟。反吟者﹐實際上是卦變沖克、爻變沖克。卦之反吟者﹐內外動而反吟。有乾變巽﹐巽變乾；坎變離﹐離變坎；震變兌﹐兌變震；坤變艮﹐艮變坤。坤艮兩卦皆土﹐為何相沖？《卜筮正宗》云：「艮卦坐于東北﹐艮右有丑﹐左有寅；坤卦坐于西南﹐右有未﹐左有申。二卦相對﹐有丑未相沖﹐寅申相沖。」

爻之反吟者﹐內外卦之反吟。如子變午﹐午變子﹐丑變未﹐未變丑；巳變亥﹐亥變巳；辰變戌﹐戌變辰。爻之反吟﹐與卦之反吟的不同之處是：卦變反吟﹐就是卦變相克。如乾變巽﹐為金克木﹐而卦中六个爻互相之間並不一定者變為相沖相克。而爻之反吟﹐就是卦和爻同時變為相沖相克。如坤變巽、巽變坤、觀變升、升變觀等。其卦中之爻﹐互相發生你沖我克。

卦之伏吟﹐則是卦變﹐其六爻地支五行不變。卦有內外伏吟、外卦伏吟和內卦伏吟三種。

序文（野鶴曰）現代語譯

野鶴老人說：卜易這門學問，是伏羲、文王、周公、孔子四位聖人的心法傳承。深入鑽研可以參透天地之理，即使粗略掌握也能趨吉避凶。學卜的人可以深求精微，也可以淺學致用。

淺學者只需先學會「裝卦」（即起卦並標註納甲六親世應），知道動爻變化與「卦逢六沖、卦變六沖」這兩種重要卦象，熟讀「用神章」明白占什麼人、什麼事該以哪個爻為用神；再瞭解什麼是旬空、月破，以及春夏秋冬四季的衰旺、生克沖刑，就能夠判斷禍福了。

一、各類占斷的吉凶總則

占功名：
- 旺官持世，或動爻作官星生合世爻 → 求名容易（如拾芥）
- 子孫持世，或子孫動於卦中 → 求名困難（如水中撈月）

占求財：
- 財星持世，或日月動爻作子孫生合世爻 → 易得
- 官鬼持世+財動生 → 易得
- 父母持世+財動克 → 易得（注意：父母被克意指文書受損，但對求財則是「財爻發力」）
- 兄弟持世或兄弟爻動，或世臨旬空月破 → 難得（如緣木求魚）

占在任官員的流年：
- 宜：官星持世、財動生官
- 凶：日月動爻作子孫沖克世爻、官鬼沖克世爻、世空官破、世動化回頭克、子孫持世

占百姓流年：
- 宜：財爻或子孫爻持世（一歲亨通）
- 凶：官鬼持世、世破世空、鬼動克世、兄動克世
- 合世之月吉，沖世之月凶

二、化爻凶兆通則

以下化爻組合主凶：
- 世爻變鬼、化回頭克 → 凶危
- 財動化父、父化財爻、鬼化父母 → 長上之災
- 弟兄變鬼、鬼變弟兄 → 手足之厄
- 財化鬼、鬼化財、財化兄、兄化財 → 傷克妻妾婢僕
- 子化鬼、鬼化子、父化子、子化父 → 小口（子女）有傷

三、六獸應驗

- 青龍天喜持世生世 → 有喜
- 白虎臨鬼發動 → 主孝服
- 螣蛇朱雀臨兄，鬼動克世 → 防口舌
- 玄武臨兄，鬼動克世 → 防賊盜及陰人（女子之事）

四、防災慮患的萬能解

凡占避訟、防仇、出行險地、防火、防瘟、防虎狼、防盜寇、防危、防毒、求赦、問藥、問危險人物等任何「防避災患」之事：
- 子孫持世，或子孫動於卦中
- 世動變出子孫
- 世動化回頭相生
- 官鬼動而生世
任何一項皆主「即使身坐虎口，亦安如泰山」。
唯獨忌官鬼持世，憂疑不解；鬼克世則災禍必侵；世動化鬼或化回頭克則禍已及身、避之不及。
但世爻空者反而無憂；世爻破者不利。

五、占病通則

自占病：
- 世爻旺相、日月動爻生合世爻、子孫持世或動 → 服藥求神立保安康
- 近病者：世值旬空、世動化空、卦逢六沖、卦變六沖 → 不藥而愈
- 久病者：官鬼持世休囚、日月動爻克世、世臨旬空月破、世動化空化破、卦逢六沖卦變六沖、世動化鬼、世化回頭克 → 速救，遲則扁鵲難醫

占父母病：用神父母爻（同上原則，置換用神）
占兄弟病：用神兄弟爻
占子孫病：用神子孫爻（注意：出痘者不宜六沖）
占妻妾病：用神財爻
朋友/外人病：用應爻（不準者多因關切不深、誠意不足）

六、問答錄（重要！關於「能否再占」的爭議）

客人問野鶴：「依您所說，占卜實在容易，凡是顯著卦象自然好判斷。但若占病不逢六沖、用神不旬空，不旺不衰、不凶不吉；又如占功名，官與子孫都不持世、六爻亂動、財父同興，該如何判斷？」

野鶴回答：
1. 賣卜為業者每日多卜，必然遇到「凶中藏吉、吉中藏凶」之卦，須細心參悟、深求奧理。
2. 自占者若做到「卦不亂占、心無雜念」，遇事即占，神不欺人。
3. 若遇卦象恍惚不明，次日竭誠再卜，再不明還可再卜，自然有應。
4. 唯不可「心懷兩事而占」，一念至誠則靈。

關於「再三瀆」的傳統警告，野鶴明確反駁：
- 「三人占聽二人之言」說明一事可三處而占，何妨再占？
- 不可再瀆的是「同事同刻而再占」，須隔日再卜。
- 不可再瀆的另一種是「卦中已明、自己不悅、執意再求」，這才叫真正的瀆。
- 野鶴自己年輕復功名占了 7 次，6 次得子孫持世，神並未厭多問而屢報。
- 神也會「報未問之事」：例如占財已得辰日得財，次日再占，卦得申金兄動表示破財，後來辰日確實得財而申日破財。

野鶴的結論：
- 凡遇卦之恍惚、心若未明，多占無礙
- 卦中已明現，不可再瀆
- 占病者一人有病一家可代占
- 遇事即占，乘心未亂之際；事多心亂則非一念之誠
