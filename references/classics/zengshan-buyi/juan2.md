月破章第二十七（月破爻的真破假破）
==============================================================

「月破」的定義：每月「月建」（當令地支）所沖之爻即為月破。逐月對照如下：

| 月 | 月建 | 月破爻 |
|---|---|---|
| 正月 | 寅 | 申 |
| 二月 | 卯 | 酉 |
| 三月 | 辰 | 戌 |
| 四月 | 巳 | 亥 |
| 五月 | 午 | 子 |
| 六月 | 未 | 丑 |
| 七月 | 申 | 寅 |
| 八月 | 酉 | 卯 |
| 九月 | 戌 | 辰 |
| 十月 | 亥 | 巳 |
| 十一月 | 子 | 午 |
| 十二月 | 丑 | 未 |

**諸書古法**：月破爻如悖時之木，逢生不起、逢傷更重，靜伏皆無透露之機，動而作忌神不能為害、作變爻不能傷克動爻。簡言之 ——「月破即廢」。

**野鶴反論（核心立場）**：神兆機於動。事無吉凶則爻不動，動則必有禍福之基。月破爻動，仍能傷他爻；動而又化，仍能傷動爻。判別真破假破有五條：
1. **目下雖破、出月則不破** —— 月破限月內；換月即解除
2. **今日雖破、實破之日則不破** —— 月破爻臨「實破之日」（即月破爻本身值日）反而充實有力
3. **逢合之日則不破** —— 月破爻得合之日，破解
4. **近事應實破之日，遠事應實破之年月** —— 月破不必死於本月
5. **唯靜而不動、又無日辰動爻生助** —— 此時才是「到底而破」之真破

**典型案例摘要**：
- 亥月己丑日占功名，得兌之訟。世動官動本應有官，但官值月破、世遇旬空。野鶴疑而再占得水地比，官星臨世。「前卦官臨月破、定於實破之年」 —— 果於巳年（實破之年）承襲世職
- 辰月戊子日占父歸期，得乾之夬。父持世又破又化空，古法判「不能歸」；野鶴斷朱雀臨父動則卯日有信、午未日必歸 —— 果於卯日得信（破而逢合）、乙未日到家（化未出空）
- 寅月丙辰日占功名，得地澤臨。亥水生官但被日克，子年沖實亥水時起用 —— 果於甲子年起用至卯年開督府；後逢辰年子水入墓、太歲克水，三月罷職

**應期判斷的關鍵原則**：月破爻動，必有應期；要找的是「實破之年／月／日」或「合之日」，不是當下廢棄。

**李我平補注**：易冒、易林補遺皆主「月破之爻不拘衰旺更作凶推、有亦如無」，但實占可見「月破爻關乎一生凶吉」 —— 月破絕非「百無所用」。

飛伏神章第二十八（用神不上卦時的取伏法）
==============================================================

當所占之事的用神不在六爻內，需取「伏神」。野鶴給出的取伏步驟，與舊書系統不同：

**取伏步驟**：
1. 用神不現 → **先看月建日辰是否就是用神**。若日月即用神，直接取日月，不必尋伏神
2. 若日月也不是 → **回到本宮首卦尋伏神**（八純卦六親俱全）
3. 取本宮首卦對應爻的六親，伏於現卦同位爻之下；上者為「飛神」，下者為「伏神」

**範例**：占得天風姤（乾宮），想看妻財。姤卦六爻無寅卯木（乾宮以木為財）→ 回到乾為天，二爻寅木妻財 → 此寅木伏於姤卦二爻亥水之下，「亥水為飛、寅木為伏」，亥水生寅木謂之「飛來生伏得長生」，伏神有用。

**伏神有用六條**：
1. 伏神得日月生
2. 伏神旺相
3. 伏神得飛神生
4. 伏神得動爻生
5. 日月動爻沖克飛神（飛神被破，伏神出露）
6. 飛神空破休囚墓絕（飛神壓不住伏神）

**伏神無用五條**：
1. 伏神休囚無氣
2. 伏神被日月沖克
3. 伏神被旺相飛神克
4. 伏神墓絕於日月飛爻
5. 伏神逢空逢月破

**黃金策曰**「空下伏神易於引拔」近理，但**不止飛神空亡才出**，飛神破、休、囚、墓皆易引出伏神 —— 因飛神弱則無壓力，伏神反易出。

**野鶴破執立場**：「用神不現，即有伏神亦不用之，再占兩卦用神必現」 —— 即放棄取伏神，改用 **多占合斷**。例：醫滿座占父病，連占三卦皆得亥水回頭克父，三卦雷同 —— 知今冬危而非目前危；應目前之愈用伏神之說可棄。

**李我平補注**：取伏神之衰旺刑沖墓空亦難把握，多占法是真秘訣；不可「用神不現尋伏，伏無用又尋互卦，互不得又尋於化」般層層附會。

進神退神章第二十九（動而化進、化退的吉凶倍減）
==============================================================

「進神退神」 = 動爻化變後的同五行十二地支對照。

| 進神 | 退神 |
|---|---|
| 亥化子 | 子化亥 |
| 寅化卯 | 卯化寅 |
| 巳化午 | 午化巳 |
| 申化酉 | 酉化申 |
| 丑化辰 | 辰化丑 |
| 辰化未 | 未化辰 |

**判定原則**：所喜者宜化進（吉凶倍增），所忌者宜化退（吉凶漸減）。化進如春木枝繁葉茂，化退如秋木凋零。

**野鶴四進四退判定法**（破古法「動日月化空破不進不退」的核心）：

進神四種：
1. 動旺相而化旺 —— **乘勢而進**
2. 動休囚而化休囚 —— **待時而進**（休囚之日進）
3. 動／變一爻值衰 —— **待旺相之日而進**
4. 動／變一爻空破 —— **待填實之日而進**

退神四種：
1. 動旺相而化旺相、又得日月動爻生扶（占近事）—— 得時而**不退**
2. 動休囚而化休囚 —— 及時而退
3. 動／變一爻旺相 —— 待休囚之時而退
4. 動／變一爻空破 —— 待填實之日而退

**典型案例摘要**：
- 申月癸卯日占鄕試得恆變大過。酉金官星持世旺相，得九五官化進神生扶 —— 不獨今秋折桂、來春佔鼇頭，果連捷
- 卯月乙丑日占婚成否。妻財持世化進神、巳火子動生世，但巳化子水回頭克 —— 必待午日沖去子水、午又合世 —— 果於午日允婚
- 申月辛卯日占病得澤天夬，世化退神疑為退災；野鶴駁：「動變皆屬秋金、占近事不可言退」，果於辰日用針即愈
- 辰月乙未日占子病，父化退神 —— 死於次日。**「化退神非病退也，乃精神命脈漸漸消枯之象」**
- 戌月己卯日占母血崩，丑土子化破為「絕其源」 —— 卒於丑月

**李我平補注**：易冒提出「大進不進、不能進」三進神理；後又謬言「動日月化空破則不進」、「動破變日月則退之不及」。實占可見「動爻臨日月，逢空不空逢破不破，何況化空破？日月如天，雖化空破乃浮雲掩日，實破之期即雲開霧散」 —— 化日月名為**化旺**，後日填實愈強。

隨鬼入墓章第三十（世入鬼墓 = 大凶？野鶴的世旺則破）
==============================================================

**三墓**：日墓、動墓、化墓（用神或世爻入此三墓之一）。

**入墓的對象**：世爻、本命（自占者的本命年支）、卦身、世身，皆可隨鬼入墓。

**覺子（李文輝）破古之言**：執三墓加四主象論，「逢辰戌丑未之日竟不敢占卦」 —— 因卦中總有墓爻、總有世命二身可入。實占四十年存驗驗證後，得結論：

**野鶴鐵則**：
1. **世爻用爻旺者**，雖隨鬼入日墓、動墓、化墓，**亦不死**
2. **休囚被克又入墓者**，是也（真凶）
3. 墓神逢日月動爻沖破，**亦非真墓**（破網易出）

**自占看世、代占看用神**：旺相者三墓不真，唯休囚被克又入墓者才是真凶。

**典型案例摘要**：
- 申月戊辰日占夫病（癸亥命）。亥水官星墓於辰日，夫星夫命皆入墓，古法必死；野鶴：「辰日沖戌生申，明日己巳沖起亥水得遇金生」 —— 果於次日大愈
- 戌月甲寅日占會試。世爻午火隨官入三墓（動墓、化墓、月德墓） —— 反斷「明歲辰年沖開墓庫定中狀元」，果中
- 未月戊辰日占重罪可赦否。世爻寅木隨鬼入動墓又化墓，古以為凶，野鶴以為吉：日月生世、丑墓月破「破網易出」 —— 果次年辰月蒙赦

**占類分門應用**（世旺者沖墓之年月即成、世空破休囚者反凶）：

| 占類 | 世旺 + 沖開墓庫之年月 | 世空破休囚 |
|---|---|---|
| 功名 | 沖開墓庫之年成名 | 終難成 |
| 身命 | 沖開墓庫之年月發跡 | 終身寂寞 |
| 出仕／出行 | 沖開墓庫之日月遂心 | 多見去而不返 |
| 求財 | 沖墓之日月而成 | 終無成日 |
| 婚姻 | 沖墓之日月成 | 病於他鄕或流落 |
| 胎產（子孫入墓）| 沖墓之日月即生 | 妻遭產厄、子必危亡 |
| 入公門 | 沖墓之年月如心 | 常遭枷鎖刑獄 |
| 家宅／祖塋 | 沖開之年月興家 | 身衰家破 |
| 訟事 | 旺者得理 | 被克者遭刑獄 |

**附則**：諸占世旺墓爻空破者，待墓爻填實之日月而吉；世空破休囚者，亦待墓爻填實之日月而凶。

獨發章第三十一（一爻獨動之應期判讀）
==============================================================

**定義**：五爻俱動、一爻不動 = **獨靜**；五爻不動、一爻獨動 = **獨發**。

**野鶴立場**：事之成敗由乎用神，應期遲速亦由乎用神；獨發獨靜古有驗、予亦試驗，**皆在驗事應之後**「如會神機」 —— 不敢「初即執之而斷禍福與應期」。卦得獨靜雖少，獨發者多，**「如舍其用神，執獨發決事者，謬也」**。

**核心方法論**：獨發爻可作應期的線索（合、值、沖之日），但不可作為獨立的吉凶判定。野鶴常與「多占合斷」搭配使用。

**典型案例摘要**：
- 辰月甲戌日占迎請父王靈柩允否？得獨發寅木化丑父。門客斷「正月得見父靈」；野鶴駁：「卦中父爻持世被寅木克，身不能動則靈不能動，須待沖開寅木之年月」 —— 再占得澤火革變既濟，世化申金回頭生 —— 申月得允、酉歲迎靈而歸
- 午月甲申日防漲水沖麥。戌土子孫一爻獨發，子動克身邊鬼示「憂非」 —— 卯日大晴（動而逢合之日）
- 辰月甲午日開煤，亥水獨發化辰土。原斷六月見煤未驗，至亥年辰月始得見煤 —— 應「獨發亥水化辰土」之年月雙合

**李我平補注**：易冒以「吉凶之生由乎動」而**重於動輕於用**，又云「不離用亦不執用爻」、「獨發獨靜可用，時日吉凶須審用卦」 —— 既曰用用神之、又曰不用用神主，「既無定何以為法？」批易冒首尾矛盾。

兩現章第三十二（用神兩現的取捨）
==============================================================

**用神兩現**：所占之事在卦中出現兩個或多個對應爻（例：占父，卦中兩爻父母）。

**古法**（傳統取捨原則）：
1. 舍休囚、用旺相
2. 舍靜、用動
3. 舍月破、用不破
4. 舍旬空、用不空
5. 舍被傷、用不傷

**野鶴反例（破執）**：「得其驗者，應乎旬空月破」 —— **舍不空而用旬空、舍不破而用月破**。

**典型案例摘要**：
- 未月庚子日占求財得風天小畜，未財應、辰財旺。許次日辛丑沖動未財日得，卻得於辰土出空之日 —— **舍不空而用旬空**
- 未月甲午日占升遷得師之渙，官星兩現一空一破。許「辰年官出空高升」 —— 果於辰年調河南、五月又調回楚、十月升督府，「皆應實空之年」
- 亥月丙午日母占子何日脫厄得豫之歸妹。卦中三現子孫俱生世，午逢日建而靜、兩爻巳火逢月破 —— **許巳年脫厄（用月破，應實破之年）**，果脫厄於巳年

**野鶴方法論**：「予竟以月破而斷年者，非只此一卦也」 —— 月破爻可作年應，前提是與多卦合斷、留神記前卦。**「卜易者一則全其靈機達變，二則卦要留神記之」**。

增刪黃金策千金賦章第三十四（誠意先生原賦增刪）
==============================================================

《黃金策千金賦》為明初劉伯溫（誠意伯）所著，是六爻最具規範性的歌訣之一。野鶴對其逐條檢驗增刪，本章是**「卷二最系統的傳統批判」**。重要條目摘要：

**一、太過不及（用爻多寡）**
- 太過者損之斯成 ——「主事爻太多，待寅卯月日克制以成事」（土多則待寅卯木克）
- 野鶴補：**亦得逢辰月辰日成事 —— 辰乃土墓庫，用爻重疊須墓庫收藏**
- 不及者益之則利 ——「主事爻只一位又不旺，須日辰動爻生之」（如旱苗得雨）
- 但要原有根蒂，**衰弱無助亦難生**

**二、太歲與卦身**
- 古法重太歲，野鶴：太歲只在「占年運、占官事朝廷大事、面聖請功」時為重；其餘**「太歲尊而不親，皆不及日月」**
- 太歲不能生衰爻、不能制旺爻；逢月破即破、逢旬空即空
- **「不驗者身位、宜刪而不宜存」** —— 野鶴明確刪去卦身判定法，「予用世爻百發無不百中」

**三、世應關係**
- 世為己、應為人，大宜生合
- **「自占者世為主、應與我無干」**（自陳功名與應何干？此迂腐之論）

**四、動變吉凶**
- 動為始、變為終，最怕交爭
- 應位遭傷不利他人之事，世爻受制豈宜自己之謀
- **世應俱空，人無準實**
- 內外競發，事必翻騰（須再占合斷）

**五、空、合、墓的應期規則**
- 爻遇旬空逢沖而有用（沖空則實）
- **動空化空，皆成凶吉**（野鶴更：動不為空、動而化空亦不為空，應沖空實空之日）
- 合遭破以無功；如逢合住沖破成功
- 動逢合而絆住 —— 沖開之日月吉凶俱成
- 靜逢沖而暗興
- 入墓難克 ——「逢沖開墓庫日依然木被金傷」
- **帶旺非空** —— 旺爻雖空不作空

**六、貪生貪合**
- 貪生忘克／貪合忘刑 —— 用爻臨卯木遇申動，卦中動出亥水則申貪生不克木

**七、爻與日月關係（核心）**
- **「日傷爻真罹其禍，爻傷日徒受其名」** —— 日月如天如君，六爻如臣如萬物
- 殺生身莫將吉斷，用克世勿作凶看 —— 但占功名官鬼克世「豈可勿作凶看」

**八、化變五情況**
- 化進神諸占皆吉
- 化官鬼吉凶有二：占功名旺得官、占其他則變鬼
- 忌回頭之克我（占病、占壽、占年運遇之十有九死）
- 化長生／化旺諸占皆吉；**土寄生於申、旺於子**
- 化墓化絕：旺相或臨日月不為大害，沖開／逢生之日成
- 化空：實空之日必成；近病出空即愈

**九、占卜倫理（七條）**
1. **筮必誠心，子日不忌** —— 黃金策「只在誠與不誠，不在子與不子」
2. **占勿二念，早晚何妨** —— 一念只占一事
3. **我事不可命人** —— 心事不可叫他人代占
4. **他是誠心、我讓他占** —— 然「我既起念，神必應我」
5. **他事由他動念、慎勿提他** —— 我提醒他叫他占，是我之念
6. **我占必以直告，卻莫昧己** —— 不可隱瞞真實所占
7. **占遠應近、占此應彼，必須詳察** —— 神有舍小事報大事、舍此應彼

**十、結論**
- 「大匠誨人必以規矩，卜者決斷全在靈機」
- **「吉凶神煞之多端，何如生克制化之一理」** —— 與星煞章呼應
- 用者重之、不用者去之，「切不可膠柱而鼓瑟」

**李我平補注**：「黃金策千金賦乃誠意先生所著，昔有人曰：『有能易一字者，予以千金』。後因刻板錯誤以訛傳訛致白玉生瑕。野鶴前後增刪另作解注以成全璧 —— 從此名之**黃金賦**可也」。

==============================================================
天時章第三十五（占天氣 = 學易入門驗證）
==============================================================

**核心立場**：「天時人事皆不離五行。天時得驗、人事驗矣」 —— 野鶴主張**初學六爻先占天時練手**，因天時驗應快速（目下即可驗）、人事驗應緩慢，由天時入門最易掌握五行機制。

**五行對應**（天時占的核心對照表）：

| 六親 | 天象意義 |
|---|---|
| 子孫 | 日月星斗 → 動則萬里晴光（萬里無雲） |
| 父母 | 雨雪霧霜 → 動則八方潤澤（厚雲、雨、雪、霜）|
| 妻財 | 晴明（克父爻而生子孫）|
| 官鬼 | 雷霆霧電（生父爻、應雷電）|
| 兄弟 | 風雲（雲淡風輕、非晴非雨） |

**重要修正**：黃金策原為「妻財發動，八方咸仰晴光」，野鶴**改為「子孫發動，萬里晴光」**，因子孫為財之元神，財動雖晴若子孫休囚空破亦不大晴。

**重要規則整理**：
1. **父持月建必然陰雨連旬** —— 野鶴試不驗：父化退神則雨將止
2. **晴利父官而化退** —— 父動化退神勢必晴
3. **木動生風、金空則響、火鬼動則電掣金蛇、金化金者迅雷霹靂**
4. **動者逢值逢合之日、靜者逢值逢沖之日** —— 但月破爻靜則不應，動而破應實破之日
5. **沖則應、填則實、最喜動空** —— 動而空非無功
6. **競發父官，連朝猛雨** —— 父鬼重疊或父變鬼鬼變父，必連旬風雨
7. **多逢財子、累日明晴**
8. **卦得反吟晴雨終須反復** —— 反吟卦不可斷一日陰晴
9. **半晴半雨：卦中財父同興；又雨又晴：父母子孫皆動**
10. **冬天不可以子孫為霜雪** —— 黃金策舊注謬，冬令占晴亦以子孫為日

**用神兼看**（野鶴反駁古法「應為太虛逢空雨晴難定、世為大地受克天變」）：
- 占雨用父爻、占晴用子孫財爻 —— **世應何干？**
- 偶然湊合一兩卦不可為法，宜刪此兩句

**應期類型**：
- 動者逢合之日（如午火動應未日）
- 動而空者逢沖空之日或填實之日
- 動而被克者逢沖去克神之日
- 父動逢長生之日（如卯木父動應亥日）
- 用爻入墓沖開之日

**典型案例摘要**（取最具代表性者）：
- 巳月甲戌日占何日雨，得小過之旅。戌父動而於辰時暗動 → 辰時起雲、辰末巳初雷雨（戌化巳火鬼為雷）
- 申月丙午日占何日雨，得離之旅。卯木父動逢空 → 出空之日卯日大雨；再占艮之謙合斷 → 寅日陰雲、卯日雨、辰日晴
- 辰月癸巳日占明日（甲午）雨。父動須看「應時日」 —— 申時雨（戌長生於申）、化退神則雨不久次日大晴

**野鶴方法論收束**（章末壓軸）：「教人以天時而驗人事，取效之速」 —— 即可由天時即時驗證的特性，反向理解占人事的五行機制。並借「官長有事命在旦夕」一例（子孫持世，再占多卦驗應於未時必雨、申時必晴），證**驗天時即可驗人事**。
