為貴人奔走效力財章第七十

最重財官兩旺﹐不宜世應皆空。世遇財應臨福﹐我益他損；應臨官世逢破﹐他益我損。兄爻持世﹐不如閒處安身；應克世爻﹐防慮他人反目。世遇財官臨應﹐無不遂也。世逢空破墓及動而變凶﹐皆非吉也。應克世兄克世﹐見災還淺。鬼克世世化鬼﹐得禍不輕。

開行開店及各色鋪面章第七十一

世為己﹐應為人﹐大宜相合。財為本﹐福為基﹐最喜同興。古法應為夥計﹐又為下顧之客。黃金策曰：開行定主有人投。覺子曰：須在來人之念；有心而問夥計﹐應為夥計﹐若得世應相生相合﹐彼此同心。應生世﹐他益于我。世生應﹐我益于他。相克相沖而情有變﹐應爻克世﹐我被他瞞；世克應爻﹐他從我願；應臨元武兄爻﹐暗中盜騙；應臨官鬼克世﹐明降災非；世財應兄﹐受他之累；世財應子﹐得他之力。世應空合﹐彼此有虛詐之心也。來人之念重在買賣者﹐須以應爻為下顧之客﹐應爻生合于世﹐生客來交成故舊；世應相沖相克﹐熟人終久變成仇。

鬼作災非須忌動﹐財為活計畏刑沖。鬼動招非克世最忌﹐古云：鬼現爻中見非﹐日有鬼搖克世災禍連系﹐須看鬼值何神克世﹐須于家室章內詳之。財為資本﹐忌衰墓空破及動而變凶﹐或日月刑沖克﹐更忌世與財空﹐俱開不成。

鬼兄發動有制何妨﹐隨墓助傷多凶少吉。鬼動招非﹐若得子孫制鬼﹐或日月沖克鬼爻﹐謂之有制。覺子曰：凡遇兄鬼同動者﹐不可曰鬼動招非﹐以鬼能制兄爻﹐不劫財也。兄弟又為陰隔破耗之小人﹐得此鬼動可以制服﹐獨不宜乃為助鬼以傷身也。

卦得反吟多反復。卦得反吟﹐欲行不行﹐行而又止。才開而又剪﹐剪而又開﹐倘店開久矣﹐遇此一定更遷。

沖中變合再重興。爻逢六合及六沖卦變六合或世財爻子孫共成三合﹐定是前後重興﹐惟六合變沖則不宜也﹐眼前熱鬧﹐終久蕭條。

合夥不嫌兄弟﹐蝕本內外無財。世應臨兄不吉﹐兄爻發動不吉﹐倘若世臨兄爻﹐日月作財星沖克世者﹐反為吉也。內外無財伏而又空﹐必然之蝕本﹐屢驗。

延醫章第一百零三

野鶴曰：有人而問予曰﹐問卜以求神延醫而服藥﹐有是理乎？予曰：若無此理﹐伏羲書卦﹐神農嘗藥﹐作何事耶？彼曰：然則又有不能救者﹐何也？予曰：有根者可以救之﹐無根者則不能也。人之星辰過宮或是交運脫運﹐謂之移花接木之年﹐移其有根之花木者﹐上則遮蓋﹐下宜澆漑則活矣﹐不澆不漑﹐見日則枯﹐服藥求神者卽此意也。若使無根之木無蒂之花﹐𨿽澆𨿽蓋﹐亦無用矣。

自占求醫﹐應爻為醫人﹐代六親而求醫者﹐亦以應為醫人。鬼作憂神休妄動﹐福為喜悅而生扶。自占代占﹐鬼𨿽憂疑之神也﹐若持世及動于卦中﹐得子孫動而制之﹐此醫可請﹐手到成功。若得子孫臨應爻者乃明醫也﹐其故何也？子孫乃制鬼之神﹐非眞正能制法魔之鬼也﹐乃先去憂神﹐我無優也。

祖塋舊家章第一百二十六

祖父之墓或葬多年或未久﹐如來卜者必有其因﹐須宜悉明﹐方可決斷。有因連年困苦疑﹐墳塋之不利。有因屢科不發﹐或有子入場﹐疑此風水可能發科否？為己者須宜自占﹐為子孫者命子孫來占﹐有因仕途蹭蹬連歲不升﹐疑其風水有礙。有因子孫不存﹐疑其風水相關。有因六親中父母兄弟妻兒或自身多生疾病﹐疑因風水所致。有因連年官非火盜。有因風水被傷﹐卜其何法而修補。有因閒問祖塋有地脈否。有問祖塋有傷損否。皆以父母為用神。卦遇六沖全無地脈﹐卦變化絕﹐勢若倒懸﹐爻遇伏吟﹐欲遷不遂﹐反吟卦現﹐不遷亦遷﹐卦變六沖龍已去﹐于孫有陵替之危。沖中變合運將來﹐後代有興隆之象。正卦六沖﹐變卦六沖﹐全無脈气﹐卦變化克﹐如倒懸之熱﹐危之急矣﹐凶災立至﹐伏吟卦地脈全無﹐𨿽有遷意而不能遷。卦得反吟﹐心不欲遷而終遷矣。卦變六沖有龍已去﹐子孫從此衰頹。六沖變合﹐先無地脈﹐目今地運將興﹐後代從此發矣。

父母旺相祖墳安然﹐墓絕休囚後裔零落。已葬之地﹐父母為用神﹐宜旺相﹐或日月動爻生扶﹐或動而化生日月化比助化進神乃吉地也﹐兒孫興發之象﹐亡者安而生者樂。旺而又遇帝旺長生者﹐來龍久遠﹐世代興隆﹐不宜墓絕空破﹐及動而破散化退神化絕墓化回頭克化旬空化破散﹐乃凶象也﹐亡者不安﹐生者寥落。

世爻變鬼﹐占者不祥﹐鬼化六親﹐各屬不吉。世爻變鬼及隨墓助傷﹐所占之人不吉﹐兄弟妻子動而化鬼、鬼化兄弟父母妻子者﹐查其所犯而斷也。又有子命化鬼﹐鼠命者殃。午命變鬼﹐馬命不利﹐餘仿此。

野鶴曰：古以父爻不宜旺相﹐父旺父動子孫不安﹐予以為謬也。占舊冢以父為用神﹐父母旺及動而化吉祖父安﹐然又扯到傷克子孫者是宜旺耶？不宜旺耶？惟子孫動而化父及父動化子孫﹐不利于子孫者是也。

因何事所傷章第一百二十七

修補祕法章第一百二十八

婚姻章第八十二（line 264-376）—— 占男女合婚（含胎孕子嗣 line 378-468）
==============================================================

**覺子開卷駁古**：
> 男家占女，不拘父母親朋而代占者，無不以財為用。女家占男，皆以官為用。用官者，官有生扶；用財者，財爻旺相，即許婚。
>
> 今見古今諸書文，**以財旺傷克父母爻**——不知宜財旺而為吉？宜財不旺而為吉？或曰：黃金策云「翁姑不睦，皆因妻位交重」；海底眼云「財動喪雙親」；易冒云「妻財動父母參商」——俱以財動傷克父母，**未說旺也**。予曰：動能傷父母，**旺則不能傷耶**？假令春天占卦，財爻屬木謂之女命旺而可取，但父母爻定然屬土——如此當權得令之財爻不克休囚之父母耶？何必曰動？殊不知者人言動而不言旺者，**正恐後人駁論**，所以不言其旺，不得已而言動也，豈知亦被看破。
>
> 據爾之論，何以為法？**予曰：分占之法也**——**先要財官旺相，既得吉卦，再占妨克父母**，是其法也。
>
> 黃金策專以應為妻位，易冒亦以應為妻，欲其旺相乃為賢良發福之女，又以應為女家父母，謂之應爻旺相必是富貴之家，若休囚必宅貧寒之宅。**執此論，則旺相賢良之女皆出於富貴之家？種種悖謬，予不知何以千百年來竟無一人道破！**

- 自占者：財爻應爻**皆宜生世持世合世，必忌破墓絕空**
- 財旺生身 + 應值破空 → 仍**可取**
- 應爻生世 + 財爻破絕 → **勿求**

**野鶴按**：財爻應爻同來生合世爻者更吉，**大抵以財爻為重**，他爻為附和耳。**黃金策曰「應為白髮之妻」，重應而又重財——謬也**。

**實占案例 1**（line 272）：子月癸酉日自占婚，得恆變鼎。
- 酉官持世，戌土財爻動而生世，世應相生
- 戌土雖值旬空，動不為空
- → 明日（出空之日）求之必允
- 結果：次日巳時允婚，夫妻白頭，兒女成行

**實占案例 2**（line 288）：寅月丙午日女家代占婚，得臨之既濟。
- 卯木旺官持世 → 古法看：婚必成
- 但應爻受克 → 古法看：男家不允
- **覺子按**：須要人活變——「**財官為重，世應為輕**」雖女家占男亦要財爻不致換陷
- 此卦男女二人財被回頭克 + 丑土克 → 不能生卯木官
- → 男女不能相合相生，婚雖成終有他變
- 結果：聘定四月，未及成婚被賊兵劫去（應巳月者，亥水逢月破）

------------------------------------------------------------
婚姻章後半（line 304-376）—— 進階組合判據
------------------------------------------------------------

**休囚破散的禁忌**：
- 財值休囚破散 → 終非舉案之姻
- 官逢衰墓絕空 → 難遂齊眉之願
- 男占忌財犯，女占忌官犯，自占世忌犯
- 不宜動化破化散化克化鬼 → 夭折貧寒之合

**世爻動靜**：
- 世空 + 動化退 → 終須失望
- 動而空 → 實空之月日必成
- 靜而空 → 終不成
- 世動化進神 → 事必成
- 化退神 → 終難就

**應爻組合**：
- 應雖空 + 財化進 → 夙有良緣
- 應空 → 他人不實
- 應動化退神 → 必有退悔之心
- 應雖空但財動化進神生世 → 雖見退悔，終是良緣
- 世應動而破者 → 應實破之時

**世應俱空 / 反吟**：
- 世應俱空 → 無准實
- 反吟卦變 → 反覆難成
- **野鶴按**：若得空而逢旺財鬼相生，先雖不允，後終還成

**男占禁忌**：
- 男占 + 兄動 → 卦中非配偶（兄為傷妻阻隔之神）
- **覺子按**：兄持世 + 財動相合 → 仍可成；兄化財 → 亦能成

**女占吉象**：
- 女占男 + 子孫持世發動 → 子孫為傷夫之神，**未成者不成，已成者傷夫再嫁**
- **官星持世合世生世旺相** → 不獨姻緣必就，可期舉案齊眉
- 卦中官鬼重疊 → 主再嫁

**夫妻和諧**：
- 財官世應刑沖 → 反目

**實占案例 3**（line 320）：巳月己亥日占夫妻不和，得需之大過。
- 子孫持世（女占夫 = 克夫之象）
- 寅木子水相生 → 不能傷夫
- 應爻為夫，逢子丑作合 → 夫已有外遇（水木相生 = 兩人情密）
- → **不是姻緣，雖不死別必生離**
- 結果：次年寅月休妻，六月再娶（應寅月者 = 夫臨寅木沖世；應六月者 = 未土沖開丑土，子水來生寅木）

**六合六沖**：
- 旺相 + 六合 → 彼此同心
- 男忌財逢破墓，女防官位克絕
- **六沖變六合者更吉**——眼下不允，日後必成；已成不睦者終究和美
- **六合變六沖**——雖許允後必更張，已成者終須折枕

**易冒占法（覺子按）**：
- 卦得**咸／臨／節／泰** + 無沖變 → 雖財官衰陷亦不為凶
- 卦得**睽／革／解／離** → 凶兆，雖用神全備而化合亦不吉

**實占案例 4**（line 340）：巳月丁卯日占婚於何時，得**泰卦**。
- 兄爻持世，亥財臨月破旬空；子水之財世克日刑
- 古以泰卦為吉，但細看財爻無生扶
- → **再占**得坤卦：世爻日破，亥水月破，卦遇六沖 → **大凶之兆**
- **結論**：「凡有所疑，勿執古法而斷，須以再占之卦而決之，後卦若吉即以吉斷，後卦凶者則以凶推，**神不欺人**」

**化爻組合**：
- 財化財 → 未必兩度婚姻（男占常為娶妾、賢妻美妾、雙娶、收妝奩甚厚）
- 鬼化鬼 → 難曰相守百年
- 兄臨玄武 → 防劫騙
- 鬼臨白虎 → 遇凶喪
- 應財世鬼 → 夫唱婦隨（陰陽得位）
- 應鬼世財 → 夫權妻奪（牡雞司晨）；惟贅婿得之反吉

------------------------------------------------------------
婚姻附章（line 378-468）—— 子嗣／父母／納寵／離婚／胎孕
------------------------------------------------------------

**此婚子嗣有無章第八十三**：
- 子孫旺相／動而化吉 → 主有子
- 子化進神／回頭生 → 有子必多
- 子孫墓絕／動而變鬼／鬼變子／父化子／子化父／父動克子 → 我命無兒（成婚亦無子）

**實占案例 5**（line 382）：申月丁丑日多占何命有子，得革之夬。
- 卯木子絕於申月 + 鬼變子又逢月破 → **公命無子**（與女命無關）
- 八字：甲寅甲戌甲子戊辰 → 純陽包陰，兵權顯赫；但並頭重疊克罡，亦主無子
- 解法：**多積陰德以培之**
- 結果：施濟貧寒、修橋砌路、施藥施棺 → 連生二子

**此婚有宜於父母否章第八十四**：
- 用神 = 父母
- 父或旺或衰皆不忌
- **所忌者財爻發動** → 此婚妨父母
- 財化父母、父母化墓絕空破回頭克、動而破散、日月克父母 → 桑榆暮景之秋

**納寵章第八十五**：
- 為子嗣：子孫為用神
- 占配僕：財爻為用，財持世／動生合世／化進化生化財 → 忠誠效力
- 散破空絕 → 懶惰或疾病
- 財克世沖世 → 負義忘恩
- 財化鬼化絕化墓化克或鬼化財 → 貧窮壽夭

**取離婦跳娼婦章第八十六**：
- 防患者用子孫
- 防攔阻者鬼動有禍
- 兄爻持世到底難成；財動相生不謀而就
- 六沖變沖 → 凶；三合六合 → 吉
- 六合變沖 → 成而不久；六沖變合 → 終必有成
- **強奪婚、遂私約 → 非祥非義，勿告於神**

**胎孕章第八十七**：
- 用神：子孫（親占代占皆同），惟**子占母孕用兄弟**
- 福神旺相遇生扶 → 麟趾兆瑞
- 子孫休囚逢破衰 → 泡孕空虛
- 子化子 → 雙生有準
- 兩動兩旺 → 雙胎
- 一陰一陽 → 一女一男

**男女判定**（覺子整理）：
1. 六爻靜 + 卦包陰（坎、大過、小過、咸、恆）→ 包陰生女
2. 六爻靜 + 卦包陽（離、中孚、頤、損）→ 包陽生男
3. 六爻靜 + 無包：子孫值陽為男，值陰為女
4. 有動爻 → **神兆機於動，先看動爻**：
   - 一爻動：陽動為女，陰動為男
   - 兩爻動：看上爻
   - 三爻動：看中爻
   - 多動：來心不誠，改日再占

**實占案例 6**（line 420）：辰月戊辰日占孕男女，得小過之豫。
- 兄爻雖動，不克變出之財爻 → 產婦平安
- 卦中陽動變陰 → 必主女
- 客曰：小過陰包陽，如何生女？
- 予曰：**神兆機於動，先重動爻**
- 結果：壬申日生女，產母平安

**問產婦安否章第八十八**：
- 用神 = 妻財（夫占代占皆同）
- 不宜鬼動；但若兄動克財則又宜鬼動制兄
- 不宜財變鬼、鬼變財、兄化財、財化兄、妻逢月破休囚破絕／化墓化絕／日月沖克 → 凶兆

**實占案例 7**（line 442）：子月乙亥日占妻孕平安，得豐之小過。
- 午火妻財臨月破 + 亥日克之 + 冬天之火囚被克 → **命難保**
- 卯木子孫動生火？**子動變鬼** → 神示「母子皆喪」（非生火之卯木）
- 子月亥日生卯？**變鬼之子，縱臨日月亦無益**

**結論**：六沖 + 子旺財強 → 終是喜；六合 + 財空子破 → 亦為憂

**產期章第八十九**：
- 產期遠則應月，近則應日
- 子孫動者 → 逢合逢值；靜者 → 逢值逢沖
- 空者 → 沖空實空之日；破者 → 實破逢合之期
- 白虎兄弟動 → 值日而生
- 子孫臨絕 → 待生旺之日
- 子遇長生胎養之日
- 伏藏者 → 出現之日而生

==============================================================
疾病章第九十九（line 638-698）—— 占病吉凶生死
==============================================================

**野鶴開卷駁古**：
> 人有七情，致生百病，急於問卜，以慰其心。**奈何卜筮諸書舛錯悖謬**！
>
> - 有取用卦身、世身、本命者
> - 有不看身命用神，**單看卦驗**——卦得「明夷、大過、觀、賁、蠱、夬、豐、同人、大畜、需」即斷必死
> - 又云「死氣喪門臨本命，早備衾棺」
> - 又云「蛇動主死，虎動主喪」——**以星煞六神而斷生死**
>
> **獨不思**：假使惡煞凶神臨身守舍，**若得用神兩身旺相**，斷之生耶？死耶？
>
> 予試多年——或生或死全憑用神，予皆不驗（指卦象論）。**人有不看用神而斷生死者，卦變為六沖是也**。

------------------------------------------------------------
野鶴疾病章核心方法論
------------------------------------------------------------

**六沖變沖的特殊地位**：
- 久病 + 卦逢六沖 / 卦變六沖 → **不論用神衰旺，乃不治之疾**
- 近病 + 六沖 → 不藥而愈
- 卦變克 → 新病亦危亡（如巽木變乾金、艮坤化震巽 = 回頭相克）
- 雖非墓絕，亦主危亡
- 化比和、化克去、回頭相生：**近病吉、久病凶**（六沖之故）

**六沖之外的判據**：必看用神。

**自占病用神**：
- 自占 → 世為用神
- 代占父母兄弟妻子 → 各按用神章取用神

------------------------------------------------------------
旬空月破的疾病判斷
------------------------------------------------------------

- **用遇旬空**：近病何須憂慮——沖空實空之日即愈
- **用逢月破**：久病難許安寧；旺則愈於實破之日及出月而愈，衰而受克者必危
- 久病 + 用神旬空月破 → **即使旺相，亦無法以治之**

**野鶴按**：海底眼、黃金策皆云「主空無救中殂」不分近病久病——**非經驗也**。

**覺子按**：近病值旬空 + 逢三合六合 → **必成久病而終**。

------------------------------------------------------------
化鬼／化忌的禁忌
------------------------------------------------------------

- 自占病 → 不宜世變鬼 / 化回頭克
- 占兄弟妻子 → 不宜鬼變弟兄妻子 / 弟兄妻子變鬼
- 不宜兄變財 / 財化兄 / 父化子 / 子化父

**野鶴按**：鬼為父母之元神——**父動化鬼乃為化生**：
- 父動化鬼 → 輕病即愈
- 久病 + 鬼化父 / 父化鬼 → 主危亡

**實占案例 1**（line 652）：申月庚寅日占子近病，得恆之解。
- 酉官持世，鬼變子孫 → 夭折之兆
- 幸子孫值旬空 → 近病即愈
- 恐其難過午年（出空之日而愈）
- 結果：辰年占卦至午年出花（痘疹）而死

------------------------------------------------------------
墓絕化墓絕的判斷
------------------------------------------------------------

- 用神逢墓絕 / 動化墓絕 → 全看衰旺
  - 旺者何須慮
  - 衰者實堪憂
- 日月動爻沖克者亦看用神旺衰
  - 旺相 → 沖去克神之日而愈
  - 休囚 → 生助克神之日而危

**散破無援脫氣**：
- 用神臨散遇破 → 全無生扶
- 脫氣休囚 → 無根蒂
- 忌神動克用神 + 元神發動 → 有救
- 仇神又動 → 凶危

------------------------------------------------------------
世持鬼／世持福的特殊判據
------------------------------------------------------------

**世持鬼**：
1. **自占**：官鬼持世 → 病難痊；雖子動克鬼，目下雖愈終不斷根；遲遲又生他病——「**病不離身，身不離藥**」。世臨官鬼 + 帝旺長生 → 久遠疾病
2. **代占六親**：官鬼持世 → 憂疑之狀；惟喜子孫發動克去憂疑——即使用神衰弱，管許平安

**實占案例 2**（line 674）：申月壬子日占子病，得遁卦。
- 官鬼持世 = 憂
- 子日沖去憂心 → **本日即愈**

**覺子重要按**：
> 占自身、兄弟、妻兒者——此法是也
> 占父母之災 → **官鬼為父母之元神**，豈宜子動而傷耶？
> 妻占夫 → 宜鬼為用神，又豈子動以傷官耶？
> 不惟不能解憂，**反添憂也**！

**身臨福德**：
- 自占 → 子孫持世，不藥而痊；臨應破者，沖空實破之日而愈
- 代占六親 → 不必看用神衰旺，即知其安。子孫為喜悅之神，旺相即吉

------------------------------------------------------------
鬼化長生／忌化進的進階判據
------------------------------------------------------------

- 鬼爻發動 → 病勢必重；若化長生 → 病勢必輕
- 克用神者為忌神 → 動化進神同重
- 忌神化退 → 疾病漸減
- **覺子按**：鬼為父母之元神（占父母即愈），亦為女人占夫之用神（占夫即愈）

------------------------------------------------------------
反吟與絕處逢生
------------------------------------------------------------

- 福搖化克卦反吟 → 病必反復
- 自占 / 占兄弟病 + 子孫動 → 制服官鬼即愈
- 子動化克 → 病愈後復發
- 卦反吟 → 同上

**絕處逢生 / 主衰得助**：
- 用爻受刑沖克害 + 日月動爻有一生扶 → **絕處逢生，臨危有救**
- 用神不宜太弱——但得生扶拱合，雖重病亦不至死
- **覺子按**：用神亦不宜太旺——易理貴乎中庸，**過猶不及**

痘疹章第一百（line 700-704）—— 占痘瘡

**平時占男女何時出花**（鬼爻為用）：
- 鬼靜 → 逢值逢沖
- 鬼動 → 逢值逢合
- 鬼衰 → 生旺之年
- 鬼旺鬼多 → 墓庫之歲
- 鬼空鬼破 → 填實之時

**出花後問吉凶**：

- 鬼動 + 日月生扶 → 花稠密
- 鬼空破休囚墓絕 → 痘來朗（疏）

------------------------------------------------------------
病源章第一百零一（line 706-712）—— 占病源
------------------------------------------------------------

**野鶴對黃金策病源篇的態度**：
> 此乃黃金策占疾病之首論也——予以為之**撞門槌**。對俗人而言不得不以此而斷之。
>
> 若知**分占之法**——另占一卦以定吉凶，庶有後驗。若即以此卦兼斷其生死者，**如果有靈，吾不信也**。
>
> 要知來人問病：**有何所犯？何以治愈？問吉凶如何？** 我且以發寒發熱而妄猜，病人不得趨避之旨，**問卦何益**？

**野鶴的疾病占斷流程**：
1. 先占吉凶卦
2. 吉者 → 調理即愈，不必服藥求神
3. 凶者 → 命之延醫
4. 卦得鬼動克用神者 → 再占（家宅 / 墳墓），務必求「何所犯、何法治」

**鬼神章第一百零二**（野鶴主流見解）：
- 鬼休囚空破墓絕 → **非鬼神之害**
- 占鬼神 → 隨爻五行 + 衰旺判神類
- **野鶴重要立場**：**祭祀無益於瘋癆氣蠱**——「節飽食、遠色欲、息氣養神，其病自減」。痛批清初貴州、四川、廣東地區「殺牲祭神救命」之風——「殺他命救命，其罪愈彰」。

**李我平按**：易冒「鬼臨金為關公」、「青龍神為寺亭候」——關公之前金鬼青龍是何神？**邪神也**！

------------------------------------------------------------
延醫章第一百零三（line 722-736）—— 占請醫服藥
------------------------------------------------------------

**野鶴哲學名段**：
> 問卜以求神延醫而服藥，有是理乎？
> 予曰：若無此理，伏羲畫卦、神農嘗藥，作何事耶？
> 彼曰：然則又有不能救者，何也？
> 予曰：**有根者可以救之，無根者則不能也**。人之星辰過宮或交運脫運，謂之「**移花接木**」之年——移其有根之花木者，上則遮蓋，下宜澆漑則活矣。不澆不蓋，見日則枯——服藥求神者即此意也。若使無根之木、無蒂之花，雖澆雖蓋亦無用矣。

**用神取法**：
- 自占求醫 → 應為醫人
- 代六親求醫 → 應為醫人
- 鬼作憂神 → 休妄動
- 福為喜悅 → 生扶

**判據**：
- 應爻空破墓絕／旺被日月動爻沖克／動化鬼化絕化回頭克 → 藥不見效
- 子孫制鬼 → 旺相生扶為喜
- 子動化鬼化克 → 藥不精
- 子動化空 → 實空之日有效
- 子動化生化旺化進神 → 藥更靈
- **古以子動化子孫其藥必雜——非也**（覺子按）：子動化子孫旺相者，另改藥品即比仙丹

**醫克用爻**：
- 自占病 + 應爻克世 → 制我之病
- 但久病及體弱之人 → 不宜醫克病身（身亦受傷）
- 應臨兄鬼克用神／世爻 → 必遭其殃

**世爻持何爻的判據**：
- 父持世 → 藥不見功，宜靜養遠色欲息氣惱
- 兄弟持世 + 子孫動 → 醫再延之；嫌鬼旺鬼興（誤服藥餌）
- 財持世 → 切莫誤食肥甘；旺相生扶 → 良醫；休囚沖克 → 無藥可醫

**「不代占」的原則**：
- 子不代父以占藥（父旺傷子，一爻不能兩用）
- 妻不代夫而卜醫（子旺又傷夫，不能代占醫）
