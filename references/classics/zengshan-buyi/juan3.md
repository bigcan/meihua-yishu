身命章第三十六

諸書占身命﹐謂妻財子祿一卦﹐能包壽夭窮通﹐六爻兼斷。殊不知父子財官兄弟﹐各有相忌相傷﹐若以一卦而兼斷者﹐卽如父母旺相﹐變應之微﹐又曰父旺傷子﹐豈世之有父母者﹐皆無子嗣之人也？又曰見兄則財莫能得﹐又為克妻之神﹐又曰兄弟爻與紫荊並茂﹐倘值旺兄持世﹐克妻耶？耗財耶？手足外傷耶？易林補遺有曰兄動妻亡財相﹐故執此論者﹐世之貧人寒士﹐皆失偶之人耶？至于財官子孫皆同此論﹐不暇細辧。必兼而斷者﹐卽先醫猶在﹐執此問之﹐知亦無從置喙。覺子曰予今得其法﹐乃分占之法也﹐占父母占兄弟﹐另占—卦終身財福何如？占終身功名有無？占終身夫妻偕老否？占終身子嗣及幫元俱宜分占。

終身財福章第三十七

福財旺相﹐鐘鳴鼎食之家。世爻旺相無傷﹐旣富且壽。覺子曰：占財宜于財旺﹐占壽獨重世爻﹐未聞財福不旺而世爻獨旺﹐卽許富與壽也。執此論者世之多壽者皆富貴之人也。凡占終身財福﹐世爻財爻子孫爻三者無一失陷﹐定是家殷戶足、福祿綿長。財世旺相而福神不旺者﹐乃先富後貧之人也。何也？子孫者﹐財之源也。水無源流﹐終須枯涸﹐謂之財旺福空榮華不久。世與福爻旺相而財爻不旺者﹐乃無財享福之人也。旣曰無財﹐焉能享福？此人心有現成之事業﹐只知坐享﹐不知生發；或是財托他人掌管﹐只知花費﹐不試艱難。財與福爻旺相而世爻不旺者﹐乃富屋之貧人也。覺子曰：以上之論﹐不旺所句者﹐無气失陷謂之不旺﹐得地有气亦謂之不旺﹐空破墓絕亦謂之不旺。財福旺而世爻不旺而有气者﹐從逢生之年復旺相。故謂之財福司權﹐榮華有日。財福兩旺﹐世爻無气者﹐不為吉也﹐𨿽旺財豐衣足食﹐不免癡瘂暗聾、或遭疾病官非、或系吝澀鄙俗。財福兩旺而世爻失陷者﹐更為凶也﹐𨿽有萬斛金銀﹐難買長生藥丹。前論財與福神﹐𨿽則不旺﹐亦宜有气﹐若失陷者﹐更為凶兆﹐財福失陷﹐立見傾家﹐惟動空及動而破者﹐不妨定﹝應實﹞破實空之年月也。

財世休囚﹐龜釜生塵之宅。世爻財爻福爻三者無气、或空破墓絕、或動而變凶﹐乃無衣乏食之人也。世爻得地、而財福失陷者﹐此人身彊力健、或有小謀小技﹐亦可支撐度日。世爻福爻有气、財爻無气者﹐𨿽則貧寒樂享淸福。世爻財爻有气、福爻而無气者﹐此人𨿽無積蓄﹐手頭常過金銀、或代人掌財﹐再得日月動爻生扶者﹐亦許小成。

求名章第四十八（line 1032-1034 + 童試章第四十九 line 1036+ + 鄉試章第五十五 line 1130+）
==============================================================

**野鶴核心立場**：
> 「凡占小考，當見占得吉卦，後竟道考無名，何也？**動之應近而不應遠**——已應府縣之有名也。**必須先占縣者，再占府考、又占道考**——但得吉卦者，許榜上有名。若有卦而不吉者，勿望道考而成名也。」

**核心方法論**：**分占法**。每一級考試（縣、府、道）需各占一卦。「動之應近而不應遠」——一卦的吉象通常只應於最近一場考試，不可一卦兼斷多級。

歲科考章第五十（line 1064-）—— 秀才歲考、科考
------------------------------------------------------------

**核心判據**：
- 父宜旺相，最忌休囚
- 官宜生身，不宜克世
- 父與世爻旺相 + 日月動爻生扶 + 化吉 → **超等**
- 父／世旺 + 逢生 → 上等
- 父／世旺 + 無生扶 + 無刑沖克破 → 次等
- 有一逢刑沖克者 → 又次等
- 世受克 → 教刑不免
- 世父同受克 → **劣等無疑**

**野鶴按**：判等須通變——看世父兩爻輕重，並注意六沖變沖、反吟、伏吟、動變克、化絕**皆是破綻**。

**覺子按**：以父爻旺衰列等第為主——進神者考上等，退神者考下等；若卦無進退之神，須另立活法。

**實占案例 1**（line 1072）：申月乙巳日占歲考，得大過之鼎。
- 亥水父爻持世，申月生父，父旺世旺；巳日沖之為暗動
- 未土財爻動但生酉金；酉官生世 → **接續相生**
- 結果：取超等補稟

**實占案例 2**（line 1094）：午月乙卯日占歲考，得地天泰。
- 午火月建為父母 + 卦中巳火伏於寅木下 → 旺而又遇飛神之生 + 爻逢六合
- 不宜日辰克世 → 發落之日宗師面叱「字法古筆」，姑取三等
- 後悔之。**說明**：父爻旺但世爻受沖克 → 不能保上等。

**結論**：
> 父與世爻旺相遇生扶為上。略有破綻者次之。刑傷有救者又次之。刑傷無救而輕者再次之。世父被刑沖克破而重者，**劣等必矣**。

增稟章第五十一（line 1112-）—— 增加廩生待遇
------------------------------------------------------------

**覺子核心方法論**：
> 占稟若不得其秘，望靈者少也。**占稟用財**——財乃文章之忌神。若兼問歲考與稟，財爻若旺則父母受傷，**休囚如何得稟**？

**正確方法**：**分占法** —— 先占歲科吉凶（父用神），再占一卦問稟（財用神）。

**稟以財爻為用神**：
- 財持世 / 動而生世 = 吉
- 兄持世 / 兄爻發動 / 財逢空破墓絕 / 動而變凶 → 失稟
- 官鬼兄爻克世 → 輕則停薪、重則道罰
- 卦中三合兄局 → 亦主停薪

鄉試章第五十五（line 1130-）—— 舉人鄉試
------------------------------------------------------------

**核心判據**：
| 條件 | 結果 |
|---|---|
| 父旺官興 | 姓標虎榜 |
| 父官持世旺相 / 臨日月 / 日月動爻生扶 / 動變生 | 名登龍虎 |
| 子搖財動 | 名落孫山 |
| 兄弟 | 奪標之惡客 |

**兄爻的辯證**（覺子重要按語）：
> 俗傳兄爻興難上榜。**殊不知有喜有忌**：
> - 卦中財爻同興 → 喜兄動制財
> - 卦中父官兩動 → 又喜兄持世（父動生世）
> 兄持世 + 臨日月旺相 + 官父兩旺 / 官動生父 / 父動生世 → **必添科名**
> 兄不能掄元而奪魁——故謂「兄弟乃奪標之客」（非奪魁，是奪標而已）

**特殊規則**：
- 日月為制煞之將軍：官父世爻被動爻克／回頭克時，日月沖制克神 = 得意
- 世旺 + 官父伏藏 → 日月沖飛神提伏神 = 高發
- 二合無沖 → 登甲第（官父世合成父局／官局）
- 合成兄局 + 官旺 → 亦可
- 合成子孫局 → 不為吉
- 合成財局 → 雖不為美，亦得驗於吉者

求財章第六十八（line 1618-1704）—— 求財／經商／借貸／賭博通章
==============================================================

**野鶴開卷立場**：
> 諸書之論求財，**弗如黃金策裏真論**。予屢試驗，惜乎未經分別。今**以前段為總論**——以後賢易曉。內有屢試而不驗者刪之，予另得驗者增之。

**野鶴本章採用「**黃金策直解的求財段**為主軸 + 自己補注」的結構**——對應 [juan2.md](../buzheng-zhengzong/juan2.md) v3.2-k 第十八類「求財」歌訣。

**核心用神**：
- **妻財** = 財（生世合世克世持世皆吉）
- **子孫** = 元神（生財）
- **兄弟** = 劫財忌神
- **官鬼** = 制兄之神，但若兄不動則鬼動洩財

**逐條野鶴／覺子注疏**：

**1. 財旺福興，公私稱意；財空福絕，上下連心**：
- 公私占卜皆以財為用神
- 子孫為生財元神
- 兩者旺而化吉、生合世爻 → 皆稱意
- 衰墓絕空 / 刑沖克害 / 變凶 / 日月沖克破 → 失敗

**2. 有福無財，兄弟交重偏有望**：
- 兄爻 = 劫財之神
- 財兄同動 → 阻隔破耗
- **若得子孫與兄同動 → 為可喜**（兄動生子）
- 財／兄／子同動於卦中 → **更喜**（兄生子、子生財，源流長之利益）

**3. 兄與財同動 + 官爻發動，亦堪求**：官克兄之故。

**4. 財福俱無，莫若守株待兔**：
- 但**覺子按**：若日月為財或伏而旺，亦可求。

**5. 父兄爻動，無殊緣木求魚**：父動克福、兄動克財，圖必失。

**實占案例 1**（line 1632）：酉月戊午日占求財，得革卦。
- 財爻不現，亥水兄爻持世
- 父臨月建（酉月）生助兄爻
- → **如緣木求魚**

**6. 多財反覆，必須墓庫以收藏**：
- 卦中財臨日月 = 重疊，得意之時須待**財爻入墓之日**
- 例：水作財神 → 辰日得（水墓於辰）

**7. 無鬼分爭，又怕友重而阻滯**：
- 兄爻為爭奪／阻隔／耗財之神，宜官鬼動制之
- **覺子按**：兄發動 → 喜鬼動制之；倘卦中兄安靜 → 又不宜鬼動（反洩財氣）

**8. 兄如太過，反不克財**：
- 舊注：兄爻多動者反不劫財
- **覺子按**：非也！**兄爻多者，待兄爻入墓之日及克損爻之日，必劫其財** = 「太旺者損之斯成」

**9. 世遇兄臨，必難求望**：
- 古以卦身臨兄難求望
- 覺子：卦身不驗，以**兄爻持世**為「難遂圖謀」
- **野鶴按**：全在通變——兄持世不必為忌。

**實占案例 2**（line 1658）：未月丁卯日借貸，得晉卦。
- 兄爻持世無財，但卯日即財星
- 古：財爻沖世者必得
- 應爻未土旺生世 → **明日辰日必獲**
- 應辰日者 = **世動逢合之日**

**實占案例 3**（line 1674）：巳月丁巳日占求財，得既濟變渙卦。
- 兄持世臨日破月破 → 不克變出之財
- 日月俱作財來沖世
- 但應爻逢空 → 明日沖實必送財
- 果於次日（戊午日）送來

**10. 財來就我終須易，我去尋財必是難**：
- 舊注：財生世合世克世持世皆「就我」
- **野鶴按**：兄持世時亦能得財（見上 2 案）——**斷卦靈機應變，不可拘執**

**11. 福變財生，利源滾滾**：子動生財、子化財、財化子皆同。

**12. 兄傷鬼克，口舌紛紛**：
- 財持世 + 兄動 → 不獨劫財，還防傷己
- 鬼動克世 → **更凶**
- **覺子按**：兄動變鬼 = 兄化回頭克 = 自顧不暇，**不能再沖克世爻**

**13. 財局合福德，萬倍利源**：世與財爻／子孫爻三合成局 → 諸占追財豐。
- **野鶴按**：須**在局中**或**合成子孫財局生世**，方為全美
  - 合成父局 → 勞碌辛苦
  - 合成兄局 → 破耗多端
  - 合成鬼局 → 口舌災非（克世更凶）

**14. 歲君逢劫煞，一年生意無聊**：劫煞 = 兄弟爻；兄臨太歲 → 一年破耗。

求財章總方法論（野鶴／覺子精華）
------------------------------------------------------------

1. **分占法**：求財 + 求名 + 婚姻 + 疾病不可同占——每事一卦
2. **兄爻辯證**：不可一律以兄為忌——遇財／父／子組合須通變

終身財福章第三十七（line 22-296）— 五行合三神（世/財/福）
------------------------------------------------------------

**用神**：
- **世爻** = 自己（壽與成）
- **妻財** = 財帛
- **子孫**（福爻）= 財之源、福德

**核心判據（三神組合表）**：
| 世 / 財 / 福 | 結果 |
|---|---|
| 三者皆旺 + 無傷 | 家殷戶足、福祿綿長 |
| 財旺世旺 + 福不旺 | **先富後貧**（無源之水終枯涸）|
| 世旺福旺 + 財不旺 | 無財享福（坐享 / 代人掌財）|
| 財旺福旺 + 世不旺 | **富屋之貧人**（衣食豐而身癡瘂暗聾 / 疾病官非）|
| 財福兩旺 + 世失陷 | **更凶**（萬斛金銀難買長生）|
| 三者皆無氣 / 空破墓絕 / 動而變凶 | 無衣乏食 |
| 世得地 + 財福失陷 | 身強力健、小謀小技、支撐度日 |

**覺子按—「不旺」三層**：
- 無氣失陷 = 不旺
- 得地有氣 = 不旺（一級）
- 空破墓絕 = 不旺（最重）

**世爻特殊象**：
| 世象 | 古法 | 野鶴／覺子修正 |
|---|---|---|
| 世旺而空 | 終身無成 | **帶旺非空**；沖空之年勃然發福 |
| 世動而空 | 終身無成 | 動不為空；沖實之年發 |
| 世入墓 | 到老求謀多淚 | **墓中人逢沖之年而發** |
| 世自旺（無生扶亦不衰）| 獨力撐持 | 必須財持世，白手成家；兄父鬼持世則僅可撐持 |
| 世衰逢生扶 | 因人創立 | 必遇好人提拔；亦要財持世或日月作財生世 |
| 兄持世 | 克妻耗財 | 看旺衰：**旺則貧而好義；衰則疾病官非**。日月生扶 → 貧而樂；日月作財合世 → 富而驕 |
| 官衰持世 | 疾病纏綿 | **公門異行資生**（衙門書吏、術業 ）|
| 衰財持世 + 日月動爻生扶合助 | — | **暴發興家**（守農豐稔／工藝精巧）|
| 青龍子孫持世 | — | 立志高遠不慕富貴；無氣亦絕俗超群 |
| 白虎財旺持世 | — | 鄙俗偏為富客（不知禮義但家道殷實）|
| 父母持世 | 辛勤勞祿 | **官旺則名盛**（古法以父持世即勞碌一生為謬）|
| 鬼克世（旺）| — | 五行分判：火炎／木窖／水溺／土厄／金刀 + 六神：玄武盜賊／螣蛇官非／勾陳田土牢獄 |

**野鶴駁古法 4 條**：
1. 「世位休囚非貧即夭」→ 非貧者觀財福，夭者單用世爻，**「非夭即貧」不是卜易之理**
2. 「六沖卦作事有始無終」→ 須兼世與財福皆無氣，方論六沖之凶
3. 「父母持世一生勞碌」→ 須見官父兩旺方論；不可一律
4. 「動而逢沖謂之散」→ 不沖世爻則不散

**實占案例**（卯月申日得復之頤）：五爻亥水財、世子水財皆長生於申日；酉金福沖破化絕 → 衣祿不少但難成家。果三十七歲入典鋪掌櫃，三易財東依然在櫃，活七十一，工食養家，生四子二女。
> **古法看子孫破化絕應斷無子，野鶴反證有子四人**——分占法之力。

壽元章第三十九（line 376-446）— 單用世爻
------------------------------------------------------------

**用神**：**世爻**（獨用，不兼財福）

**核心判據**：
| 世爻 | 壽元 |
|---|---|
| 旺相 / 臨日月 / 日月動爻生扶 / 動而化生化旺化回頭生 | **大壽** |
| 休囚 + 沖克之年 | 防夭折（應期章內斷）|
| 休囚 + 隨鬼入墓 | 凶 |
| 休囚 + 助鬼之傷 | 凶 |
| 動化退 / 化鬼 / 化回頭克 / 化絕 / 化墓 / 化破空 | 「夕陽無限好，只恐不多時」|

**元神（生世之爻）規律**：
- 占壽**宜旺而靜**，不宜動搖
- 動則「已有限期」——非元神逢絕墓之年，即在元神被沖克之歲
- 與占他事規律相反（占他事元神動則有力）

**忌神規律**：
- **忌神不動自是平安**
- 動則有限期——應逢合之年 / 逢值之歲

**野鶴破古法「初爻五年法」**：
> 「占以初爻管五年，二爻管五年......予試四十餘載，並無應驗，不以為法。**止以世爻衰旺而斷知長短，欺人之法予不為之**。」

**實占案例**（辰月乙巳日得中孚）：世臨未土，巳日生、月扶 → 必享長年；占時 53 歲，許 20 年後再占。果至 71 歲再卜，又延數年。

**對照案例**（占父壽，亥水父臨月破又被未土相傷）：旺木衰土相剋之局 → 不獨不能長年，且無長月。果終於本年冬至日。

趨避章第四十（line 448-670）— 占卜倫理核心
------------------------------------------------------------

**野鶴哲學名段**（占卜的存在論基礎）：
> 「聖人作易，原令人趨吉避凶。若使吉不可趨，凶不可避，聖人作之何益？世人卜之何用？或曰：『年炎月晦，可以避之；死生如何能避？』予曰：**安於正寢者，雖有可避之方，亦不須避**。康節先生臨終呼弟子沐浴更衣......『要作神仙不甚難。』」

**避法核心三原則**：
1. **占得死於水者，莫近河旁**
2. **占得死於刑者，不可違條犯法**
3. **未有不化凶而為吉**

**忌神位置 → 避法**：
| 克神 / 世位 | 避法 |
|---|---|
| 克神在內卦、世在外卦 | **宜出外避** |
| 克神在外卦、世在內卦 | **宜在家避** |
| 世與忌神皆在外卦 | 不可出行，家居可避 |
| 世與忌神皆在內卦 | 不可家居，宜出外避 |

> 對照出行章：「路上有官休出外，宅中有鬼莫居家。」

**鬼克世的五行避法**：
- **火鬼** → 防火炎
- **木鬼** → 防木壓木器之害
- **水鬼** → 沈溺之憂
- **土鬼** → 巖牆之厄
- **金鬼** → 劍刀斧鍬
- **玄武鬼** → 盜賊兵戈
- **螣蛇雀鬼** → 官非兼防火
- **勾陳鬼** → 田土牢獄
- **青龍鬼** → 因酒色危亡（吉神克世亦凶）

**八卦輔助**：乾兌金形（寺）／坎兌水災／離火／震舟車／巽婦女姦／坤艮郊野山林老婦妖童。

**避難方位**：
- **生世之方** = 吉避所
- **子孫之方** = 福神之地

> 「子孫若發動於卦中，**不拘克世持世，俱以為吉**。」

**避於子孫局的反證案例**（巳月戊辰日占防流兵得臨之睽）：
- 巳酉丑合金局克木鬼世，眾以為「萬莫能逃」
- 野鶴：**子孫合成金局，克去身邊之鬼，何憂之有？**
- 「世人看書不到、論理不徹、以吉為凶者，不獨我一人」

**避於生方而非子孫之方的案例**（巳月丙戌日占避亂得乾之大有）：
- 眾以子孫屬水、金動生之，以為北方吉
- 野鶴：**午火（鬼方）得令而生世，往南避之為吉**。子孫屬水但生方更要——「火乃鬼方，生我何礙」
- 信予言者皆從南避，後北軍燒村而去
> **要害**：子孫之方亦吉，但須與旺衰參看；「生方」優先於「子孫方」。

**「漢口卯月丙寅日索債卦」自傳**：
- 占得益之中孚，世辰土落空，辰時出空被克 → 苦留客人不渡江
- 早飯後行至江邊，四隻大船覆沒於暴風 → 拜謝活命
- 野鶴：**非謂之避之則吉也？**

**世臨子孫遇旬空**：事不結，終日憂煎；待沖空 / 出旬之日結。**未至出旬，盡屬虛疑空憂**。

**終身結果（橫亡 vs 善終）判讀**：

|---|---|
| 旺相 / 日月生扶 / 動化吉 | **無疾而終** |

父母壽元章第四十一（line 672-756）— 分占父母
------------------------------------------------------------

**覺子核心立場**：
> 「凡占祖父祖母，占父母**須宜分占**。」

**用神**：父母爻（不分陰陽）

**野鶴破古法**：
> 「**古以陽爻為父，陰爻為母，此論非也**。」——一卦中父母爻或陽或陰，皆作父或皆作母用，全憑來人占問所指。

**核心判據**：
| 父母爻 | 結論 |
|---|---|
| 旺相 / 臨日月 / 日月動爻生扶 / 動化吉 | **長享遐齡** |
| 休囚 + 無刑沖克害 | 晚年多病但無傷 |
| 休囚 + 刑沖克 / 變凶 | **承歡宜早**（及時行孝）|

**實占案例 1**（先占父壽得姤、再占母壽得大壯之升）：
- 父占（姤）：丑土父母臨月建旺於子日 → 大椿永茂
- 母占（大壯之升）：午火父動被日沖、子水又動克之 → 防丙子年（太歲沖之）
- 卅餘年後相遇問之，父健、母終於子年——**分占父母各靈驗**

**實占案例 2**（祖母壽 + 急救）：丑月庚子日得歸妹變復，丑土父臨月建、午火暗動相生 → 雖木動克土化退神絕於申，**仍有救耳**。野鶴急赴其家，臨終灌白生（白朮）三錢，移床燒坑，又活十三載——**「救其於將死」典範**。

3. 從無兄弟 → 將來有否
4. 兄弟壽元

夫婦章第四十三（line 798-848）— 妻 vs 應的辨正
------------------------------------------------------------

**問法（分占）**：
1. 妻壽
2. 夫妻偕老否
3. 夫妻和睦否
4. 妻有刑傷破敗否
5. 占妾／婢（一卦只占一人）

**野鶴破《黃金策》「應爻為妻」**：
> 「黃金策專以應爻為妻，其故何也？因身命章中一卦兼斷，**設以財為妻者，又以何爻為財帛？不得不以應爻為妻位**。」——分占之後，**妻財爻為重，應爻次之**。

**判據優先級**：
| 財 | 應 | 結論 |
|---|---|---|
| 旺 | 空破 | **吉**（財為主）|
| 空破 | 旺 | **凶**（財為主）|
| 旺 | 旺 | 吉而又吉 |
| 破 | 破 | 凶而更凶 |

**核心判據**：
| 卦象 | 結論 |
|---|---|
| 財旺相 / 臨日月 / 日月生扶 / 動化吉 + 生合世爻 | 舉案齊眉、和睦不反目 |
| 財動化兄 / 化鬼 / 化回頭克 / 化退神 / 化絕墓破空 | 玉碎珠沈（妻亡）|
| 兄持世 / 兄動 / 日月臨兄傷財 / 兄化財 | 不得長年 |
| 兄持世 + 財旺（兄不能克）| **生離**（雖不亡，相見如仇）|
| 兄持世 + 財化退神 | 生離 |
| 財多現（分正庶）| 應爻臨財者為正妻 |
| 財爻克世 + 世休囚空破墓絕 | **天命夭亡** |
| 旺財克世 + 世得地 | 雖無刑傷必遭妒悍 |
| 財生合世 | 非和美必獲妻家之財（內助善持家）|

**駁古「世應相沖即斷生離」**：
> 「滿盤俱是財爻，世爻變出之未土與世相合......此卦若以世應相沖相克而斷之，**遠如天壤**。」
- 案例（巳月丁未日得無妄變觀）：夫八十二、妻八十一，同庚偕老，正妻賢而無妒、美妾十餘。

**重要案例**（酉月辛巳日得泰卦占夫婦和好否）：
- 兄爻持世，財爻亥水酉金生之 → 財旺難克
- 但巳日沖動亥水又臨驛馬 → **妻財臨馬而暗動，心去難留**
- 「生離之象」——後果生離

子嗣章第四十四（line 850-932）— 問明方能斷
------------------------------------------------------------

**問法（必須指明）**：
1. 已有子，後還生否
2. 現有之子，將來可保長年否
3. 從未生育，終有子否

**野鶴重要哲學**：
> 「**易告未來，不告已往**。」——含糊問則神報將來之無，不報以前之有。斷卦人依卦而斷，問者反不能服。

**核心判據**：
| 卦象 | 結論 |
|---|---|
| 子孫旺 + 生扶 / 臨日月 / 帝旺長生 / 動化吉 | **必產賢兒** |
| 子孫衰弱 | 生子愚癡 |
| 子孫衰 + 受克 / 休囚無氣 / 落絕空破 / 變凶 / 日月沖克 | 生而不有 |
| 父化子、子化父 | 不育 |
| 鬼變子孫、子孫變鬼 | **百無一活**（少年無嗣，老年立侄）|
| 父空持世 | 不育 |
| 子孫逢空（無沖克）| 不礙，沖空實空之年得子 |
| 卦有財無子（變爻為子）| 庶出 |
| 卦有子又有子變爻 | 子化子 = 晚子 |
| 子孫化子孫 | 少年無子者多生子；現有子者多子多孫；**老年無子者扶他人之子**（化本宮→族中子，化他宮→異姓子）|

**野鶴重大破古論**：
- 古：「子化兄，生兒不肖」→ **化兄及回頭生反吉，何不肖？**
- 古：「子化鬼亦不肖」→ **化鬼者百無一活，豈不肖而已耶？**
- 古：「子帶貴人，童年拜相」→ 野鶴查天乙貴人臨爻並無祿到，**信筆妄談**

**重大反證案例**（亥月丙辰日占子易養否得姤之旅）：
- 亥水子孫臨月建五爻、兄動相生——**旺莫旺於此也**
- 若論吉神：丙日以亥為貴人——**貴莫貴於此也**
- **但**：子爻變鬼 → 死於次年五月
> **教訓**：旺與貴皆不敵「化鬼」之凶。

**代占六親通則**：
> 「先看世爻。雖占六親，常有帶出自身之凶者——**世爻化鬼、化回頭克、休囚化空破墓絕、隨鬼入墓、日月同克：休問他人，且防自己**。」

**「占此應彼」現象**（覺子）：
> 「常見占父母之卦爻中帶出刑傷兄弟兒孫妻妾者，有占兒孫亦兼應父母兄弟妻妾者......此謂占此而應彼也，**予占而屢見**。」

仕宦類 11 章（line 1240-1618）— 關鍵歌訣與野鶴反駁
------------------------------------------------------------

#### 升選候補章第五十五

**核心判據**：
| 卦象 | 結論 |
|---|---|
| 旺官持世 / 日月作官生合世 / 世動化官生世 | 升遷 |
| 子孫持世 / 子孫動 | 無望 |
| 官空破墓絕 / 動變凶 | 補升無望 |
| 世官三合官局／財局生世 / 卦六合 / 動變六合 | **天賜之恩** |
| 隨官入墓（世旺）| **官升** |
| 助鬼傷身（元神同動）| 必升 |
| 助鬼傷身（世休囚）| **禍至** |
| 子孫持世值旬空（兩三日內占）| 必得（未出空不克官）|

**野鶴重大警告**：
> 「子孫值旬空兩三日內占者必得官——**但占過三個俱皆得官、得官之後隨即死亡**。」

**反證案例**（申月戊寅日得夬卦占得差否）：
- 子孫持世空 → 三日內未出空可得差
- 果不出三日得差 → **行至中途而死**（世空之故）

#### 升遷何方章第五十六

- 古法：官金應西、木官應東、水北、火南、土中。
- **野鶴修正**：木官**可應山東、廣東**；金官**可應江西、廣西**——以**住處 + 卦中爻位**綜合判斷。
- **更高方法**：不問「何方」直問「能得此缺否？」官星持世必得、三合官局生世必得、世空官空不得、子孫持世不得、三合局生應爻不得。

#### 在任吉凶章第五十七

- 官旺財興 → 顯赫；子搖兄動 → 減俸休官。
- 兄動 + 旺官 = 為官清正非義不取（非主減俸）。
- 兄鬼亂動 → 旱澇兵戈、地方不寧。
- 朱雀螣蛇克世 → 誹謗多招；世動化鬼化回頭克 → 命盡當危。
- 父母空破 + 財爻持世動 → **防孝服**（不可以財論吉）。

#### 援例章第五十八

- 財官世三者，**任一陷則終敗**。
- 反證案例（丑月乙卯日得豫之否）：財動生官，但世臨月破被克 → **後竟納之，名敗身喪**。
- 兄持世／兄發動 + 無官同動 → 白費其財終不成名。

#### 武試章第五十九（兼文試）

- 文試重官父；武試**首重官爻**。
- 子孫持世 / 子動 → 「力能扛鼎、百步穿楊皆無益」。
- 日月動爻克世 / 世化鬼克 → 試期病阻或災禍臨身。

#### 投麾效用章第六十

- 官星持世財動相生 → 封侯拜將可期。
- **覺子辨**：問名利看官鬼為功名；問能否收用看官鬼為**收用之官**（生世必收）—**「不知其此竅，竟以此官星斷自己之功名，如天遠也」**。

#### 署印謀差章第六十一

- 財官皆旺、子兄不動 → 可成。
- 衰世隨鬼入墓 / 世動化克化鬼 / 子兄持世 → **自投羅網**。

#### 面聖上書章第六十二（含叩閽、獻策、條陳、劾奏）

- 旺父持世（文書）+ 歲五日月生 → 奏必見準。
- 兄持世 + 父動化吉 → 喜動龍顏因文見用。
- 財持世 / 財動 → 克破文書（無益之奏）；但若財官同動 + 父持世 → 反喜（財生官生世）。
- **野鶴重大辨證**（子動之兩面）：
  - **有職之官**章奏：子動克官 → 不宜（輕降罰、重休官）
  - **無職之人**章奏：分問——為求名則子動克官（不成名）、為避禍則子動為福神（**反保無禍**）
  - **將相諫君**（殿折廷諍）：**反喜子孫持世**——縱削職可保身，名垂竹帛
- **野鶴占卜倫理**：「凡上書劾奏，**果有興利除害益於國者，舍身為國雖蹈湯赴火不辭，何用占卜？**」——占卜限於兩可之事。

#### 防劾／大計／已參案件

- 旺官持世（自身）→ 喜照雙眉，**已參未結案無憂，定蒙卓異**。
- 子動財空 → 減祿罰俸；身衰鬼克 → 貶責凌辱（輕者世有救、重者世休囚被克）。
- **野鶴破古「身空殺動避禍」**：世動空者出空之月日必遭其害，**不能避**。唯世空逢六神生則可。
- **野鶴新解「身辱官存」**（世遇五爻之克）：若旺官持世，雖受辱仍可為官。

#### 養親告病辭官章第六十三

- 子持世 / 子動 → 遂願偷閒。
- 官動克世合世 → 難脫利名韁鎖。
- **告病／養親**得反吟卦不忌（將來再任之兆）；**辭官**得反吟反不宜（將來遠入留任）。
- 世官化退神 → 如願；化進神 → 五事羈身難歇手。

#### 修陵修河公務章第六十四

- 初入工程而防患：子動無尤；兄動防賠財。
- **已見疏憂而防患**：子動傷官 → 反有功名之慮；兄動 → 防罰俸。
- 工程堅久：父爻為用神（六沖不久；六合堅牢）。

#### 僧官道紀醫官雜職章第六十五

- **野鶴重大辨**：
  - 他占僧道醫人 → 以子孫旺相 = 道業高強
  - 僧道自占道藝 → 以子孫為用神 = 鬼服龍降
  - 僧道**自占功名** → 仍以官爻為用神（不可因「子動制鬼」而誤判）
- 陰陽雜職 → 子動子持世皆不成名（同俗家求名）。

#### 功名到何品級章第六十六

- 古法：水一火二木三金四土五。
- **野鶴破之**：「**水至二數位至公侯者，何以定之？**官設九品而數止五六七品者，何以定？」
- **正確方法**：**指問**——「我能公侯否？將相否？一品否？八座否？」旺官臨日月指日可期；官破刑沖終生失望。
- 反證案例（巳月乙卯日小過卦）：午火官星持世 → 官至命事；後再占得節之蹇 → 世臨巳火青龍持世 + 變辰土官 → **中鼎甲而應辰年**（甲辰科殿試果得首唱傳臚）。

#### 子占父功名章第六十七

- 父動官動相生 / 日月作官生父 / 父旺化官 → 成名。
- 反證案例（卯月壬辰日渙之夬占父何時起用）：寅木父化子水官 → 寅年起用；**上爻卯木父動化未土，占時未墓爻空，謂之「空墓以待死」** → 終於實墓未年。
