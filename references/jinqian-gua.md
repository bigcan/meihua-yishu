金錢卦（六爻起卦法）

定位
金錢卦又稱「火珠林法」、「六爻納甲法」，相傳出自漢代京房，宋代《火珠林》流行後成為民間最廣的占卜法。
與梅花易數的關係：
- 梅花易數：以數理起卦，單動爻，重「象+理」推演
- 金錢卦：以擲幣起卦，可多動爻（0~6），重「納甲六親+用神」
- 兩者可互補使用。問抽象趨勢用梅花，問具體人事物用六爻。

操作方法
器具
- 銅錢 3 枚（古錢最佳，現代硬幣亦可）
- 雙手合攏，誠心默念所問之事

擲卦規則（自下而上，共 6 次）
1. 第 1 次擲 → 初爻（最下）
2. 第 2 次擲 → 二爻
3. 第 3 次擲 → 三爻
4. 第 4 次擲 → 四爻
5. 第 5 次擲 → 五爻
6. 第 6 次擲 → 上爻（最上）

每次擲 3 枚銅錢，看正背面組合：
銅錢面|數值|說明
背（無字面）|3（陽）|乾道
字（有字面）|2（陰）|坤道
現代硬幣可約定：頭像面=背=陽=3，數字面=字=陰=2

爻象判定
3 枚銅錢之和對應 4 種爻象：
和|陰陽|名稱|符號|動否|變化
6（三字）|陰|老陰|X|動|變陽
7（一背二字）|陽|少陽|—|不動|—
8（二背一字）|陰|少陰|- -|不動|—
9（三背）|陽|老陽|O|動|變陰
口訣：少不變，老則變；陽變陰，陰變陽。

成卦範例
範例 1：擲出 字字背 / 背背字 / 字背字 / 字背背 / 背背背 / 字背字
位置|爻|擲出|和|爻象|陰陽|動否
上爻|6|字背字|7|少陽|陽|靜
五爻|5|背背背|9|老陽|陽|動 ←
四爻|4|字背背|8|少陰|陰|靜
三爻|3|字背字|7|少陽|陽|靜
二爻|2|背背字|8|少陰|陰|靜
初爻|1|字字背|8|少陰|陰|靜
本卦：自下而上 8-8-7-8-9-7 → 陰陰陽陰陽陽 → 下卦坎(010)、上卦兌(011) → 澤水困
變卦：五爻動，陽變陰 → 上卦變為 001(震) → 雷水解
動爻：第 5 爻

解卦法（朱熹《易學啟蒙》）
金錢卦可有 0~6 個動爻，依朱熹法判斷主爻與主辭：
動爻數|主要參考|主爻判定
0|本卦卦辭|無主爻
1|本卦該動爻爻辭|該動爻
2|本卦兩動爻爻辭|以「上爻」為主，下爻為輔
3|本卦卦辭 + 變卦卦辭|本卦為「貞」（事之初），變卦為「悔」（事之終）
4|變卦兩不動爻爻辭|以「下爻」為主，上爻為輔
5|變卦唯一不動爻爻辭|該不動爻
6|變卦卦辭|乾卦例外看「用九：見群龍無首吉」；坤卦例外看「用六：利永貞」
理解口訣：
- 動爻少：看本卦動爻
- 動爻多（≥4）：事情已大幅轉變，改看變卦不動爻
- 動爻 3：本卦變卦各半，貞悔並參
- 動爻 6：盡變，全看變卦

互卦處理（金錢卦版本）
與梅花易數同：取本卦 2-3-4 爻為下互、3-4-5 爻為上互。
特殊情況：本卦為純陽（乾）或純陰（坤）時，若有動爻，互卦從變卦取；若無動爻，互卦同本卦。

scripts/jinqian_gua.py 使用方式
模式 1：隨機模擬
python jinqian_gua.py random           # 完全隨機
python jinqian_gua.py random 42        # 指定隨機種子（可重現）
模式 2：一次性手動輸入
python jinqian_gua.py manual 字背字 字字字 背背字 字背背 背背背 字字背
順序：自下而上（初爻 → 上爻）
符號彈性：
- 中文：背、字、陽、陰、正、反、花
- 英文：H/T（heads/tails）
- 數字：1/0（1=背=陽，0=字=陰）
- 符號：+/-（+=背，-=字）
模式 3：互動式逐次輸入（推薦給用戶實際擲銅錢時）
python jinqian_gua.py interactive
腳本會依次提示「第 1 次擲（初爻）：」等，用戶輸入後即時驗證。

進階：搭配納甲
金錢卦的真正力量在於「納甲六親」系統。起完卦後，可呼叫 najia.py 加上：
- 每爻的天干地支與五行
- 世爻、應爻位置
- 六親（父母、兄弟、子孫、妻財、官鬼）
- 六獸（青龍、朱雀、勾陳、螣蛇、白虎、玄武）
- 旬空、月破、日破
- 用神判定（依問事類型）
詳見：references/najia-guide.md、references/najia-tables.md
解卦完整流程
1. 確定問事（具體、清楚）
2. 擲幣 6 次成卦
3. 計算本卦、變卦、動爻
4. 依朱熹法找主辭
5. （納甲）裝卦：標註六親、世應、旬空
6. （納甲）定用神
7. （納甲）看用神旺衰、生克、動變
8. 結合本卦/變卦卦辭爻辭綜合判斷
9. 給出策略建議

注意事項
誠心
- 擲卦前需誠心默念所問，雜念太多卦不準
- 一事一占，不可反覆問同一問題（筮不過三）

擲法細節
- 雙手合攏搖晃，自然落下，不可故意控制
- 3 枚銅錢應同時落下，分開擲不算
- 若銅錢立著或滾走，該次重擲

問題明確
- 「我下個月升職會成嗎？」（具體）
- 「我會不會好？」（過於模糊，難斷）
- 涉及他人，只能以問卦者自身視角分析

倫理
- 不預測死亡時間
- 不替代醫療、法律、財務專業諮詢
- 結果僅供參考，最終決策仍由人自負
