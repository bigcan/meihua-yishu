六、旬空（六甲空亡）
六十甲子分六旬，每旬有兩個地支為「空亡」：
旬首|旬內 10 日|空亡支
甲子|甲子→癸酉|戌、亥
甲戌|甲戌→癸未|申、酉
甲申|甲申→癸巳|午、未
甲午|甲午→癸卯|辰、巳
甲辰|甲辰→癸丑|寅、卯
甲寅|甲寅→癸亥|子、丑
判斷規則：
- 卦中某爻地支屬「日旬空」→ 該爻旬空
- 旺相之爻旬空 → 暫時不發力，出旬後恢復
- 休囚之爻旬空 → 真空，事難成
- 動爻旬空 → 不為真空（動則出空）
- 用神旬空 → 重要訊號：應期常在「出旬之日」或「沖空之日」

七、六沖、六合、三合
六沖（互克對沖）：
子↔午、丑↔未、寅↔申、卯↔酉、辰↔戌、巳↔亥
六合（陰陽相吸）：
子合丑、寅合亥、卯合戌、辰合酉、巳合申、午合未
三合局（三支聚一氣）：
水局|申子辰
木局|亥卯未
火局|寅午戌
金局|巳酉丑
應用：
- 卦中三爻成三合局 → 聚合有力，事易成
- 用神與日月成三合 → 加強用神力量
- 卦逢六合（六沖卦變六合卦、或上下卦逢合）→ 事和諧

八、月破、日破
月破：被月支沖之爻
- 例：寅月，申爻為月破
日破：被日支沖之爻，且該爻休囚（無氣）
- 旺相被沖 → 暗動（沖起，反有力）
- 休囚被沖 → 日破（受傷）
口訣：「月破難扶，日破可救」（月破整月不用，日破過該日即恢復）

九、進神、退神
動爻變化的特殊情況：
進神（順行）：寅→卯、巳→午、申→酉、亥→子、丑→辰→未→戌
- 力量增強、事情前進、吉象
退神（逆行）：卯→寅、午→巳、酉→申、子→亥、戌→未→辰→丑
- 力量減弱、事情後退、漸消
判斷：用神化進神為吉，忌神化退神為吉。

十、反吟、伏吟
反吟：本卦變卦六爻地支兩兩相沖（≥3 爻）
- 意義：反覆動盪、悲泣呻吟
- 「反吟不利出行、不利病人、不利官訟」
伏吟：本卦變卦六爻地支兩兩相同（≥3 爻）
- 意義：呻吟憂愁、停滯不前
- 「伏吟不利動作、宜守不宜動」
卦反吟：本卦上下卦與變卦上下卦相易（特殊大反吟）

十二、伏神（用神不上卦時的處理）
規則：若本卦無對應六親的用神 → 從「本宮首卦（八純卦）」對應位置取伏神
- 伏神：藏起來的用神
- 飛神：壓在伏神上方的本卦該位置之爻
飛伏關係的吉凶：
- 飛生伏 → 提拔（吉）
- 伏克飛 → 出現（吉）
- 飛克伏 → 受制（凶）
- 伏生飛 → 洩氣（不利）
- 飛伏同（比和） → 安穩
出現條件：
- 日月生扶伏神
- 飛神空、破、墓、絕
- 動爻沖飛神
