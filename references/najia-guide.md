納甲（六爻預測）操作指南
本檔說明六爻系統的「裝卦 → 定用神 → 看旺衰 → 看動變 → 判吉凶 → 推應期」完整流程。
數據總表見 najia-tables.md，用神對照詳細版見 liuyao-yongshen.md。
原典依據見 classics/classics-index.md，遇到疑難時可查 增刪卜易 / 卜筮正宗 對應章節。

一、裝卦流程（scripts/najia.py 已自動化）
1. 起卦：金錢卦擲幣（jinqian_gua.py）或數字、時間（meihua_calc.py）轉本卦
2. 確定動爻位置
3. 對本卦做以下標註：
   a. 各爻納甲（天干、地支、五行）—— 由 PURE_GUA_NAJIA 推導
   b. 歸宮（哪一宮、第幾世）—— 查 HEX_TO_PALACE
   c. 世爻、應爻位置 —— 查 SHI_POS / YING_POS
   d. 六親（父母/兄弟/子孫/妻財/官鬼）—— 比較地支五行與宮五行
   e. 六獸 —— 依日干起例
   f. 旬空 —— 依日干支查旬
   g. 月破、日破、日合 —— 依月支、日支
4. 標註變卦：每動爻變後的地支、五行、六親
5. 整體分析：反吟、伏吟、進神、退神、三合、六合
6. 找用神：依問事類型；若不上卦則取本宮首卦伏神
7. 判用神旺衰：日月生扶、空破、動變
8. 推應期：用神之沖、合、值、實之日／月

二、用神旺衰判定
五級狀態：
旺 > 相 > 休 > 囚 > 死
依月令季節：
月令|當令五行
正月（寅）|木旺、火相
二月（卯）|木旺、火相
三月（辰）|土旺
四月（巳）|火旺、土相
五月（午）|火旺、土相
六月（未）|土旺
七月（申）|金旺、水相
八月（酉）|金旺、水相
九月（戌）|土旺
十月（亥）|水旺、木相
十一月（子）|水旺、木相
十二月（丑）|土旺
五行旺衰公式：
- 當令者旺、所生者相
- 生我者休、克我者囚、我克者死
日辰扶抑：
- 用神被日辰生扶 → 加強
- 用神被日辰沖克 → 減弱
- 用神就是日支 → 大旺（日建）
- 用神就是月支 → 大旺（月建）

三、生克沖合的優先順序
傳統口訣：「日月為先，動爻次之，靜爻再次」
具體優先：
1. 月建（月支）作用最強
2. 日辰（日支）次之
3. 動爻可作用其他爻
4. 變爻只回頭作用本動爻（不出爻位）
5. 靜爻之間不相生克
動爻作用範圍：
- 動爻可生、可克、可沖、可合任何爻
- 動爻變出的變爻：只能回頭生、克、沖、合本動爻
- 變爻不影響其他爻（除非變爻本身又動，但單爻只動一次）

四、六親作用
六親之間的生克：
- 父母生兄弟、兄弟生子孫、子孫生妻財、妻財生官鬼、官鬼生父母
- 父母克子孫、子孫克官鬼、官鬼克兄弟、兄弟克妻財、妻財克父母
記憶：
- 父母 = 生我者 → 在卦中是長輩、文書、房屋
- 子孫 = 我生者 → 是子女、下屬、解神（剋官鬼）
- 妻財 = 我克者 → 是錢財、妻妾
- 官鬼 = 克我者 → 是工作、丈夫、疾病
- 兄弟 = 同我者 → 是同輩、競爭者，剋妻財

五、忌神、原神、仇神
針對用神，定義其他六親為：
- 用神：問事所求
- 原神：生用神者（用神之父母）
- 忌神：克用神者（用神之官鬼）
- 仇神：克原神、生忌神者
判斷重點：
- 原神旺動 → 用神有源
- 忌神旺動 → 用神受克
- 仇神動：原神受克或忌神被生 → 不利

六、伏神（用神不上卦）
規則：
1. 本卦中無對應六親的爻 → 用神不上卦
2. 從「本宮首卦（八純卦）」找對應六親之爻 → 此為伏神
3. 伏神之上的本卦該位置爻 → 飛神
伏神能否出現的條件：
出現（吉）|不出（凶）
日月生伏神|日月克伏神
飛神空破墓絕|飛神旺相壓制
動爻沖飛神（破飛現伏）|飛神得地支扶
伏神自旺有氣|伏神休囚無氣
應期：伏神出現之日（出空、值日、衝飛）即應。

七、反吟、伏吟（卦象凶兆）
反吟：本變相沖
- 卦象意義：事情反覆，內心悲泣
- 「反吟卦：謀事兩番，悲泣呻吟」
- 不利出行、病人、官訟
伏吟：本變相同
- 卦象意義：呻吟憂愁，停滯不前
- 「伏吟卦：憂愁呻吟，動而不變」
- 不宜任何動作，宜守
卦反吟：上下卦互易
- 大震盪，常表「位置顛倒」「物是人非」

八、進神、退神
進神（地支順行變）：
- 寅→卯（木進）
- 巳→午（火進）
- 申→酉（金進）
- 亥→子（水進）
- 丑→辰、辰→未、未→戌（土進）
退神（地支逆行變）：
- 卯→寅、午→巳、酉→申、子→亥
- 戌→未、未→辰、辰→丑（土退）
判斷：
- 用神化進神：力量增強、事必成
- 用神化退神：力量減弱、事漸消
- 忌神化退神：威脅減輕（吉）
- 忌神化進神：威脅加重（凶）

九、化空、化破、化墓、化絕（動爻變化的吉凶）
化空：動爻變出之爻地支屬旬空 → 力量打折
化破：變出之爻被月沖 → 力量受傷
化墓：變出之爻地支為墓庫
- 五行墓庫：木墓未、火墓戌、土墓辰、金墓丑、水墓辰（土墓於辰）
- 妻財化墓 → 財進不出
- 官鬼化墓 → 凶事漸隱（吉）
化絕：變出之爻地支為絕地
- 五行絕地：木絕申、火絕亥、土絕巳、金絕寅、水絕巳
- 用神化絕 → 大凶
回頭克／回頭生：
- 動爻變出之爻克本爻 → 回頭克（凶）
- 動爻變出之爻生本爻 → 回頭生（吉）

十、應期推算
應期 = 卦象顯示之吉凶於何時發生。常用判法：
1. 用神靜旺：值用神地支之日／月
2. 用神靜衰：逢沖實之日（沖即實，去其虛弱）
3. 用神動：值日或變爻地支之日
4. 用神空：出旬之日、或沖空之日（沖空則實）
5. 用神破：合破之日（合住破爻方有用）
6. 用神入墓：沖墓之日（衝開墓庫）
7. 用神絕：生絕之日（長生之地）
8. 伏神：出現之日（飛神空破或日月引出）
時間單位：
- 近事用日（小卦）
- 中期用月
- 遠事用年（年支）
- 急事用時辰
- 卦中有合 → 合處逢沖則應
- 卦中有沖 → 沖處逢合則應（合住沖才結束）

十一、實戰流程範例
問：本月能否求得財？日：甲子，月：寅。
擲得：地風升，三爻、四爻動。
1. 裝卦（najia.py 自動）：
   - 升卦 = 震宮四世卦，世第4、應第1
   - 木為我，妻財是土
   - 三爻辛酉金（官鬼）動 → 變坎下卦 → 戊午火（子孫）
   - 四爻癸丑土（妻財、世）動 → 變震上卦 → 庚午火（子孫）
2. 找用神：求財 → 妻財
   - 第1爻辛丑土（妻財、應）— 用神之一
   - 第4爻癸丑土（妻財、世，動）— 用神之二（更重要：持世）
3. 判旺衰：
   - 妻財丑土，寅月 → 月木克土（休囚）
   - 日支子，子丑相合（日合）→ 略有救
   - 第4爻動化午火（子孫）→ 回頭生（吉！）
4. 整體：
   - 持世妻財雖月衰但日合，又化子孫回頭生 → 財可得
   - 第3爻官鬼動 → 競爭、或耗財事項，但化子孫即被克化掉（凶象解）
   - 兩動化進化退要看具體：丑→午（土到火）非進非退，為「化子孫」即「化生我克之物」對求財有利
5. 應期：
   - 用神丑：沖實之日未、合之日子（今日即是！）
   - 動爻：今日或巳午之日（午即子孫旺日）
   結論：今日或近 1-3 日內有財訊，金額不大但可得。
